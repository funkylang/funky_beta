# std::set_buttons (a polymorphic function)

  sets the buttons of a window

  Parameters:
    window_manager: a window manager
    id: the id of the window
    buttons: the buttons to set

  Result:
    window_manager: the updated window manager

  If no id is given then the id of the focused window is used.

  Topic: Window Manager

  See also: std::get_buttons, std::create_window

  Example:

    $my_window_manager create_window_manager(terminal)
    $my_window create_window(my_window_manager "My Window")
    set_buttons &my_window_manager my_window list(BACK_BUTTON FORWARD_BUTTON)
