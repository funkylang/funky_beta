# std_types::hash_bag/std::merge (a method)

  merges two hash bags

  Parameters:
    self: the base bag
    other: the bag to merge

  Result:
    bag: the merged bag

  The result is a new bag that contains all elements of the base bag and
  the elements of the other bag.

  If an element is contained in both bags then the number of occurrences
  of this element in the result is the sum of the number of occurrences
  in the base bag and the other bag.

  Topic: Bags

  See also: std_types::hash_bag/std::difference
