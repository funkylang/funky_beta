# std::has_prefix (a polymorphic function)

  checks whether the first argument has the second argument as a prefix

  Parameters:
    string: the string to check
    prefix: the prefix to check for

  Result:
    boolean: *true* if the string has the prefix, *false* otherwise

  Topic: Collections

  See also: std::has_suffix, std::without_prefix, std::contains

  Example:

    $path "/home/user/"
    if
      path .has_prefix. "/home/":
	println! "path starts with /home/"
      :
	println! "path does not start with /home/"

  Output:

    path starts with /home/
