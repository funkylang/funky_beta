# std_types::hash_table/std::map_reduce (a method)

  combines a map and a reduce operation into a single operation

  Does not create an intermediate table to hold the "mapped" values.

  Parameters:
    self: the table to work on
    map_function: the function to apply to each element of the table
    reduce_function: the function to apply to pairs of values
    unit: the value to return for empty tables

  Result:
    value: the result of the reduction

  Topic: Tables

  See also: std_types::hash_table/std::map, std_types::hash_table/std::reduce
