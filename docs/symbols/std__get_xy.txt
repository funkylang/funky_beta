# std::get_xy (a polymorphic function)

  returns the cursor position

  Parameters:
    window_manager: the window manager
    id: the id of the window (optional)

  Results:
    x: the x-coordinate of the cursor
    y: the y-coordinate of the cursor

  If no id is given then the id of the focused window is used.

  Topic: Window Manager

  See also: std_types::window_manager/get_xy, std::goto_xy

  Example:

    $x $y get_xy &my_window_manager
