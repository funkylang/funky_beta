# std::get_inbounds_width (a polymorphic function)

  returns the inbounds width of a window

  Parameters:
    wm: a window manager
    id: the id of the window (optional)

  Result:
    width: the inbounds width of the window

  If no id is given then the inbounds width of the focused window is
  returned.

  Topic: Window Manager

  See also: std::set_inbounds_width

  Example:

    $my_wm create_window_manager(terminal)
    $my_window create_window(&my_wm CONTENT "Hello, World!")
    set_inbounds_width &my_wm my_window 100
    println! get_inbounds_width(&my_wm my_window)
