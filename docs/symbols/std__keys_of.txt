# std::keys_of (a function)

  returns a list of all keys of the specified collection

  Parameters:
    collection: the collection

  Result:
    keys: a list of keys

  Topic: Collections

  See also: std::values_of

  Example:

    $table empty_hash_table
    $table("John") 42
    $table("Mary") 25
    $table("Irene") 49
    $keys keys_of(table)
    dump! 2 `keys

  Output:

    keys:
      list
        "Mary"
        "Irene"
        "John"
