# html::TBODY (a function)

  generates a TBODY element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the TBODY element

  The function is used as a block function, with the content placed between the
  TBODY tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      TBODY
	: TR
	  : TH "Name"
	  : TH "Age"
	: TR
	  : TD "Alice"
	  : TD "30"
	: TR
	  : TD "Bob"
	  : TD "25"

  Output:

    <tbody>
    <tr>
    <th>Name</th>
    <th>Age</th>
    </tr>
    <tr>
    <td>Alice</td>
    <td>30</td>
    </tr>
    <tr>
    <td>Bob</td>
    <td>25</td>
    </tr>
    </tbody>
