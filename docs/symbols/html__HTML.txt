# html::HTML (a function)

  generates an HTML document

  Parameters:
    items*: the content of the document

  Result:
    a string containing the HTML document

  The function generates a complete HTML document with the given content.

  The content is typically a sequence of elements such as HEAD and BODY.

  The function is used as a block function, with the content placed between the
  HTML tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      HTML
	: HEAD
	  : TITLE "My great Homepage"
	  : LINK "rel" = "stylesheet" "href" = "style.css"
	: BODY
	  : H1 "My great Homepage"
	  : P "This is my great homepage."

  Output:

    <html>
    <head>
    <title>My great Homepage</title>
    <link rel="stylesheet" href="style.css"/>
    </head>
    <body>
    <h1>My great Homepage</h1>
    <p>This is my great homepage.</p>
    </body>
    </html>
