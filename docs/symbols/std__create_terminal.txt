# std::create_terminal (a function)

  creates a new terminal object

  Parameters:
    width: the width of the terminal
    height: the height of the terminal
    options: a list of key-value-pairs
      DRAW_COLOUR: the default draw colour (default: BLACK)
      CLEAR_COLOUR: the default clear colour (default: WHITE)
      VISIBLE_CURSOR: a boolean value indicating whether the cursor is visible
	(default: true)

  Result:
    terminal: a new terminal object

  Topic: Terminal

  See also: std_types::terminal, std::update_terminal

  Example:

    <require basic/stdlib>
    <require terminal/terminal>

    get_terminal_size! STDIN_FILENO $width $height
    create_terminal $terminal width height
      DRAW_COLOUR = WHITE
      CLEAR_COLOUR = BLUE
    draw_text &terminal
      10 # column
      5 # row
      "
	This is a multi-line
	text.
    update_terminal &terminal $output # create output character stream
    print! output # print the generated character stream
