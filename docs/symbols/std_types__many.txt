# std_types::many (a type)

  the protoype grammar object for matching one or more occurences of something

  This is a grammar object that matches one or more occurrences of the given
  expression.

  Topics: Grammar Objects

  See also: std::many

  Example:

    $str "123abc"
    match many(DIGIT) str $len
    println! len

  Output:

    3

  Example:

    $str "abc"
    match many(DIGIT) str $len
    dump! `len

  Output:

    len: <undefined>
