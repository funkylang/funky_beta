# std::set_mouse_reporting (a polymorphic function)

  sets the mouse reporting mode

  Parameters:
    terminal: a terminal
    mouse_reporting: the mouse reporting mode

  Result:
    terminal: the updated terminal

  The mouse reporting mode can be one of the following:

  * <std::DISABLED>: no mouse reporting
  * <std::CLICKS>: mouse clicks are reported
  * <std::DRAG>: mouse clicks and drags are reported
  * <std::MOVE>: mouse clicks, drags and movements are reported

  Topic: Terminal

  See also: std::get_mouse_reporting

  Example:

    set_mouse_reporting &terminal CLICKS
