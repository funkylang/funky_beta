# std::closedir (a builtin I/O-function)

  closes a directory stream

  Parameters:
    directory: the directory stream to close

  Results:
    none

  **This function must be called with I/O-access rights!**

  Topic: Input-Output

  See also: std::opendir

  Example:

    # open a directory
    directory = std::opendir("/tmp");
    # close the directory
    std::closedir(directory);
