# html::TH (a function)

 generates a TH element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the TH element

  The function is used as a block function, with the content placed between the
  TH tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      TH
	"Name"

  Output:

    <th>Name</th>
