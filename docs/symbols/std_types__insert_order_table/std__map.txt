# std_types::insert_order_table/std::map (a method)

  applies a function to all values of the insert order table

  Parameters:
    self: the insert order table which values will be mapped
    function: the function to apply to each element of the table

  Result:
    table: a table containing the "mapped" items

  The function is applied to each value of the table.

  The function results are collected in a table of the same type as the
  original table.

  The function applications might be executed in parallel!

  Topic: Tables
