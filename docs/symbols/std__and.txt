# std::and (a polymorphic function)

  performs a "short cut" and-operation

  Parameters:
    left: must be <std::true> or <std::false>
    right: a function that evaluates to <true> or <false>

  Result:
    the result of the right argument if the left argument is <std::true>,
    otherwise <std::false>

  The right argument is only evaluated if the left argument is <std::true>.

  Topic: Boolean

  See also: std::or, std::if

  Example:

    $x 5
    println!
      if
        x > 3 && x < 10
        -> "x is between 3 and 10"
        -> "x is not between 3 and 10"

  Output:

    x is between 3 and 10
