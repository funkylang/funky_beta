# std_types::value_range/std::for_each (an I/O-method)

  iterates over all items of a value-range

  Parameters:
    self: the value-range to iterate over
    body: the function to call for each item
    finally: the function to call after the iteration

  Results:
    none

  **This function must be called with I/O-access rights!**

  The function "body" is called for each item of the value-range.
  The function "finally" is called after the iteration.

  Topic: Loops

  See also: std::next, std::break
