# ai::LOG_LEVEL (a unique item)

  the log level to use for the AI-server

  Topic: Deprecated AI

  The log level is used to control the amount of output generated by the
  AI-server.

  The log level is an integer value. The following log levels are defined:

  * 0: no output
  * 1: only the generated tokens are printed
  * 2: the generated tokens and the confidence values are printed
  * 3: the generated tokens, the confidence values and the time taken to
    generate the tokens are printed

  See also: ai_types::model/ai::generate
