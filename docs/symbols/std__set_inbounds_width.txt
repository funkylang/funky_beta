# std::set_inbounds_width (a polymorphic function)

  sets the inbounds width of a window

  Parameters:
    window_manager: a window manager
    id: the id of the window (optional)
    width: the new inbounds width

  Result:
    window_manager: the updated window manager

  If no id is given then the focused window is used.

  Topic: Window Manager

  See also: std_types::window_manager/set_inbounds_width,
            std::get_inbounds_width

  Example:

    $my_window_manager create_window_manager(terminal)
    $id create_window(&my_window_manager CONTENT "Hello, World!")
    set_inbounds_width &my_window_manager id 100
