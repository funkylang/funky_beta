# std::pad_bottom (a polymorphic function)

  pads a screen to the specified height

  Parameters:
    screen: some kind of screen
    height: the desired height
    clear_colour: the clear colour to use

  Result:
    screen: the updated screen

  The screen is padded at the bottom using the current clear colour. Use
  <std::set_clear_colour> to change the current clear colour.

  Topic: Terminal

  See also: std_types::screen/std::pad_bottom, std::pad_top, std::pad_left,
            std::pad_right, std::set_clear_colour

  Example:

    $my_screen create_screen(terminal 40 25)
    set_clear_colour &my_screen GREEN
    pad_bottom &my_screen 30
