# html::TABLE (a function)

 generates a TABLE element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the TABLE element

  The function is used as a block function, with the content placed between the
  TABLE tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      TABLE
	: TBODY
	  : TR
	    : TH "Name"
	    : TH "Age"
	  : TR
	    : TD "John Doe"
	    : TD "30"

  Output:

    <table>
    <tbody>
    <tr>
    <th>Name</th>
    <th>Age</th>
    </tr>
    <tr>
    <td>John Doe</td>
    <td>30</td>
    </tr>
    </tbody>
    </table>
