# html::SPAN (a function)

  generates a SPAN element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the SPAN element

  The function is used as a block function, with the content placed between the
  SPAN tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      SPAN
	"class" = "highlight"
	"This is highlighted text"

  Output:

    <span class="highlight">This is highlighted text</span>
