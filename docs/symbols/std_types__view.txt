# std_types::view (a type)

  a view of a screen

  A *view* is a rectangular region that restricts drawing operations on their
  parent screen.

  All drawing operations occur on a screen (not the view).

  To draw using a *view* both the view's parent screen and the view must be
  specified.

  Topics: Terminal, IO

  See also: std_types::generic_terminal
