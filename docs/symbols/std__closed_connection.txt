# std::closed_connection (a function)

  closes a connection

  Parameters:
    io: the I/O-object
    fd: the file descriptor of the connection to close

  Result:
    none

  The connection is closed and the protocol type is set to "undefined".

  Topic: Network Clients and Servers

  See also: std::close_connection, std::upgrade_to_websocket

  Example:

    $std::closed_connection: (io fd)
      !io.protocols_of(fd) undefined
      log io 4 -> "closed connection @(serialize(fd))"
