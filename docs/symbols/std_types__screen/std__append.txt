# std_types::screen/std::append (a method)

  appends two or more screens to form a single screen

  Parameters:
    self: the first screen
    screens*: the screens to be appended

  Result:
    screen: the resulting screen

  The screens to be appended must have the same width as the first screen.

  The resulting screen has the same width as the first screen and its height
  is the sum of the heights of the screens to be appended.

  Topic: Terminal

  See also: std_types::screen/std::adjoin
