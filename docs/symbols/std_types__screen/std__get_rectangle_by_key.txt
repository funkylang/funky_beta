# std_types::screen/std::get_rectangle_by_key (a method)

  returns the rectangle with the specified key

  Parameters:
    self: the screen
    key: the key of the rectangle to search for

  Result:
    rectangle: the rectangle with the specified key

  The rectangle is returned as a tuple:

    (info x y width height)

  If no rectangle with the specified key is found then <std::undefined> is
  returned.

  Topic: Terminal

  See also: std_types::screen/std::get_rectangle_at,
