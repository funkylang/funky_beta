# std_types::screen/std::invert (a method)

  inverts the colours of the specified rectangular area

  Parameters:
    self: the screen to be (partially) inverted
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    width: width of the area
    height: height of the area

  Result:
    screen: the updated screen

  The specified area is inverted.

  The area is clipped at the screen's borders.

  This function also works for "graphical screens" (line_height != 1).

  Topic: Terminal

  See also: std_types::screen/std::set_draw_colour,
            std_types::screen/std::set_clear_colour,
            std_types::screen/std::clear
