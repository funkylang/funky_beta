# std_types::screen/std::draw_shadow (a method)

  draws a shadow around a rectangular area

  Parameters:
    self: the screen to draw the shadow on
    factor: the factor by which the colours are darkened
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    width: width of the area
    height: height of the area

  Result:
    screen: the updated screen

  The shadow is drawn using the current draw- and clear colour.

  The area is clipped at the screen's borders.

  This function also works for "graphical screens" (line_height != 1).

  Topic: Terminal

  See also: std_types::screen/std::set_draw_colour,
            std_types::screen/std::set_clear_colour

  Example:

    <require basic/stdlib>
    <require terminal/terminal>

    initialize_terminal!: (terminal)
      draw_text &terminal 10 5 "Hello, world!"
      draw_shadow &terminal 10 5 15 3
      update_terminal &terminal $output # create output character stream
      print! output # print the generated character stream
