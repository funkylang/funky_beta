# std_types::screen/std::scroll_right (a method)

  scrolls the screen to the right

  Parameters:
    screen: the screen to be scrolled
    delta: the number of columns to scroll

  Result:
    screen: the updated screen

  The screen is scrolled to the right by the specified number of columns.

  The leftmost columns are filled with spaces using the current draw- and
  clear colour.
