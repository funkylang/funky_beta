# std_types::screen/std::scroll_down (a method)

  scrolls the screen down

  Parameters:
    self: the screen to be scrolled
    delta: the number of lines to scroll

  Result:
    screen: the updated screen

  The screen is scrolled down by the specified number of lines.

  The lines that are scrolled out of the screen are lost.

  The lines that are scrolled into the screen are filled with spaces using
  the current draw and clear colour.

  Topic: Terminal

  See also: std_types::screen/std::scroll_up,
            std_types::screen/std::scroll_left,
            std_types::screen/std::scroll_right
