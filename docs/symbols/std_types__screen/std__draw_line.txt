# std_types::screen/std::draw_line (a method)

  draws a line between two points

  Parameters:
    self: the screen to draw on
    x1: x-coordinate of the first point
    y1: y-coordinate of the first point
    x2: x-coordinate of the second point
    y2: y-coordinate of the second point

  Result:
    screen: the updated screen

  The line is drawn using the current draw colour.

  The line is clipped at the screen's borders.

  This function also works for "graphical screens" (line_height != 1).

  Topic: Terminal

  See also: std_types::screen/std::set_draw_colour,
