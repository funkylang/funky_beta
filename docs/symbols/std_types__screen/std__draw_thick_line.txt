# std_types::screen/std::draw_thick_line (a method)

  draws a thick line between two points

  Parameters:
    self: the screen to draw on
    x1: x-coordinate of the first point
    y1: y-coordinate of the first point
    x2: x-coordinate of the second point
    y2: y-coordinate of the second point

  Result:
    screen: the updated screen

  The line is drawn using the current draw colour.

  The line must be horizontal or vertical.

  The line is clipped at the screen's borders.

  Topic: Terminal

  See also: std_types::screen/std::draw_line,
            std_types::screen/std::draw_border,
            std_types::screen/std::draw_thick_border

  Example:

    <require basic/stdlib>
    <require terminal/terminal>

    create_terminal! $terminal 80 25
    draw_thick_line! terminal 10 10 70 10
    draw_thick_line! terminal 70 10 70 20
    update_terminal! terminal $output
    print! output
