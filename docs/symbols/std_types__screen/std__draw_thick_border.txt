# std_types::screen/std::draw_thick_border (a method)

  draws a thick border around the specified rectangular area

  Parameters:
    self: the screen to be (partially) cleared
    x1: x-coordinate of the upper left corner
    y1: y-coordinate of the upper left corner
    x2: x-coordinate of the lower right corner
    y2: y-coordinate of the lower right corner

  Result:
    screen: the updated screen

  The specified area is surrounded by a border using the the current draw-
  and clear colour.

  The area is clipped at the screen's borders.

  This function also works for "graphical screens" (line_height != 1).

  Topic: Terminal

  See also: std_types::screen/std::set_draw_colour,
            std_types::screen/std::set_clear_colour,
            std_types::screen/std::draw_border
