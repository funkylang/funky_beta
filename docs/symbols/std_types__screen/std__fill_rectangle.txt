# std_types::screen/std::fill_rectangle (a method)

  fills a rectangular area with the specified colour and opacity

  Parameters:
    screen: the screen to be filled
    colour: the fill colour
    opacity: the fill opacity (0 = transparent, 1 = opaque)
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    width: width of the area
    height: height of the area

  Result:
    screen: the updated screen

  The specified area is filled with the specified colour and opacity.

  The area is clipped at the screen's borders.

  This function also works for "graphical screens" (line_height != 1).

  Topic: Terminal

  See also: std_types::screen/std::clear, std_types::screen/std::copy
