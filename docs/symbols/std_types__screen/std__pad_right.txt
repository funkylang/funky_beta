# std_types::screen/std::pad_right (a method)

  pads the screen on the right side

  Parameters:
    self: the screen to pad
    width: the desired width of the screen
    clear_colour: the clear colour to use for the padding

  Result:
    screen: the padded screen

  If the screen's width is less than the specified width then the screen is
  padded on the right side with spaces using the specified clear colour.

  If the screen's width is greater than or equal to the specified width then
  the screen is not changed.

  Topic: Terminal

  See also: std_types::screen/std::pad_left, std_types::screen/std::truncate
