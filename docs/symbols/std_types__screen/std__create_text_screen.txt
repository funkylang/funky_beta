# std_types::screen/std::create_text_screen (a method)

  creates a screen from a text string

  Parameters:
    template: a screen used to determine the default draw and clear colour
    text: the text string
    draw_colour: the draw colour (optional)
    clear_colour: the clear colour (optional)

  Result:
    screen: the created screen

  Creates a screen from a text string.

  The draw and clear colour of the screen is determined by the draw and clear
  colour of the template screen.

  If the draw and clear colour is specified then the template screen is
  ignored.
