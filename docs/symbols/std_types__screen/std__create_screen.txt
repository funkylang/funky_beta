# std_types::screen/std::create_screen (a method)

  creates a new screen

  Parameters:
    screen: the screen to be copied
    width: the width of the new screen
    height: the height of the new screen
    options: a list of options

  Result:
    screen: the new screen

  Creates a new screen with the specified width and height.

  The new screen is filled with spaces using the specified draw and clear
  colours.

  If the width or height is not specified then the width and height of the
  specified screen is used.

  The options are:

    DRAW_COLOUR: the draw colour to use (default: the draw colour of the
                 specified screen)
    CLEAR_COLOUR: the clear colour to use (default: the clear colour of the
                  specified screen)

  Topic: Terminal

  See also: std_types::screen/std::resize, std_types::screen/std::copy
