# std::initialized_boolean_array (a builtin function)

  constructs an initialized array

  Parameters:
    dimension_1, ..., dimension_n: the dimensions of the array
    initial_value: the initial value for all array items

  Result:
    array: the array

  Topic: Arrays

  See also: std::boolean_array, std_types::boolean_array/new

  Example:

    # create a 3x4 array of boolean values, all initialized to false
    array = std::initialized_boolean_array(3, 4, false)

    # create a 3x4 array of boolean values, all initialized to true
