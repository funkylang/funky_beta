# std::package_version (a builtin function)

  returns the version of the Funky installation

  Result:
    version: the version of the current package

  Topic: Environment

  See also: std::installation_prefix

  Example:

    $shared_path
      append
	installation_prefix()
	"/share/funky-" package_version()

    println! shared_path

  Output:

    /usr/local/share/funky-0.0.0
