# std::draw_text (a polymorphic function)

  draws text on a screen

  Parameters:
    screen # some kind of screen
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    args*: text to draw

  Result:
    screen: the updated screen

  The text is drawn using the current draw colour. Use
  <std::set_draw_colour> to change the current draw colour.

  Topic: Terminal

  See also: std::draw_attributed_text, std::clear, std::set_draw_colour

  Example:

    $my_screen create_screen(terminal 40 25)
    set_draw_colour &my_screen RED
    draw_text &my_screen 11 6 "Hello World!"
