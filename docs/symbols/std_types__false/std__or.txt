# std_types::false/std::or (a method)

  evaluates its second argument and returns its result

  Parameters:
    self: <std_types::false>
    right: an expression that evaluates to <std::true> or <std::false>

  Result:
    result: the result of the evaluation of the right argument

  The first argument is already <std::false>, so the second argument decides
  whether the *or*-expression ist *true* or not.
