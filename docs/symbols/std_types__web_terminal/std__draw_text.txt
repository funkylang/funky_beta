# std_types::web_terminal/std::draw_text (a method)

  draws a text at the specified position

  Parameters:
    self: the screen to draw on
    x: x-coordinate of the text's position
    y: y-coordinate of the text's position
    text: the text to draw

  Result:
    screen: the updated screen

  The text is drawn using the current draw colour.

  The text is clipped at the screen's borders.

  This function also works for "graphical screens" (line_height != 1).

  Topic: Terminal

  See also: std_types::screen/std::set_draw_colour,
            std_types::screen/std::draw_character,
            std_types::screen/std::plot_line

  Example:

    draw_text &screen x y "Hello World!"
