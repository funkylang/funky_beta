# std_types::web_terminal/std::update (a method)

  updates the terminal

  Parameters:
    self: the terminal to update

  Result:
    command: the command to update the terminal

  This function is called by the main loop to update the terminal.

  If the terminal has not been updated since the last call then the
  command to update the terminal is returned.

  If the terminal has been updated since the last call then an empty
  string is returned.
