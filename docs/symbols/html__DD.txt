# html::DD (a function)

  generates a DD element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the DD element

  The function is used as a block function, with the content placed between the
  DD tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      DD
	"Alice"

  Output:

    <dd>Alice</dd>
