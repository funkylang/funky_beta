# std::get_content (a polymorphic function)

  returns the content of a window

  Parameters:
    window_manager: a window manager
    id: the id of the window

  Result:
    content: the content of the window

  If the window does not exist then "undefined" is returned.

  Topic: Window Manager

  See also: std::set_content

  Example:

    $my_window_manager create_window_manager(terminal)
    $my_window create_window(&my_window_manager "My Window")
    set_content &my_window_manager my_window "Hello, World!"
    println! get_content(&my_window_manager my_window)
