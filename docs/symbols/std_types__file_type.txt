# std_types::file_type (a builtin type derived from std_types::object)

  a file type

  Parameters:
    value: an integer

  Result:
    file_type: a file type

  File types are used to identify the type of a file.

  The following file types are defined:

  * BLOCK_DEVICE
  * CHARACTER_DEVICE
  * DIRECTORY
  * NAMED_PIPE
  * SYMBOLIC_LINK
  * REGULAR_FILE
  * SOCKET
  * UNKNOWN_FILE_TYPE

  Topics: POSIX

  See also: std_types::file_descriptor, std_types::file_type/to_integer

  Example:

    opendir! $dir "/bin"
    loop:
      readdir! $entry dir
      if
        entry.is_an_error
        break
        :
          if
            type_of(entry) == SYMBOLIC_LINK: # check file type
              println! name_of(entry)
              next!
            next
