# std::all_of (a function)

  checks whether all items of a collection pass the specified test

  Parameters:
    collection: the collection which items to check
    test: the test function
    argument: optional additional argument that is passed to the test function

  Result:
    true if all items in the collection fullfill the test

  The function returns true if all items in the specified collection fullfill
  the specified test.

  The test function is called for each item in the collection until it returns
  false or all items have been processed.

  If the test function returns false for any item, the function returns false.
  Otherwise it returns true.

  Topic: Collections

  See also: std::any_of, std::count

  Example:

    $persons
      hash_table
	"Gregory" = 39
	"Julia" = 37
	"Maria" = 33
	"James" = 41
	"Sarah" = 38

    if
      all_of(persons less 40):
	println! "true"
      :
	println! "false"

  Output:

    false
