# std::goto_xy (a polymorphic function)

  moves the cursor to a specific position

  Parameters:
    window_manager: a window manager
    id: the id of the window (optional)
    x: the x-coordinate
    y: the y-coordinate

  Result:
    window_manager: the updated window manager

  If no id is given then the focused window is used.

  Topic: Window Manager

  See also: std_types::window_manager/goto_xy, std::get_xy

  Example:

    $my_window_manager create_window_manager(terminal)
    $my_window_id create_window(&my_window_manager CONTENT "Hello, World!")
    goto_xy &my_window_manager my_window_id 10 5
