# std::from_html (a function)

  parses a string containing HTML code

  Parameters:
    text: the text to parse

  Result:
    element: the root element of the HTML document

  The function parses the specified text and returns the root element of the
  HTML document.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/import/html>

    $html "
      <html>
	<head>
	  <title>My first HTML document</title>
	</head>
	<body>
	  <h1>My first HTML document</h1>
	  <p>This is a paragraph.</p>
	  <p>This is another paragraph.</p>
	</body>
      </html>
    $element from_html(html)
