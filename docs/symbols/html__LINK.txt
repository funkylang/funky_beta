# html::LINK (a function)

 generates a LINK element

  Parameters:
    options*: the options for the element

  Result:
    a string containing the LINK element

  The function is used as a block function, with the options placed between the
  LINK tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      LINK
	"rel" = "stylesheet" "href" = "style.css"

  Output:

    <link rel="stylesheet" href="style.css"/>
