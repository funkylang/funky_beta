# std::unset_selection (a polymorphic function)

  unsets the selection of a window

  Parameters:
    window_manager: a window manager
    id: the id of the window

  Result:
    window_manager: the updated window manager

  If the window has no selection then the function does nothing.

  Topic: Window Manager

  See also: std::set_selection

  Example:

    $my_window_manager create_window_manager(terminal)
    $window_id create_window(&my_window_manager "My Window")
    set_selection &my_window_manager window_id 1 1 10 5
    unset_selection &my_window_manager window_id
