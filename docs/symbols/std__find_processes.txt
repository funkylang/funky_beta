# std::find_processes (an I/O-function)

  finds the process IDs of all processes with the specified name

  Parameter:
    name: the name of the processes to find

  Results:
    a list of process IDs

  **This function must be called with I/O-access rights!**

  The function searches for all processes with the specified name in the
  "/proc" directory.

  Topic: Input-Output

  See also: std::find_first_process, std::get_process_name, std::which
