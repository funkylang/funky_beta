# std::enlarge (a polymorphic function)

  enlarges a window

  Parameters:
    window_manager: a window manager
    id: the id of the window to enlarge

  Result:
    window_manager: the updated window manager

  If no id is given then the focused window is enlarged.

  If the window is minimized then it is enlarged.

  If the window is not maximized then it is maximized.

  If the window is maximized then it is minimized.

  Topic: Window Manager

  See also: std::shrink, std::create_window_manager,
            std::create_window

  Example:

    $wm create_window_manager(terminal)
    $id create_window(&wm "My Window" CONTENT "Hello, World!")
    enlarge &wm id
