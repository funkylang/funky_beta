# std_types::object/std::pipe (an I/O-method)

  creates a pipe of processes that are connected via their standard input and
  output streams

  Parameters:
    args*: the arguments to the pipe

  Results:
    the output of the last process in the pipe

  **Attention**: This function must be called with I/O-access rights!

  The arguments are either

  * a string that is sent to the first process in the pipe

  * a file descriptor that is used as the input for the first process in the
    pipe

  * a function that is called as a process in the pipe

  * a file descriptor that is used as the output for the last process in the
    pipe

  The output of the last process in the pipe is returned.

  Topic: Input-Output

  See also: std::pipe_commands
