# std_types::object/std::pad_left (a method)

  extends a string to the specified width

  Parameters:
    self: the expression that is converted into a string
    n: the length of the resulting string
    pad: the pattern used to fill up the string

  Result:
    string: the extended string

  This function creates a string representation of *self* which is at least
  *n* characters long. If it would be shorter than it is padded to the left
  with the string pattern stored in *pad*.

  The default value for *pad* is a single space.

  Topic: Strings

  See also: std_types::object/std::pad_right

  Example:

    $str "123"
    $padded_str pad_left(str 5 "0")
    println! padded_str

  Output:

    00123
