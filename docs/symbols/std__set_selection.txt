# std::set_selection (a polymorphic function)

  sets the selection of a window

  Parameters:
    window_manager: a window manager
    id: the id of the window
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    width: width of the area
    height: height of the area

  Result:
    window_manager: the updated window manager

  If the id is undefined then the focused window is used.

  Topic: Window Manager

  See also: std::unset_selection

  Example:

    $my_window_manager create_window_manager(terminal)
    $window_id create_window(&my_window_manager "My Window")
    set_selection &my_window_manager window_id 1 1 10 10
