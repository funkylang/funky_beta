# ai::piece_table_of (a polymorphic function)

  returns a hash table containing the mapping between pieces and tokens

  Parameter:
    self: the AI model

  Result:
    hash_table: a hash table containing the mapping between pieces and tokens

  The hash table contains the following keys:

  * "begin_of_stream": the token for the beginning of a stream
  * "end_of_stream": the token for the end of a stream
  * "classification": the token for a classification token
  * "separator": the token for a separator
  * "newline": the token for a newline
  * "end_of_text": the token for the end of text
  * "prefix": the token for a prefix
  * "suffix": the token for a suffix
  * "middle": the token for a middle token

  The values in the hash table are the corresponding tokens.

  Topic: Deprecated AI

  See also: ai::tokenize, ai::detokenize

  Example:

    $modelname "sauerkrautem"
    $model ai::load_ai_model! $modelname
    $piece_table ai::piece_table_of($model)
