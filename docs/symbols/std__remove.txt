# std::remove (a polymorphic function)

  removes some item(s) from a collection

  Parameters:
    collection: the collection from which to remove the item(s)
    item: the item to remove

  Result:
    collection: the updated collection

  Removes the first occurrence of the specified item from the collection.

  If the item is not found in the collection then the collection is returned
  without alteration.

  Topic: Collections

  See also: std::add

  Example:

    $my_list create_list(1 2 3 4 5)
    remove &my_list 3
    dump `my_list

  Output:

    my_list: list <5>
      1
      2
      4
      5
