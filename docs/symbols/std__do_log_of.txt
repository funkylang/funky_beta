# std::do_log_of (a polymorphic function)

  the do_log_of function is used to determine if logging is enabled for a
  specific object.

  Parameter:
    self: the object for which to check logging status

  Result:
    true if logging is enabled for the object, false otherwise

  Topics Input-Output Events

  Example:

    $io std_types::io # create an input-output object
    enable_logging &io 3 # enable logging for the io object
    if
      do_log_of &io:
	log &io 1 "Logging is enabled!"
      :
	log &io 1 "Logging is disabled!"
