# std_types::mersenne_twister/std::next_raw_value (a method)

  generates a pseudo random number

  Parameters:
    self: the pseudo random number generator

  Result:
    number: a pseudo random number

  The pseudo random number generator uses the Mersenne Twister algorithm
  to generate pseudo random numbers.
