# std::without_prefix (a polymorphic function)

  remove a prefix from a string

  Parameters:
    string: the string to process
    prefix: the prefix to remove

  Result:
    string: the string without the prefix

  Returns the string without the specified prefix if the string starts with
  the prefix. Otherwise the string is returned without alteration.

  Topic: Strings

  See also: std::without_suffix, std::has_prefix, std::has_suffix,
            std::behind, std::truncate_from

  Example:

    $path "/home/user"
    $user_name path .without_prefix. "/home/"
    println! user_name

    Output:

    user
