# std::get_process_name (an I/O-function)

  gets the name of a process

  Parameters:
    pid: the process ID

  Results:
    the name of the process or <std::undefined>

  **Attention**: This function must be called with I/O-access rights!

  The function reads the command line of the specified process from the
  "/proc" directory and returns the name of the process.

  Topic: Input-Output
