# std_types::some (a type)

  the prototype grammar object to match zero or more occurrences of something

  This is a grammar object that matches zero or more occurrences of the given
  expression.

  Topics: Grammar Objects

  See also: std::some

  Example:

    $str "123abc"
    match some(DIGIT) str $len
    println! len

  Output:

    3
