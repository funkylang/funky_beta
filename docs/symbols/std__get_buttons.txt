# std::get_buttons (a polymorphic function)

  returns the additional buttons of a window

  Parameters:
    window_manager: a window manager
    id: the id of the window

  Result:
    buttons: the additional buttons of the window

  If no id is given then the id of the focused window is used.

  Topic: Window Manager

  See also: std::set_buttons

  Example:

    $my_window_manager create_window_manager(terminal)
    $my_window create_window(my_window_manager "My Window"
      CONTENT "Hello, World!"
      BACK_BUTTON true
      FORWARD_BUTTON true
    )
    $buttons get_buttons(my_window_manager id_of(my_window))
    # buttons: list(BACK_BUTTON FORWARD_BUTTON)
