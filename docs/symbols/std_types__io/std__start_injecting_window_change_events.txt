# std_types::io/std::start_injecting_window_change_events (a method)

  starts injecting window change events

  Parameters:
    self: the I/O-object

  Result:
    self: the updated I/O-object

  This starts injecting window change events into the I/O-object.

  When the size of the current terminal window changes an appropriate control
  sequence will be injected into the *stdin* data stream.

  Topics: I/O, Events

  See also: std_types::io/std::stop_injecting_window_change_events
