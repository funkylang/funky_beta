# std_types::io/std::join_process (a method)

  joins a process

  Parameters:
    io: the I/O-object
    pid: the process id to wait for

  Result:
    io: the updated I/O-object

  This joins the specified process. The I/O-object will generate a
  <std::PROCESS_JOINED> event when the process has finished.

  Topic: Input-Output Events

  See also: std_types::io/std::close
