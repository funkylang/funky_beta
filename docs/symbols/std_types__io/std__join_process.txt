# std_types::io/std::join_process (a method)

  joins a process

  Parameters:
    self: the I/O-object
    pid: the process id

  Result:
    self: the updated I/O-object

  This joins the specified process. The I/O-object will generate a
  <std::PROCESS_JOINED> event when the process has finished.
