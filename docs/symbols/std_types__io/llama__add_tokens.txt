# std_types::io/llama::add_tokens (a method)

  asynchronously adds tokens to a sequence

  Parameters:
    io: the I/O-object to write to
    model: the model to use
    sequence_id: the sequence to add the tokens to
    tokens: the tokens to add

  Result:
    io: the updated I/O-object

  A job is added to the I/O-object to add the tokens to the specified sequence.

  The tokens are added to the end of the sequence.
