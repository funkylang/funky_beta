# std_types::io/std::close (a method)

  closes a file descriptor

  Parameters:
    self: the I/O-object
    fd: the file descriptor

  Result:
    self: the updated I/O-object

  This closes the specified file descriptor. The I/O-object will generate a
  <std::CLOSED> event when the file descriptor is closed.

  Topics: I/O, Events
