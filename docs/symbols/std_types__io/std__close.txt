# std_types::io/std::close (a method)

  closes a file descriptor

  Parameters:
    io: the I/O-object
    fd: the file descriptor

  Result:
    io: the updated I/O-object

  Instructs the event system (<std::get_events>) to close the specified file
  descriptor. A <std::CLOSED> event will be created when the file descriptor is
  closed.

  Topic: Input-Output Events
