# std_types::io/std::start_catching_signals (a method)

  starts catching signals

  Parameters:
    self: the I/O-object
    signals: the signals to catch

  Result:
    self: the updated I/O-object

  This starts catching the specified signals.

  When one of the catched signals arrives a <std::SIGNAL> event will be created.
