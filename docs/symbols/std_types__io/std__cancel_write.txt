# std_types::io/std::cancel_write (a method)

  cancels the pending write operation for the file descriptor

  Parameters:
    io: the I/O-object
    fd: the file descriptor

  Results:
    io: the updated I/O-object
    text: unwritten text

  This removes a pending write operation for the specified file descriptor from
  the I/O-object.

  The part of the text (data) that has not yet been written is returned as the
  second result.

  Topics: I/O, Events

  See also: std_types::io/std::write
