# std_types::io/std::call_with_input (a method)

  calls a function with input asynchronously

  Parameters:
    io: the I/O-object
    id: a caller supplied id to identify the I/O-job
    feed: the input to pass to the function
    cmd: the function to call
    args: the arguments to pass to the function
    context: some caller supplied context to attach to the I/O-job (optional)

  Result:
    the updated I/O-object

  This function adds a job for calling a function with input to the I/O-object.
  The job is executed asynchronously within <std::get_events>.

  When the function has been executed successfully, a <std::JOB_COMPLETED>-event
  is generated.

  If the function cannot be executed, a <std::JOB_FAILED>-event is generated.

  Topic: Input-Output Events

  See also: std::call, std::execute, std::system, std::pipe, std::pipe_commands
