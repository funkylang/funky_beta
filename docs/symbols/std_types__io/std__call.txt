# std_types::io/std::call (a method)

  calls an external command asynchronously

  Parameters:
    io: the I/O-object
    id: a caller supplied id to identify the I/O-job
    cmd: the command to call
    args: the arguments to pass to the command
    context: some caller supplied context to attach to the I/O-job (optional)

  Result:
    the updated I/O-object

  This function adds a job for calling an external command to the I/O-object.
  The job is executed asynchronously within <std::get_events>.

  When the command has been executed successfully, a <std::JOB_COMPLETED>-event
  is generated.

  If the command cannot be executed, a <std::JOB_FAILED>-event is generated.

  Topic: Input-Output

  See also: std::call, std::extern, std::execute, std::system, std::pipe,
	    std::pipe_commands
