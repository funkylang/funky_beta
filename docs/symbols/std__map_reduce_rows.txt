# std::map_reduce_rows (a polymorphic function)

  applies a map and a reduce function to each row of a table

  Parameters:
    self: the table

    map_function: a function that is applied to each row

    reduce_function: a function that is applied to the results of the map
      function

  Result:
    a new table with the same number of rows as the original table

  The map function takes a row as input and returns a list of values.

  The reduce function takes two values as input and returns a single value.

  The reduce function is applied to the results of the map function for each
  row.

  Topic: Collections

  See also: std::map, std::reduce, std::map_reduce_columns

  Example:

    $my_table create_table(
      "name" string
      "age" integer
    )
    $my_table.append(
      ("John", 25)
      ("Mary", 30)
      ("Peter", 20)
    )
    $result std::map_reduce_rows my_table
      : (row) -> list(row.age)
      : (left right) -> left + right

    println! result
