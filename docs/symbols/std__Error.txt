# std::Error (an I/O-function)

  displays an error message and terminates the program

  Parameters:
    args*: the arguments to convert to a string

  Results:
    none

  **Attention**: This function must be called with I/O-access rights!

  The function converts its arguments to a single string, appends a line feed
  character if necessary and writes this string to the standard error output.

  Afterwards the program terminates with exit code <std::EXIT_FAILURE>.

  Topic: Errors

  See also: std::exit, std::eprint, std::eprintln

  Example:

    Error! "An error occurred"

  Output:

    ERROR: An error occurred
