# std::fill_rectangle (a polymorphic function)

  fills a rectangle with the current draw colour

  Parameters:
    screen: the screen to fill
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    width: width of the rectangle
    height: height of the rectangle

  Result:
    screen: the updated screen

  The area is filled using the current draw colour. Use
  <std::set_draw_colour> to change the current draw colour.

  The area is clipped at the screen's borders.

  Topic: Terminal

  See also: std::clear, std::draw_text, std::set_draw_colour
