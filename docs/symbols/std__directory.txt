# std::directory (an I/O-function)

  returns a list of entries in a directory

  Parameters:
    options*: options to control the behavior of the function
    pathname: the path to the directory

  Result:
    a list of directory entries

  **Attention**: This function must be called with I/O-access rights!

  The function opens the specified directory, reads all entries and returns
  them as a list.

  The function supports the following options:

    - SORT: if set to true, the entries are sorted by name

  Topic: Input-Output

  See also: std::file_exists, std::is_a_directory, std::stat, std::readdir

  Example:

    directory! $entries "/usr/bin"
    for_each entries
      : (entry)
	println! name_of(entry)
	next!
      pass
