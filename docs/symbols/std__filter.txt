# std::filter (a polymorphic function)

  returns a collection containing all the items that fullfill the
  filter-condition

  Parameters:
    self: the collection to filter
    function: a function that takes an item of the collection as input and
      returns *true* if the item should be included in the result, *false*
      otherwise

  Result:
    collection: a new collection containing only the items for which the
      function returned *true*

  Topic: Collections

  See also: std::map, std::find_first

  Example:

    $my_list create_list(1 2 3 4 5 6)
    $filtered_list filter &my_list (x) -> x % 2 == 0

    # filtered_list will contain: (2 4 6)
