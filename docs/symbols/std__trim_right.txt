# std::trim_right (a function)

  removes trailing whitespace

  Parameters:
    self: the string to process
    test: a function that checks whether a character is a whitespace character

  Result:
    the string without any trailing whitespace

  The function returns a string which is the specified string without any
  trailing whitespace.

  Topic: Strings

  See also: std::trim, std::trim_left

  Example:

    $str "   text   "
    $trimmed_str trim_right(str)
    println! '*' trimmed_str '*'

  Output:

    *   text*
