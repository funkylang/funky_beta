# std::pselect (a builtin I/O-function)

  waits for one or more file descriptors to become ready

  Parameters:
    read_descriptors: a list of file descriptors to wait for readability
    write_descriptors: a list of file descriptors to wait for writability
    except_descriptors: a list of file descriptors to wait for exceptions
    timeout: the maximum time to wait in seconds

  Results:
    signals: a list of signals that have been caught
    read_descriptors: a list of file descriptors that are ready for reading
    write_descriptors: a list of file descriptors that are ready for writing
    except_descriptors: a list of file descriptors that have exceptions

  Topic: Input-Output

  See also: std::select, std::poll

  Example:

    pselect! [STDIN_FILENO] [] [] 10 $signals $read_descriptors $write_descriptors $except_descriptors
