# std::concatenate_non_empty (a function)

  concatenates two or more collections

  Parameters:
    separator: the separator to use between the collections
    collections: the collections to concatenate

  Result:
    the concatenated collection

  The collections are concatenated in the order they are specified.

  If a collection is empty then it is ignored.

  If all collections are empty then an empty collection is returned.

  Topic: Lists

  See also: std::concatenate

  Example:

    $str "This is a very long string that should be wrapped onto multiple lines"
    $lines wrap_words(str 30)
    dump! 2 `lines

  Output:

    lines:
      list
        "This is a very long string"
        "that should be wrapped"
        "onto multiple lines"
