# std_types::string/std::matches (a method)

  checks whether the string matches the specified expression

  Parameters:
    self: the string to check
    expr: the expression to match

  Result:
    match: <true> if the string matches the expression

  The string matches the expression if the expression matches the start of the
  string and the length of the match is equal to the length of the string.

  Topic: Strings
