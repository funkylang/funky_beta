# std_types::string/std::to_number (a method)

  converts the whole string into a number value

  Parameters:
    self: the string to convert

  Result:
    number: the number representation of the string

  If the complete string can be converted into a number then the
  resulting number (integer or real) is returned.

  If the conversion fails or does not consume the complete string then
  <undefined> is returned.

  Topic: Strings

  See also: std::to_integer, std::to_real, std::to_string

  Example:

    $str "123.456"
    $value str.to_number
    println! "value: " value

  Output:

    value: 123.456

  Example:

    $str "123.456, 789"
    $value str.to_number
    dump! `value

  Output:

    value: <undefined>
