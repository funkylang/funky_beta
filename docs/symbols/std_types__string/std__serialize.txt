# std_types::string/std::serialize (a method)

  returns a string representation of the string

  Parameters:
    self: the string to serialize
    indent: the indentation level

  Result:
    string: the serialized string

  The string is serialized in a way that it can be deserialized using
  <std_types::string/std::deserialize>.

  Topic: Strings

  See also: std_types::string/std::deserialize

  Example:

    $str "Hello World"
    $serialized_str serialize(str)
    dump! `serialized_str

  Output:

    serialized_str: "@quot;Hello World@quot;@0xa;"
