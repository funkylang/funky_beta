# std_types::string/ai::load_ai_model (an I/O-method)

  loads the specified AI model

  Parameters:
    name: the name of the model
    options*: the options for the model

  Results:
    the model object

  The model is loaded from the AI server at the specified address and port
  number.

  The options are:

  * ADDRESS: the address of the AI server (default: "127.0.0.1")
  * PORT_NO: the port number of the AI server (default: 8080)
  * UUID: the UUID of the model (default: undefined)
  * ai::CONTEXT_SIZE: the context size of the model (default: undefined)

  Topic: Deprecated AI

  See also: ai::load_ai_model
