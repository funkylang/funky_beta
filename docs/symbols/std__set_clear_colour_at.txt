# std::set_clear_colour_at (a polymorphic function)

  sets the clear colour of a character at a given position in a screen

  Parameters:
    screen # the screen object
    x: x-coordinate of the character
    y: y-coordinate of the character
    colour: the new clear colour

  Result:
    screen: the updated screen

  The clear colour is used when clearing the screen or parts of it.

  Topic: Terminal

  See also: std::set_draw_colour_at, std::get_clear_colour_at

  Example:

    $my_screen create_screen(terminal 40 25)
    set_clear_colour &my_screen 10 5 RED
    # now the character at position (10,5) has a red clear colour
