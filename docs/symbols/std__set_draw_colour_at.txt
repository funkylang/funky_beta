# std::set_draw_colour_at (a polymorphic function)

  sets the draw colour of a character at a given position

  Parameters:
    screen # some kind of screen
    x: x-coordinate of the character
    y: y-coordinate of the character
    colour: the new draw colour

  Result:
    screen: the updated screen

  The draw colour is used when drawing text on the screen.

  Related topics: Terminal

  Example:

    $my_screen create_screen(terminal 40 25)
    set_draw_colour &my_screen RED
    draw_text &my_screen 11 6 "Hello, world!"
