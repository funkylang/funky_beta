# html::TR (a function)

 generates a TR element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the TR element

  The function is used as a block function, with the content placed between the
  TR tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      TR
	: TH "Name"
	: TH "Age"

  Output:

    <tr>
    <th>Name</th>
    <th>Age</th>
    </tr>
