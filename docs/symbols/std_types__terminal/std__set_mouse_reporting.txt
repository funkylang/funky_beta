# std_types::terminal/std::set_mouse_reporting (a method)

  sets the mouse reporting mode

  Parameters:
    self: the terminal
    mouse_reporting: the mouse reporting mode

  Result:
    terminal: the updated terminal

  The mouse reporting mode can be one of the following:

  * <std::DISABLED>: mouse reporting is disabled
  * <std::CLICKS>: mouse clicks are reported
  * <std::DRAG>: mouse drags are reported
  * <std::MOVE>: mouse movements are reported

  Topic: Terminal

  See also: std_types::terminal/std::get_mouse_reporting
