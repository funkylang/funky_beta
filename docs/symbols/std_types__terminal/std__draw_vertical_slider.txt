# std_types::terminal/std::draw_vertical_slider (a method)

  draws a vertical slider

  Parameters:
    self: the terminal to draw on
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    width: width of the slider
    height: height of the slider
    total: the number of total items
    current: the number of currently visible items
    pos: the index of the first visible item (one-based)

  Result:

  Draws a vertical slider on the specified terminal.
