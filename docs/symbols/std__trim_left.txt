# std::trim_left (a function)

  removes leading whitespace

  Parameters:
    self: the string to process
    test: a function that checks whether a character is a whitespace character

  Result:
    the string without any leading whitespace

  The function returns a string which is the specified string without any
  leading whitespace.

  Topic: Strings

  See also: std::trim, std::trim_right

  Example:

    $str "   text   "
    $trimmed_str trim_left(str)
    println! '*' trimmed_str '*'

  Output:

    *text   *
