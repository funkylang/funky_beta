# std::realpath (a builtin I/O-function)

  returns the canonicalized absolute pathname

  Parameters:
    pathname: the path to canonicalize

  Result:
    path: the canonicalized absolute pathname

  On success a <std_types::string> is returned, on failure an appropriate
  error-object is returned.

  This function needs I/O-access rights.

  Topic: POSIX

  See also: std::chdir, std::getcwd

  Example:

    realpath! $path "/etc/../etc/passwd"
    println! path

  Output:

    /etc/passwd
