# std::draw_thick_line (a polymorphic function)

  draws a thick line

  Parameters:
    screen: some kind of screen
    x1: x-coordinate of the start point
    y1: y-coordinate of the start point
    x2: x-coordinate of the end point
    y2: y-coordinate of the end point

  Result:
    screen: the updated screen

  The line is drawn using the current draw colour. Use
  <std::set_draw_colour> to change the current draw colour.

  The line is clipped at the screen's borders.

  Topic: Terminal

  See also: std_types::screen/std::draw_thick_line, std::draw_line,
            std::draw_text, std::set_draw_colour

  Example:

    $my_screen create_screen(terminal 40 25)
    set_draw_colour &my_screen RED
    draw_thick_line &my_screen 11 6 20 15
