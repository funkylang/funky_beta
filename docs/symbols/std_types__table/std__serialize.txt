# std_types::table/std::serialize (a method)

  returns a machine readable string representation of the table

  Parameters:
    self: the table to serialize
    indent: the indentation level

  Result:
    string: the serialized table

  Topic: Tables

  See also: std::serialization_tag_of, std::to_string

  Example:

    $tools
      hash_table
	"hammer" = 47
	"saw" = 11
	"screw driver" = 8
    println! serialize(tools)

  Output:

    hash_table <6>
    "screw driver"
    8
    "saw"
    11
    "hammer"
    47
