# std_types::table/std::add (a method)

  adds some items to the table

  Parameters:
    self: the table to add the items to
    pairs: the items to add

  Result:
    table: the updated table

  The items are added to the table.

  If a key is already contained in the table then the value is replaced.

  Topic: Tables

  See also: std_types::table/std::remove

  Example:

    $table empty_table
    add table
      "John" = 42
      "Mary" = 25
      "Irene" = 49
    dump! `table

  Output:

    table: table <6>
      "Mary"
      25
      "Irene"
      49
      "John"
      42
