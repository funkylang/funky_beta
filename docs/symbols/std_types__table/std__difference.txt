# std_types::table/std::difference (a method)

  computes the difference of two tables

  Parameters:
    self: the base table
    other: the table of entries to remove
    difference_function: the function used to compute the difference

  Result:
    table: the difference of the two tables

  The difference of two tables is a table that contains all entries of the
  base table that are not contained in the other table.

  The difference function is applied to each entry of the base table that is
  also contained in the other table.

  Topic: Tables

  See also: std_types::table/std::merge
