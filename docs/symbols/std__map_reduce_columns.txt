# std::map_reduce_columns (a polymorphic function)

  applies a map and a reduce function to each column of a table

  Parameters:
    self: the table
    map_function: a function that maps a single item to a new item
    reduce_function: a function that combines two items into a single item

  Result:
    a new table with the same number of rows as the original table

  The map function is applied to each item in each column of the table. The
  reduce function is then applied to the results of the map function for each
  column.

  Topic: Collections

  See also: std::map_reduce_rows, std::map, std::reduce

  Example:

    $my_table create_table(
      "Name" "Age" "City"
      "John" 30 "New York"
      "Jane" 25 "London"
      "Peter" 40 "Paris"
    )
    $result std::map_reduce_columns my_table
      (
        (item) item * 2
      )
      (
        (a b) a + b
      )
    dump! `result
