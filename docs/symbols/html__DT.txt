# html::DT (a function)

  generates a DT element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the DT element

  The function is used as a block function, with the content placed between the
  DT tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      DT
	"Name"

  Output:

    <dt>Name</dt>
