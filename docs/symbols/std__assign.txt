# std::assign (a function)

  returns all its arguments unaltered

  Parameters:
    args*: the arguments to return

  Results:
    the arguments passed to the function

  This function is used to assign multiple values to multiple variables. Because
  all arguments are evaluated first it implements a parallel assignment and can
  be used to swap values between variables.

  Topic: Primitives

  See also: std::do, std::pass, std::chain

  Example:

    # swap

    assign
      !x y
      !y x
