# std::get_size (a polymorphic function)

  returns the size of a window

  Parameters:
    wm: a window manager
    id: the id of the window (optional)

  Result:
    width: the width of the window
    height: the height of the window

  If no id is given then the size of the focused window is returned.

  Topic: Window Manager

  See also: std::get_xy, std::get_focused_window, std::get_window_at

  Example:

    $wm create_window_manager(terminal)
    $id create_window(wm CONTENT "Hello, World!")
    $width $height get_size(wm id)
