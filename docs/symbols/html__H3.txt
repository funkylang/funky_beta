# html::H3 (a function)

 generates a H3 element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the H3 element

  The function is used as a block function, with the content placed between the
  H3 tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      H3
	"My great Homepage"
