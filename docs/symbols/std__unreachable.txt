# std::unreachable (a function)

  indicates that the program flow reached a point that should never be reached

  Parameter:
    none

  Result:
    an error object

  Calls to this function can be placed in the code at positions that should not
  never be reached.

  An appropriate error object is created and returned.

  Topic: Errors

  See also: std::Error
