# std::create_window (a polymorphic function)

  creates a new window

  Parameters:
    window_manager: the window manager to create the window in
    title: the title of the window
    options*: optional options for the window

  Result:
    window_manager: the updated window manager
    window_id: the id of the newly created window

  The following options are supported:

    CONTENT: the content of the window
    CLEAR_COLOUR: the clear colour of the window
    HORIZONTAL_SCROLLBAR: whether the window has a horizontal scrollbar
    VERTICAL_SCROLLBAR: whether the window has a vertical scrollbar

  Topic: Window Manager

  See also: std::create_window_manager, std_types::window_manager/update

  Example:

    $my_window_manager create_window_manager(terminal)
    $my_window_id create_window(my_window_manager "My Window" CONTENT: "Hello, World!")
