# std::blue_component_of (a polymorphic function)

  extracts the blue component of a character

  Parameter:
    chr: a character

  Result:
    blue: the blue component of the character

  The blue component is the least significant byte of the character.

  Topic: Colours

  See also: std::red_component_of, std::green_component_of, std::rgb_character

  Example:

    $my_character create_character(0xff00ff)
    $blue std::blue_component_of(my_character)
    println! blue # prints 255
