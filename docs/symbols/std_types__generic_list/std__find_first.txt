# std_types::generic_list/std::find_first (a method)

  returns the first item of a collection that fullfills the condition

  Parameters:
    self: the collection to search
    test: the condition to fullfill

  Results:
    item: the first item that fullfills the condition
    index: the index of the first item that fullfills the condition

  If the collection contains no item that fullfills the condition then
  <undefined> is returned.

  If the caller requests two result values then the index of the first
  item that fullfills the condition is also returned.
