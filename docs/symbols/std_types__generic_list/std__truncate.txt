# std_types::generic_list/std::truncate (a method)

  truncates a list to the specified number of items

  Parameters:
    self: the list to truncate
    n: the maximum length of the result list

  Result:
    list: the truncated list

  If the length of the list is greater than *n* then the list is truncated
  to the first *n* items.

  If the length of the list is less than or equal to *n* then the list is
  returned unchanged.

  Topic: Lists

  See also: std::truncate
