# std::initialized_uint32_array (a builtin function)

  constructs an initialized array

  Parameters:
    dimension_1: the size of the first dimension
    ...
    dimension_n: the size of the n-th dimension
    initial_value: the initial value for all array items

  Result:
    array: the array

  Topic: Arrays

  See also: std::uint32_array, std_types::uint32_array/new

  Example:

    # create a 3x4 array of 32-bit unsigned integers
    # with all elements initialized to 0
    array = std::initialized_uint32_array(3, 4, 0)

    # create a 3x4 array of 32-bit unsigned integers
    # with all elements initialized to 1
    array = std::initialized_uint32_array(3, 4, 1)
