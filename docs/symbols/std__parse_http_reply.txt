# std::parse_http_reply (a function)

  parses a HTTP reply

  Parameters:
    buf: the buffer containing the reply

  Result:
    data: the data part of the reply
    status: the HTTP status code
    header: the HTTP header
    buf: the remaining data in the buffer

  The function parses a HTTP reply contained in the buffer.

  Topic: Network Clients and Servers

  See also: std::http_request

  Example:

    $reply "
      HTTP/1.1 200 OK@cr;
      Content-Type: text/html@cr;
      Content-Length: 12@cr;
      @cr;
      Hello, World!
    parse_http_reply &reply $status $header $data
    dump! 2 `status `header `data `reply
