# std::get_options (a function)

  fetches the options of a function call

  Parameters:
    options: the options passed to the function
    templates*: the option templates

  Result:
    the values of the options

  This function returns a variable number of results - one for each option
  template. Options which do not match any template are ignored.

  Each template has the form

  OPTION = *default value*

  To make the code more readable it is recommended to interleave the option
  templates and the results.

  Topics: Parameters

  See also: std::program_parameters, std::extract_options

  Example:

    $std_types::screen/create_screen:
      (
	screen
	options*
      )
      get_options options
	DRAW_COLOUR = BLACK $draw_colour
	CLEAR_COLOUR = WHITE $clear_colour
	TEXT_ATTRIBUTES = NORMAL $text_attributes
      ...
