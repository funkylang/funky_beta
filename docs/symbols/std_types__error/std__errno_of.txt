# std_types::error/std::errno_of (a builtin method)

  the errno associated with the error object

  This field is used in conjunctions with the "IO ERROR" error.

  Topic: Errors

  See also: std_types::error, std::error_category_of,
	    std::error_message_text_of, std::error_details_of,
	    std::failed_attribute_of

  Example:

    load! $buf "some-non-existing-file"
    on buf.is_an_error:
      dump! `errno_of(buf)

  Output:

    errno_of(buf): object <0>
