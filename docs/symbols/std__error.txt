# std::error (a builtin function)

  creates an error object

  Parameters:
    category?: the category of the error (default: <APPLICATION_ERROR>)
    message_text: the error message
    details?: additional details (default: <undefined>)

  Result:
    the error object

  This function constructs an (application specific) error object. There are no
  restrictions about the kind of object that *details* may hold. It's up to the
  application program to handle this information when displaying the error
  message.

  Topic: Errors

  See also: std::RUNTIME_ERROR, std::IO_ERROR, std::APPLICATION_ERROR,
	    std_types::error, std::error_category_of,
	    std::error_message_text_of, std::error_details_of, std::errno_of
