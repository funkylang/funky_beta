# std::which (an I/O-function)

  finds the full path of an executable file

  Parameters:
    name: the name of the executable file

  Results:
    the full path of the executable file or <std::undefined>

  **Attention**: This function must be called with I/O-access rights!

  The function searches for the specified executable file in the directories
  listed in the PATH environment variable.
