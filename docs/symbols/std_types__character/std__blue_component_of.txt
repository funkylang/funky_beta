# std_types::character/std::blue_component_of (a method)

  returns the blue component of the given rgb-character

  Parameters:
    chr: a rgb-character

  Result:
    the blue component (0-255)

  A colour is represented as a 24-bit value (256 shades of red, green and blue)
  and stored as a character. The bits 0..7 of the character are used to store
  the blue component.

  Topic: Colours, Terminal

  See also: std::rgb_character, std::text_attributes_of

  Example:

    $BLUE rgb_character(0 0 0xff)
    $blue blue_component_of(BLUE)
