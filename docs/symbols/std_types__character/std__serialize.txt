# std_types::character/std::serialize (a method)

  serializes the character

  Parameter:
    self: the character to serialize
    indent: the indentation depth in character positions

  Result:
    str: the serialization of the character

  Topic: Strings

  The optional indent is obeyed.

  Example:

    $str serialize('a')
    println! str

  Output:

    'a'

  Example:

    $str serialize('a' 2)

      'a'
