# std_types::character/std::match (a method)

  matches the start of a string with the character

  Parameters:
    self: the character to match with
    str: the string which's start is matched

  Results:
    len: 1 if the character is the first character of the string

  If the match is unsuccessful, then *len* is <undefined>.

  Topic: Strings

  See also: std_types::character/std::match_back,
            std_types::character/std::search

  Example:

    $str "abcdef"
    $len match('a' str)
    dump! `len

  Output:

    len: 1

  Example:

    $str "abcdef"
    $len match('z' str)
    dump! `len

  Output:

    len: <undefined>
