# std::readdir (a builtin I/O-function)

  reads the next directory entry

  Parameters:
    dir: a directory stream

  Result:
    entry: the next directory entry

  On success a <std_types::dirent> is returned, on failure an appropriate
  error-object is returned.

  This function needs I/O-access rights.

  Topic: POSIX

  See also: std::opendir, std::closedir

  Example:

    opendir! $dir "/bin"
    loop:
      readdir! $entry dir
      if
        entry.is_an_error
        break
        :
          if
            type_of(entry) == SYMBOLIC_LINK: # check file type
              println! name_of(entry)
              next!
            next

    lzip-compressor
    a2ping
    ...
