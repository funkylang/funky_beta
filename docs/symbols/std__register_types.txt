# std::register_types (a function)

  registers a list of types for deserialization

  Parameters:
    deserializers: a hash table that maps serialization tags to types
    types: a list of types

  Result:
    deserializers: the updated hash table

  Topic: Serialization

  See also: std::register_type, std::deserialize

  Example:

    $deserializers empty_hash_table
    register_types(deserializers [list hash_table])
    dump! 2 `deserializers

  Output:

    deserializers: hash_table <2>
      "list"
      list
      "hash_table"
      hash_table
