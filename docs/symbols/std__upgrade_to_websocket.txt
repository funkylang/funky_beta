# std::upgrade_to_websocket (a function)

  Upgrades a TCP connection to a WebSocket connection.

  Parameters:
    io: the I/O object
    fd: the file descriptor of the TCP connection

  Result:

  This function sends a WebSocket handshake request to the server.

  Topic: Network Clients and Servers

  See also: std::handle_web_client_events

  Example:

    $io new(std_types::io)
