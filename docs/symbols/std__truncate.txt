# std::truncate (a polymorphic function)

  truncates a collection to the specified number of items

  Parameters:
    self: the collection to truncate
    n: the maximum length of the result collection

  Result:
    a truncated collection

  The result collection contains at most *n* items.

  Topic: Collections

  See also: std::append, std::insert, std::remove

  Example:

    $my_list create_list(1 2 3 4 5)
    $truncated_list std::truncate(my_list 3)
    println! truncated_list

    prints

    1 2 3
