# std::pad_top (a polymorphic function)

  pads a collection with a specified value

  Parameters:
    collection: the collection to pad
    value: the value to use for padding
    length: the desired length of the collection

  Result:
    collection: the padded collection

  The collection is padded at the beginning.

  Topic: Collections

  See also: std::pad_bottom, std::pad_left, std::pad_right

  Example:

    $my_list create_list(1 2 3)
    pad_top &my_list 0 5

    Output:

    my_list: list <5>
      0
      0
      1
      2
      3
