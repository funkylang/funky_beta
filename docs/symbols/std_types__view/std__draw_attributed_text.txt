# std_types::view/std::draw_attributed_text (a method)

  draws attributed text at the given position in the view

  Parameters:
    view: the view to draw on
    screen: the screen object
    x: the x coordinate (1-based)
    y: the y coordinate (1-based)
    attributed_text: the attributed text to draw

  Topic: Terminal, IO

  See also: std::draw_text, std::clear, std::invert,
	    std::draw_shadow, std::copy

  Example:

    $attributed_text ""
    push &attributed_text "O"
    push &attributed_text BLUE
    push &attributed_text WHITE
    push &attributed_text "K"
    push &attributed_text BLUE
    push &attributed_text WHITE
    draw_attributed_text view &terminal 1 1 attributed_text
      # draws the text "OK" in BLUE on WHITE background
