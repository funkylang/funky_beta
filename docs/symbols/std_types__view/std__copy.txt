# std_types::view/std::copy (a method)

  copies a region of the source to the destination view

  Parameters:

    destination: the destination screen
    source: the source screen
    x: the x coordinate (1-based) in the destination view (default: 1)
    y: the y coordinate (1-based) in the destination view (default: 1)
    sx: the x coordinate (1-based) in the source view (default: 1)
    sy: the y coordinate (1-based) in the source view (default: 1)
    width: the width of the region to copy (optional)
    height: the height of the region to copy (optional)

  Result:
    destination: the updated destination screen
