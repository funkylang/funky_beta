# llama::copy_sequence (a polymorphic function)

  copies a sequence

  Parameters:
    model: the model
    sequence_id: the id of the sequence to copy
    template_id: the id of the sequence to use as a template
    token_count: the number of tokens to copy

  Result:
    model: the updated model

  Copies the sequence with the given id from the given model. The sequence
  is copied to the sequence with the given template id. The number of tokens
  to copy is given by the *token_count* parameter.
