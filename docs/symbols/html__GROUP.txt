# html::GROUP (a function)

  groups multiple elements together

  Parameters:
    items*: the elements to be grouped

  Result:
    a string containing the grouped elements

  The function is used to group multiple elements together without adding any
  additional HTML tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      GROUP
	: H1 "My great Homepage"
	: P "This is my great homepage."

  Output:

    <h1>My great Homepage</h1>
    <p>This is my great homepage.</p>
