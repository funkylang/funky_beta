# std::program_parameters (an I/O-function)

  parses the program parameters

  Parameters:
    templates*: the accepted options and arguments

  Result:
    the parsed parameters

  **This function must be called with IO-access rights.**

  This function parses the program parameters according to the given templates.

  To make the code more readable it is recommended to interleave the option
  templates and the results.

  Topics: Parameters

  See also: std::extract_options

  Example:

    program_parameters!
