# std::match_back (a polymorphic function)

  matches the end of a string with the expression

  Parameters:
    self: the object from which to get the expression
    str: the string which's end is matched

  Result:
    len: the length of the match if the match is successful

  If the match is unsuccessful, then *len* is <undefined>.

  Topic: Grammar Objects

  See also: match, search

  Example:

    $str "12345abcdef"
    match_back many(DIGIT) str $len
    dump! `len

    Output:

      len: 6
