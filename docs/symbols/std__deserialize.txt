# std::deserialize (a function)

  deserializes a machine readable string representation of an object

  Parameters:
    text: the string to deserialize

  Result:
    the deserialized object

  The string must have been created by std::serialize.

  Topic: Serialization

  See also: std::serialize, std::deserializers, std::register_type,
            std::register_types

  Example:

    $str serialize(42)
    println! str
    $obj deserialize(str)
    println! obj

  Output:

    42
    42
