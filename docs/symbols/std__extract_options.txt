# std::extract_options (a function)

  extracts options from a list

  Parameters:
    arguments: the list of arguments
    templates*: the option templates

  Result:
    the remaining arguments and the values of the options

  This function returns a variable number of results - a list containing the
  remaining arguments and one for each option template.

  Topics: Parameters

  See also: std::program_parameters, std::get_options

  Example:

    $std::horizontal_group:
      (
	widgets*
      )
      extract_options &widgets
	TOP = false $do_align_top
	BOTTOM = false $do_align_bottom
	MIDDLE = false $do_align_middle
      ...
