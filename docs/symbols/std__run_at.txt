# std::run_at (a polymorphic function)

  schedules a command to be executed at a specific time

  Parameters:

    self: the input-output object

    time: the time at which the command should be executed

    command: the command to be executed

    arguments: the arguments to be passed to the command

  Result:

    self: the updated input-output object

  The command will be executed at the specified time. If the time is a
  number, it is interpreted as the number of seconds from the current time.

  If the time is a date and time object, it is interpreted as the absolute
  time at which the command should be executed.

  Related topics: Input-Output Events

  Example:

    $io std_types::io # create an input-output object
    run_at &io 10 "ls -l" # schedule the command "ls -l" to be executed in 10 seconds
