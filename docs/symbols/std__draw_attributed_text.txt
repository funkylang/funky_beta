# std::draw_attributed_text (a polymorphic function)

  draws attributed text on a screen

  Parameters:
    screen # some kind of screen
    x: x-coordinate of the upper left corner
    y: y-coordinate of the upper left corner
    attributed_text: the text to draw

  Result:
    screen: the updated screen

  The text must not contain any newline characters and is clipped on the
  borders of the current window.

  Topic: Terminal

  See also: std::draw_text, std::clear, std::set_draw_colour,
            std::set_clear_colour, std::set_text_attributes

  Example:

    $my_screen create_screen(terminal 40 25)
    set_draw_colour &my_screen RED
    set_text_attributes &my_screen BOLD
    draw_attributed_text &my_screen 1 1 "Hello World!"
