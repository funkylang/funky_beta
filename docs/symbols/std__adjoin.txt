# std::adjoin (a polymorphic function)

  joins screens horizontally

  Parameters:
    self: the screen to join to
    screens: a list of screens to join

  Result:
    screen: the joined screen

  The screens are joined horizontally. All screens must have the same height.

  Topic: Terminal

  See also: std::create_view, std::create_vertical_panes, std::create_horizontal_panes

  Example:
