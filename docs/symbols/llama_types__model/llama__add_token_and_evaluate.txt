# llama_types::model/llama::add_token_and_evaluate (an I/O-method)

  adds a token and evaluates the sequence

  Parameters:
    model: the model to use
    sequence_id: the sequence to add the token to
    token: the token to add

  Results:
    none

  **This function must be called with I/O-access rights!**

  The specified token is added to the server stored sequence and then the
  sequence is evaluated.

  This might be faster than separate calls to
  <llama_types::model/llama::add_tokens> and
  <llama_types::model/llama::evaluate>.
