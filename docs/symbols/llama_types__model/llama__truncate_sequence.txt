# llama_types::model/llama::truncate_sequence (an I/O-method)

  truncates a sequence

  Parameters:
    model: the model
    sequence_id: the sequence id
    token_count: the token count

  Results:
    none

  **Attention**: This function must be called with I/O-access rights!

  The server stored sequence is truncated to the specified length.

  This can be used to "recycle" the start of an already evaluated sequence for
  another query without the need to reevaluate the already evaluated tokens.

  Topic: AI

  See also: llama::truncate_sequence, llama::copy_sequence
