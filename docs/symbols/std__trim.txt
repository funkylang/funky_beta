# std::trim (a function)

  removes leading and trailing whitespace

  Parameters:
    self: the string to trim
    test: a function that checks whether a character is a whitespace character

  Result:
    the trimmed string

  The function returns a string which is the specified string without any
  leading or trailing whitespace.

  Topic: Strings

  See also: std::trim_left, std::trim_right, std::normalize

  Example:

    $str "   text   "
    $trimmed_str trim(str)
    println! '*' trimmed_str '*'

  Output:

    *text*
