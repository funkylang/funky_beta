# std::rgb_character (a function)

  creates a character representing a colour with the given red, green and blue
  components

  Parameters:
    text_attributes: e.g. bold, underline, etc. (optional)
    red: the red component (0-255)
    green: the green component (0-255)
    blue: the blue component (0-255)

  Result:
    a character representing the specified colour

  A colour is represented as a 24-bit value (256 shades of red, green and blue)
  and stored as a character.

  Some functions use the bits 24..31 of a colour character to store text
  attributes like *bold* or *italic*.

  Topic: Colours, Terminal

  See also: std::BLACK, std::WHITE, std::text_attributes_of

  Example:

    $BOLD_ITALIC_RED rgb_character(BOLD|ITALIC 255 0 0)
