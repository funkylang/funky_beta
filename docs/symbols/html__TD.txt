# html::TD (a function)

  generates a TD element

  Parameters:
    items*: the content of the element

  Result:
    a string containing the TD element

  The function is used as a block function, with the content placed between the
  TD tags.

  Topic: HTML

  Example:

    <require basic/stdlib>
    <require basic/export/html>

    <using std>
    <using html>

    println!
      TD
	"Alice"

  Output:

    <td>Alice</td>
