# std_types::window_manager/std::set_inbounds_width (a method)

  sets the inbounds width of a window

  Parameters:
    wm: the window manager
    id: the window's id
    width: the new inbounds width

  Result:
    wm: the updated window manager

  The inbounds width of a window is the width of the window's content
  without the width of the window's border.
