# std_types::window_manager/std::get_buttons (a method)

  returns the additional buttons of the specified window

  Parameters:
    self: the window manager
    id: the id of the window

  Result:
    buttons: the additional buttons of the specified window

  If no window id is specified then the id of the focused window is used.

  If the window does not exist then <undefined> is returned.

  Topic: Window Manager

  See also: std_types::window_manager/std::set_buttons
