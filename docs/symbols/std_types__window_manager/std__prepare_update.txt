# std_types::window_manager/std::prepare_update (a method)

  prepares the window manager for an update

  Parameters:
    self: the window manager
    terminal: the terminal to be updated

  Results:
    window_manager: the updated window manager
    terminal: the updated terminal
    update_ranges: a list of ranges to be updated

  This function prepares the window manager for an update.

  If the window manager is up-to-date then the result is the window manager
  itself, the terminal and an empty list.

  Otherwise the window manager is updated and a list of ranges to be updated
  is returned.

  Each range is a tuple of the form (row_range window_id content).
