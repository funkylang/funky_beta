# std_types::window_manager/std::set_menu (a method)

  sets the menu bar

  Parameters:
    wm: the window manager
    menus: the menu bar

  Result:
    wm: the updated window manager

  The menu bar is a list of tuples.

  Each tuple contains the menu title and the menu items.

  The menu items are tuples of the form (title function).

  The function is called when the menu item is selected.

  The function must have the form (wm id) and return the updated window
  manager.

  The function may return <std::undefined> to close the window manager.

  Example:

    $wm std::create_window_manager(std::terminal)
    $wm std::set_menu(wm
      tuple("File"
        tuple("New" new_file)
        tuple("Open" open_file)
        tuple("Save" save_file)
        tuple("Save as" save_file_as)
        tuple("Close" close_file)
        tuple("Quit" quit)
      )
      tuple("Edit"
        tuple("Cut" cut)
        tuple("Copy" copy)
        tuple("Paste" paste)
      )
      tuple("Help"
        tuple("About" about)
      )
    )
