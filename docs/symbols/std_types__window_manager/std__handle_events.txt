# std_types::window_manager/std::handle_events (a method)

  handles events

  Parameters:
    wm: the window manager
    io: the I/O-object
    events: the events to handle
    wm_fd: the file descriptor of the window manager

  Results:
    wm: the updated window manager
    new_events: the events that were not consumed

  This method handles events that effect the window manager as a whole.

  The method also handles events that effect the currently focused window.

  The method returns the updated window manager and the events that were not
  consumed.
