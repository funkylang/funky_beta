# std_types::window_manager/std::shrink (a method)

  minimizes a window

  Parameters:
    window_manager: a window manager
    id?: the id of the window to minimize (default: the focused window)

  Result:
    window_manager: the updated window manager

  Minimizes the window with the specified *id*.

  If no *id* is specified then the focused window is minimized.

  Topic: Window Manager

  See also: std_types::window_manager/std::enlarge

  Example:

    $wm create_window_manager(terminal)
    create_window &wm $id "My Window"
    ...
    shrink &wm id
