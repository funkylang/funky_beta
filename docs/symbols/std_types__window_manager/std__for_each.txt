# std_types::window_manager/std::for_each (a method)

  iterates over all windows

  Parameters:
    wm: the window manager
    body: the function to apply to each window
    continuation: the function to apply after the iteration

  Result:
    result: the result of the continuation function

  The body function is called for each window with the window's id and
  content as arguments.

  The continuation function is called after the iteration with no arguments.

  Topic: Window Manager

  See also: std::next, std::break
