# std_types::window_manager/std::goto_xy (a method)

  moves the cursor to the specified position

  Parameters:
    window_manager: a window manager
    id?: the id of the window to move (default: the focused window)
    x: the x coordinate
    y: the y coordinate

  Result:
    window_manager: the updated window manager

  Moves the cursor of the window with the specified *id* to the specified
  *x* and *y* coordinates.

  If no *id* is specified then the cursor of the focused window is moved.

  The *x* and *y* coordinates are clamped to the valid range of the window.

  Topic: Window Manager

  See also: std_types::window_manager/std::get_xy,
	    std_types::window_manager/std::get_content,
	    std_types::window_manager/std::get_size

  Example:

    $wm create_window_manager(terminal)
    create_window &wm $id "My Window"
    ...
    goto_xy &wm id 10 5
