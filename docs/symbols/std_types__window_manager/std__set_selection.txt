# std_types::window_manager/std::set_selection (a method)

  sets the selection of the specified window

  Parameters:
    self: the window manager
    id: the id of the window
    x: the x-coordinate of the upper left corner of the selection
    y: the y-coordinate of the upper left corner of the selection
    w: the width of the selection
    h: the height of the selection

  Result:
    window_manager: the updated window manager

  If the window manager does not contain a window with the specified id then
  the window manager is not changed.

  If the window manager contains a window with the specified id then the
  selection of this window is set to the specified rectangle.

  If the id is not specified then the selection of the focused window is set.

  If the window manager does not contain a focused window then the window
  manager is not changed.

  Topic: Window Manager

  See also: std_types::window_manager/std::unset_selection
