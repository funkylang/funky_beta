# std_types::window_manager/std::get_xy (a method)

  returns the cursor position of the specified window

  Parameters:
    wm: the window manager
    id: the id of the window

  Results:
    x: the x-coordinate of the cursor
    y: the y-coordinate of the cursor

  If the window has no cursor then <undefined> is returned for both
  coordinates.

  If the window does not exist then <undefined> is returned for both
  coordinates.

  Topic: Window Manager

  See also: std_types::window_manager/std::goto_xy
