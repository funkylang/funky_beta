# std_types::window_manager/std::move_page_up (a method)

  moves the content of a window up one page

  Parameters:
    window_manager: a window manager
    id?: the id of the window to move (default: the focused window)

  Result:
    window_manager: the updated window manager

  Moves the contents of the window with the specified *id* up one page.

  If no *id* is specified then the contents of the focused window are moved up.

  If the window has a cursor, then the cursor is moved up one page instead.
  In this case the contents do not necessarily scroll!

  Topic: Window Manager

  See also: std_types::window_manager/std::move_page_down,

	    std_types::window_manager/std::move_end

  Example:

    $wm create_window_manager(terminal)
    create_window &wm $id "My Window"
    ...
    move_page_up &wm id
