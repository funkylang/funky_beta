# std_types::window_manager/std::focus_next_window (a method)

  focuses the next window

  Parameters:
    self: the window manager

  Result:
    window_manager: the updated window manager

  The window manager's focus is moved to the next window.

  If the currently focused window is the last window then the focus is
  moved to the first window.
