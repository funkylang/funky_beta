# std_types::window_manager/std::create_dialog (a method)

  creates a dialog

  Parameters:
    window_manager: a window manager
    info: the dialog information
    content: the dialog content

  Result:
    window_manager: the updated window manager

  Creates a dialog with the specified *content* and *info*.

  The *info* parameter is passed to the dialog handler when a button is pressed.
