# std::next_key_of (a polymorphic function)

  returns the key after the specified key

  Parameters:
    self: the collection from which to get the next key
    key: the key

  Result:
    key: the next key

  The function returns the key after the specified key.

  If the specified key is the last key in the collection then the function
  returns undefined.

  Topic: Collections

  See also: std::previous_key_of

  Example:

    $my_table create_avl_table()
    insert_into my_table "a" 1
    insert_into my_table "b" 2
    insert_into my_table "c" 3
    println! std::next_key_of(my_table "a")
    println! std::next_key_of(my_table "c")
