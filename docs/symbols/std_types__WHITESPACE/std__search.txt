# std_types::WHITESPACE/std::search (a method)

  searches for whitespace

  Parameters:
    self: the grammar object
    str: the string to search
    nth: the nth occurrence to find

  Results:
    pos: the position where the whitespace was found
    len: the length of the whitespace

  This method searches for whitespace in the given string. If no whitespace is
  found, the method <std::undefined> for the position and length.

  If *nth* is specified, the method searches for the nth occurrence of
  whitespace. If *nth* is not specified, the method searches for the first
  occurrence.

  Topics: Grammar Objects

  See also: std::WHITESPACE, std::FUNKY_IDENTIFIER/std::search

  Example:

    $str "This is a text."
    search $pos $len WHITESPACE str
    dump! `pos `len

  Output:

    pos: 5
    len: 1

  Example:

    $str "This is a text."
    search $pos $len WHITESPACE 2 str
    dump! `pos `len

  Output:

    pos: 8
    len: 1
