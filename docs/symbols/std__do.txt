# std::do (an I/O-function)

  executes a statement sequence in its own scope

  Parameter:
    body: the statement sequence to execute

  Results:
    the results of the last statement in the body

  This function supports I/O-access. If it is called with I/O-access rights than
  the access rights are passed on to the *body*.

  The function executes the statement sequence of the *body*.

  *do* is used to explicitely create a new scope.

  Topic: Primitives

  See also: std::pass, std::assign, std::chain

  Example:

    do:
      $y f(x)
      println! y
