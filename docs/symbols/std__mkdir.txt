# std::mkdir (a builtin I/O-function)

  creates a directory

  Parameters:
    pathname: the path of the directory to create
    mode: the mode of the directory (default: ""rw-r--r--")

  The optional mode parameter is a string of the form "rwxrwxrwx". The
  default value is "rw-r--r--".

  This function needs I/O-access rights.

  Topic: POSIX

  See also: std::mkfifo

  Example:

    mkdir! "test_dir" "rw-rw----"
