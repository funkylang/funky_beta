# std::tuple_or_error (a builtin function)

  constructs a tuple from the specified arguments

  If any of the arguments is an error object then this error object (the
  leftmost one in case there is more than one) is returned instead of the
  tuple.

  Parameters:
    item_1, ..., item_n: the items of the tuple

  Result:
    value: the tuple or an error object

  Topic: Tuples

  See also: std::tuple, std_types::tuple

  Example:

    # create a tuple
    t = std::tuple_or_error(1, 2, 3)
    # -> (1, 2, 3)

    # create an error object
    e = std::error("an error")
    # -> <error: an error>

    # create a tuple with an error object
    t = std::tuple_or_error(1, e, 3)
    # -> <error: an error>
