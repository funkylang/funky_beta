# std::next_item_of (a polymorphic function)

  returns the next item after the specified item

  Parameters:
    collection: some kind of collection
    item: an item of the collection

  Result:
    item: the next item

  If the specified item is the last item of the collection then the first
  item is returned.

  Topic: Collections

  See also: std::previous_item_of, std::next_key_of, std::previous_key_of

  Example:

    $my_list create_list(1 2 3 4 5)
    println! next_item_of(&my_list 5)

  Output:

    1
