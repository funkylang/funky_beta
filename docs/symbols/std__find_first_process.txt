# std::find_first_process (an I/O-function)

  finds the process ID of the first process with the specified name

  Parameter:
    name: the name of the process to find

  Results:
    the process ID of the first matching process or <std::undefined>

  **This function must be called with I/O-access rights!**

  The function searches for the first process with the specified name in the
  "/proc" directory.

  Topic: Input-Output

  See also: std::find_processes, std::get_process_name, std::which
