# std::intersection (a polymorphic function)

  compute the intersection of two collections

  Parameters:
    left: the first collection
    right: the second collection

  Result:
    intersection: a new collection containing only the elements present in both
                  input collections

  The result shares type with the first collection.

  Topic: Collections

  See also: std::merge, std::zip

  Example:

    $my_list1 create_list(1 2 3 4 5)
    $my_list2 create_list(3 5 6 7 8)
    $intersection = std::intersection(my_list1 my_list2)
    println! intersection # output: (3 5)
