# std::broadcast_websocket_text_message (an I/O-function)

  broadcasts a websocket text message to all connected websocket clients
  except the one with the specified file descriptor

  Parameters:
    io: the I/O-object
    omit_fd: the file descriptor of the client to omit
    text: the text to send

  Results:
    the I/O-object

  **This function must be called with I/O-access rights!**

  Topic: Network Clients and Servers

  See also: std::send_websocket_text_message

  Example:

    broadcast_websocket_text_message! &io STDIN_FILENO "Hello, World!"
