# std::scroll_right (a polymorphic function)

  Scrolls the contents of a screen to the right.

  Parameters:
    screen: the screen to scroll
    delta: the number of characters to scroll (default: 1)

  Result:
    screen: the updated screen

  The function scrolls the contents of the screen to the right by the
  specified number of characters. Characters that are scrolled off the
  right edge of the screen are lost.

  Topic: Terminal

  See also: std::scroll_left, std::scroll_up, std::scroll_down

  Example:

    $my_screen create_screen(terminal 40 25)
    draw_text &my_screen 1 1 "Hello, world!"
    scroll_right &my_screen 5
    # The text "Hello, world!" will be scrolled 5 characters to the right.
