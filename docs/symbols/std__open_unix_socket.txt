# std::open_unix_socket (a builtin I/O-function)

  opens a UNIX domain socket

  The optional mode parameter is a string of the form "rwxrwxrwx". The
  default value is "rw-r--r--".

  Parameters:
    filename: the path and name of the socket

  Result:
    socket: the opened and connected socket

  Topic: Input-Output

  See also: std::open_tcp_client_socket, std::open_tcp_server_socket,
            std::open, std::close, std::read, std::write

  Example:

    open_unix_socket! $fd "/tmp/my_socket"
    write! $_bytes_written fd "Hello, world!"
