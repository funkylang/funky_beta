# std::sort (a polymorphic function)

  sorts a collection

  Parameters:
    self: the collection to sort
    compare: a comparison function (optional)

  Result:
    a new sorted collection

  The result is a new collection containing all the items of the original
  collection in sorted order.

  If no comparison function is supplied then the items are sorted in ascending
  order using the <less>-method.

  Topic: Collections

  See also: std::map, std::reduce, std::filter

  Example:

    $my_list create_list(5 2 8 1 9)
    $sorted_list std::sort(my_list)
    println! sorted_list
