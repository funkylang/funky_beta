# std::wait (a builtin I/O-function)

  suspends the execution until a child process terminates.

  Results:

    * **child_process_id**: the process ID of the terminated child process
    * **status**: the exit status of the terminated child process

  **This function must be called with I/O-access rights!**

  Topic: POSIX

  See also: std::fork
