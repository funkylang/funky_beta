# std::shm_unlink (a builtin I/O-function)

  removes a shared memory object

  Parameters:
    name: the name of the shared memory object to remove

  Results:
    none

  **Attention**: This function must be called with I/O-access rights!

  Topic: POSIX

  See also: std::shm_open

  Example:

    # create a shared memory object
    fd = std::shm_open("my_shared_memory", std::O_CREAT | std::O_RDWR, 0644);
    # ...
    # remove the shared memory object
    std::shm_unlink("my_shared_memory");
