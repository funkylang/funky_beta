# std::chroot (a builtin I/O-function)

  changes the root directory of the current process

  Parameters:
    pathname: the name of the new root directory

  Results:
    none

  **Attention**: This function must be called with I/O-access rights!

  Topic: POSIX

  See also: std::chdir

  Example:

    # change the root directory to "/tmp"
    chroot! "/tmp"
