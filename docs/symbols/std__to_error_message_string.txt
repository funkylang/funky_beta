# std::to_error_message_string (a function)

  converts an error object to a string

  Parameter:
    err: the error object

  Result:
    the error message as a string

  The function converts an error object to a string. The error message is
  composed of the error message text, the error number and the error details.

  Topic: Errors

  See also: std::Error, std::error_message_text_of, std::errno_of, std::error_details_of

  Example:

    $err error("An error occurred")
    println! to_error_message_string(err)

  Output:

    An error occurred
