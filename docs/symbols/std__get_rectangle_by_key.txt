# std::get_rectangle_by_key (a polymorphic function)

  returns a rectangle by its key

  Parameters:
    screen: some kind of screen
    key: the key of the rectangle

  Result:
    rectangle: the rectangle or undefined

  Returns the rectangle with the given key.

  Topic: Terminal

  See also: std::get_rectangle_at, std::get_next_rectangle,
            std::get_previous_rectangle

  Example:

    $my_screen create_screen(terminal 40 25)
    draw_border &my_screen 1 1 38 24 "My Screen"
    get_rectangle_by_key &my_screen "My Screen" $rectangle

    draw_text &my_screen 26 3 width_of(rectangle)
