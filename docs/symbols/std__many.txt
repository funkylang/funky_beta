# std::many (a function)

  creates a grammar object to match one or more occurrences of an expression

  Parameters:
    expr: the expression to match multiple times

  Result:
    a grammar object

  This function creates a grammar object that matches one or more occurrences of
  the given expression.

  Topics: Grammar Objects

  Example:

    $str "123abc"
    match many(DIGIT) str $len
    println! len

  Output:

    3

  Example:

    $str "abc"
    match many(DIGIT) str $len
    dump! `len

    len: <undefined>
