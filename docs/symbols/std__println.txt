# std::println (an I/O-function)

  prints its arguments and a line feed to the standard output

  Parameter:
    args: the arguments to write

  Results:
    none

  **Attention**: This function must be called with I/O-access rights!

  The function converts its arguments to a single string, encodes this string
  to UTF-8, appends a line feed character and writes this encoded string to
  the standard output.

  Topic: Input-Output

  See also: std::print, std::eprintln, std::println_to

  Example:

    println! "Hello, world!"
