# std_types::stat.std::type_of (an attribute)

  the type of the file

  Allows to check whether a file is a directory, a symbol link, etc.

  Topic: POSIX

  See also: std_types::stat.std::mode_of, std_types::file_type

  Example:

    stat! $info "/usr/local/bin/funky"
    debug::dump `type_of(info)

  Error output:

    type_of(info): <file_type 8>
