# std_types::stat.std::mode_of (an attribute)

  the file's access rights

  The file's access rights are stored in a string of the form "trwxrwxrwx".

  Topic: POSIX

  See also: std::chmod

  Example:

    stat! $info "/usr/local/bin/funky"
    println! mode_of(info)

  Output:

    -rwxr-xr-x
