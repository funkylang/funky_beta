Colours

Description

The Terminal library uses std_types::characters for text and background colours.

Related topics

Terminal

Functions

std::ansi_background_colour returns an ANSI escape sequence that sets the background colour
std::ansi_reset_colour resets the text and background colour to the default
std::ansi_text_colour returns an ANSI escape sequence that sets the text colour
std::blue_component_of extracts the blue component of a character
std::colour_mixture mixes colours
std::csi generates an ANSI escape sequence
std::green_component_of extracts the green component of a character
std::red_component_of returns the red component of a character
std::rgb_character creates a character that represents a colour

Constants

std::CLEAR_COLOUR used to specify a clear colour
std::DRAW_COLOUR used to specify a draw colour
(defined in terminal/terminal.fky)
