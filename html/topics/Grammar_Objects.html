Grammar Objects

Description

Grammar objects support methods for matching and searching.

Types

std_types::FUNKY_IDENTIFIER the prototype object for all Funky identifiers
std_types::WHITESPACE the prototype object for all whitespace matchers
std_types::alt This object represents an alternation of expressions.
std_types::any_character_but the prototype object for the "any character but" grammar object
std_types::anything the prototype object for the "anything"-grammar object
std_types::capture the prototype object for capturing expressions
std_types::many the prototype object for "many"-expressions
std_types::optional the prototype object for optional expressions
std_types::some the prototype object for "some"-expressions

Functions

std::any_character_but creates a grammar object that matches any character but the specified
std::capture captures the result of a match
std::many creates a grammar object that matches multiple occurrences of the specified
std::match matches the start of a string with the specified expression
std::match_back matches the end of a string with the expression
std::optional creates an optional expression
std::search searches for the first occurrence of a grammar object within a string
std::some creates a "some"-object

Objects

std::ANYTHING matches any character
std::FUNKY_IDENTIFIER a value used to match a Funky identifier
std::WHITESPACE a value used to match whitespace
std::alt a grammar object that matches any of the specified expressions
std::set a set of unique values
(defined in basic/grammar.fky)
