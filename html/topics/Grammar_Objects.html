Grammar Objects

Description

Grammar objects support methods for matching and searching. They take the place of regular expressions used in some other languages and libraries.

A simple string or character is a grammar object that matches its contents.

A range of characters is also a grammar object, e.g.

'a'..'z'

takes the place of the regular expression

[a..z]

The grammar object

some(expr)

takes the place of the regular expression

expr*

The grammar object

many(expr)

takes the place of the regular expression

expr+

The grammar object

optional(expr)

takes the place of the regular expression

expr?

To concatenate several grammar objects the are put into a sequence:

expr_1, expr_2, ..., expr_n

The grammar object

alt(expr_1 expr_2)

takes the place of the regular expression

expr_1|expr_2

Grammar objects can be built stepwise from already defined grammar objects.

They can be used in string functions like std::before or std::has_prefix.

Types

std_types::FUNKY_IDENTIFIER the prototype grammar object for Funky identifiers
std_types::WHITESPACE the prototype grammar object for whitespace
std_types::alt a grammar object to match one of several alternatives
std_types::any_character_but the prototype grammar object for matching anything but the specified
std_types::anything the prototype for grammar objects that match everything
std_types::capture the prototype grammar object for capturing text
std_types::many the protoype grammar object for matching one or more occurences of something
std_types::optional the prototype grammar object for matching zero or one occurrence of something
std_types::some the prototype grammar object to match zero or more occurrences of something

Functions

std::alt creates a grammar object for matching one of several alternatives
std::any_character_but creates grammar object to match anything but the specified expression
std::capture creates a capture grammar object
std::many creates a grammar object to match one or more occurrences of an expression
std::match matches the start of a string with the specified expression
std::match_back matches the end of a string with the expression
std::optional creates a grammar object to match zero or one occurrence of an expression
std::search searches for the first occurrence of a grammar object within a string
std::some creates a grammar object to match zero or more occurrences of an expression

Objects

std::ANYTHING matches any character
std::FUNKY_IDENTIFIER a value used to match a Funky identifier
std::WHITESPACE a value used to match whitespace
std::set a set of unique values

Constants

std::DIGIT a grammar object to match a digit
std::LETTER a grammar object to match a letter
(defined in basic/grammar.fky)
