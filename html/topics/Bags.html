Bags

Description

Bags are "multi-sets". A bag keeps a record of how many of each elements are stored.

A bag can contain a negative number of elements!

Types

std_types::hash_bag the prototype object for all hash-based bags

Functions

std::hash_bag creates a hash bag from a list of key-value-pairs
std::to_hash_bag converts a collection into a hash bag

Objects

std::empty_hash_bag an empty hash bag
(defined in basic/types/collections/bags/hash_bag.fky)
