Serialization

Functions

std::deserialize deserializes a machine readable string representation of an object
std::deserializer_of returns a function that can deserialize an object of the given type
std::register_type registers a type for deserialization
std::register_types registers a list of types for deserialization
std::serialization_fields_of returns a list of field names and getter functions for serialization
std::serialization_tag_of returns the serialization tag of the object
std::serialize returns a machine readable string representation of the object
