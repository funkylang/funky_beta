Random Numbers

Types

std_types::lcg the prototype object for linear congruential generators
std_types::mersenne_twister the prototype object for a pseudo random number generator
std_types::random_number_generator the prototype object for all random number generators

Functions

std::is_a_random_number_generator checks if an object is a random number generator
std::lcg creates a pseudo random number generator
std::mersenne_twister creates a pseudo random number generator
std::modulus_of returns the modulus of a random number generator
std::next_raw_value returns the next raw value of a random number generator
std::seed_of returns the current seed of a random number generator
