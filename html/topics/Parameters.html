Parameters

Description

These functions make it easy to parse

  • command line parameters - std::program_parameters
  • "named" options of functions - std::extract_options

Functions

std::extract_options extracts the options of a function call
std::get_options fetches the options of a function call

Constants

std::COPYRIGHT used to specify the copyright message of a program
std::MANDATORY_PARAMETER used to specify a mandatory parameter
std::MANY_PARAMETERS used to specify a parameter that accepts one or more values
std::MULTI_VALUED_OPTION used to specify a multi-valued option
std::OPTIONAL_PARAMETER specifies an optional parameter
std::SOME_PARAMETERS used to specify a parameter that can be specified zero or more times
std::VALUED_OPTION used to specify a valued option
std::VERSION used to specify the version of a program
(defined in basic/parameters.fky)
