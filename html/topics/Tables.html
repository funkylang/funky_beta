Tables

Description

Tables are associative arrays.

Types

std_types::association_list a very primitive kind of a table
std_types::avl_table the prototype object for all AVL-based tables
std_types::binary_table a table based on binary trees
std_types::hash_table a table for fast random access of items
std_types::insert_order_table the prototype object for all insert-order tables
std_types::key_order_table a table which items are stored in key-order
std_types::lb2_table the prototype object for all left-balanced 2-3-4-trees

Functions

std::create_index_table creates an index table for the specified collection
std::hash_table creates a hash table from a list of key-value-pairs
std::insert_order_table creates an insert-order table from a list of key-value-pairs
std::is_a_binary_table checks if an object is a binary table
std::is_a_hash_table checks if an object is a hash table
std::is_a_key_order_table checks if an object is a key-order table
std::is_a_lb2_table checks if an object is a lb2_table
std::is_an_avl_table checks if a given object is an AVL-table
std::is_an_insert_order_table checks if an object is an insert-order table
std::key_order_table creates a key-order table from a list of key-value-pairs
std::swap_key_and_value swaps the keys and values of the specified table
std::to_hash_table converts a list containing key-value-pairs into a hash table
std::validate validates the internal structure of a binary table

Objects

std::empty_avl_table an empty AVL-table
std::empty_binary_table an empty binary table
std::empty_key_order_table an empty key-order table
std::empty_lb2_table an empty left-balanced binary tree
(defined in basic/types/collections/tables/table.fky)
