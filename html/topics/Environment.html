Environment

Description

Allows retrieval of information about the operating system's "environment" and also about where the Funky tools are installed.

Example

$shared_path append installation_prefix() "/share/funky-" package_version() println! shared_path

Output

/usr/local/share/funky-0.0.0

Functions

std::installation_prefix returns the installation prefix of Funky
std::package_version returns the version of the Funky installation

Objects

std::command_line_arguments a list containing the application's command line arguments
std::environment_variables a list containing the environment variables
std::program_name the name of the current process
(defined in basic/environment.fky)
