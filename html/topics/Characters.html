Characters

Description

Characters are unicode code points in Funky. But they can hold any positive 32-bit integer value.

Characters are the building blocks of strings.

See also

Strings Strings consist of zero or more characters.

Functions

std::is_a_character tests if an object is a character
std::to_wide converts a character or a string to its wide character representation
(defined in basic/types/character.fky)
