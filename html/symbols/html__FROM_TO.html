html::FROM_TO

generates a string by calling a function for each index in a range

Parameters

from
the start of the range
to
the end of the range
body
a function that is called for each index in the range

Result

text
the generated string

Description

The function is called with the current index as argument and the returned value is appended to the result string.

Topic

HTML

See also

html::FOR_EACH iterates over a collection and calls a function for each item

Example

println! FROM_TO 1 5 : (idx) -> "idx = @(idx) "

Output

idx = 1 idx = 2 idx = 3 idx = 4 idx = 5
