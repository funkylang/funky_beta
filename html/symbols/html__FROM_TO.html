html::FROM_TO

generates a sequence of elements from a range of indices

Parameters

from
the starting index
to
the ending index
body
a function that generates the element for a given index

Result

a string containing the generated elements

Description

The function is used to generate a sequence of elements from a range of indices.

Topic

HTML

Example

<require basic/stdlib> <require basic/export/html> <using std> <using html> println! FROM_TO 1 3 : (idx) P "This is paragraph @(idx)"
