std::is_a_hash_table

checks if an object is a hash table

Parameter

self
the object to check

Result

true if the object is a hash table, false otherwise

Topic

Tables

See also

Example

$my_hash_table create_hash_table(10) println! std::is_a_hash_table(my_hash_table) # prints true
