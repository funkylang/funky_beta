std_types::hash_bag

the prototype object for all hash-based bags

Description

This object is never used directly but cloned to create a new bag.

Topics

Collections

See also

Example

$numbers empty_hash_bag !numbers(1) !numbers(2) !numbers(3) !numbers(2) !numbers(4) !numbers(5) !numbers(6) !numbers(5) println! numbers

Output

hash_bag <3> 1 2 3 4 5 6
