std_types::hash_bag

the prototype object for all hash-based bags

Derived from

std_types::hash_table a table for fast random access of items

Derived types

std::empty_hash_bag an empty hash bag

Description

This object is never used directly but cloned to create a new bag.

A hash bag is an "unordered" bag optimized for quick access. Each entry stores a key and an associated count. The count can be negative!

The type function of a hash bag allows for retrieval of the count associated with a key when called with the key as its single argument.

When the type function is called with two arguments (a key and a count), it creates a new bag which is an updated version of the current bag. The new bag contains all the entries of the current bag, plus an a new entry with the given key and count. If the key already exists in the bag, its count is updated by adding the given count. If the resulting count is zero, the entry is removed from the bag.
