std_types::list/std::reduce_rows

reduces the rows of a list of lists

Parameters

Description

self: the list of lists

reduction: the reduction function

Result

a list

The reduction function is applied to each row of the list.

The first argument of the reduction function is the current row.

The second argument of the reduction function is the corresponding row of the right-hand side list.
