std_types::list/debug::to_string

creates a string representation of the list

Parameters

self
the list to convert
indent
the indentation level
depth
the maximum depth

Result

string
the string representation of the list

Description

The string representation is a multi-line string.

The indentation level is used to indent the output.

The depth parameter is used to limit the depth of the output.
