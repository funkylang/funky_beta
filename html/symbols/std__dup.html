std::dup

creates a concatenation of n copies of the specified collection

Description

The function aims to duplicate the specified collection in a smart way.

Topic

Collections

See also

std::spaces returns a string consisting of the specified number of spaces
std::tabs_and_spaces returns a sequence of tabulator- and space-characters

Example

$str "abc"
$str_x_3 dup(str 3)
println! str_x_3

Output

abc abc abc
