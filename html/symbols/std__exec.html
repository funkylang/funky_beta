std::exec

replaces the current process with the given command

Parameters

filename
the executable image of the new process
arguments
a list of arguments to supply to the new process

Description

This function does not return on success.

On failure it returns an appropriate error object.

Topic

Input-Output

See also

std::call calls the specified command with the specified arguments
std::create_process creates a new child process

Example

exec! "ls" list("-l" "/usr/local/share/funky-0/runtime/")
