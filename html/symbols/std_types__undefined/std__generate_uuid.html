std_types::undefined/std::generate_uuid

generates and returns a Universally Unique Identifier (UUID)

Parameters

self
ignored
_kind
how to create the UUID

Result

uuid
the generated UUID

Description

The UUID is a 128-bit number that can be used to identify information in a network.

The UUID is generated using one of the following methods:

  • std::RANDOM_BASED_UUID: the UUID is generated using a random number generator
  • std::TIME_BASED_UUID: the UUID is generated using the current time
  • std::NAME_BASED_UUID: the UUID is generated using a name
