std_types::undefined/debug::to_string

returns a string representation of the object

Parameters

self
the object to convert
indent
the indentation level
depth
the maximum recursion depth

Result

string
the string representation

Description

The function returns a string representation of the object.

The string representation is converted to utf-8 before beeing written.

The optional indent argument specifies the indentation level.

The optional depth argument specifies the maximum recursion depth used to display the object's attributes.
