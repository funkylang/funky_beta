std_types::view/std::set_cursor

sets the cursor position

Parameters

self
the view to set the cursor position for
x
the x-coordinate of the cursor position
y
the y-coordinate of the cursor position

Result

view
the updated view

Description

The cursor position is set to the specified position.

If the cursor position is outside the view's area then the cursor is switched off.
