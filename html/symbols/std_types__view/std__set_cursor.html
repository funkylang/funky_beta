std_types::view/std::set_cursor

sets the cursor position

Parameters

view
the view for which to set the cursor
terminal
the terminal for which to set the cursor
x
the x-coordinate of the cursor
y
the y-coordinate of the cursor

Result

terminal
the updated terminal

Description

The cursor position is set to the specified position.

The cursor position is clipped at the view's borders.
