std_types::view/std::draw_shadow

draws a shadow around the specified rectangular area

Parameters

self
the view to draw the shadow in
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area
shadow_colour
the colour of the shadow

Result

view
the updated view

Description

The shadow is drawn using the specified colour.

The shadow is drawn outside the specified area.

This function also works for "graphical views" (line_height != 1).
