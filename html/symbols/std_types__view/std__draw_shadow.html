std_types::view/std::draw_shadow

draws a shadow around the specified rectangular area

Parameters

self
the view to draw the shadow in
screen
the screen to draw the shadow on
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area

Result

screen
the updated screen

Description

The shadow is drawn using the the current draw- and clear colour.

The area is clipped at the view's borders.

Terminal
