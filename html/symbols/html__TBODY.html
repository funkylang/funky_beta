html::TBODY

creates a "tbody"-tag

Parameters

items
the items to be enclosed in the "tbody"-tag

Result

a string containing the "tbody"-tag

Description

The "tbody"-tag is used to group rows in a table.

Topic

HTML

See also

html::TABLE creates a HTML table
html::TR creates a table row
html::TH creates a table header cell
html::TD creates a table data cell

Example

printnln! TABLE : TBODY : TR : TH "Name" : TH "Age" : TR : TD "John" : TD "42" : TR : TD "Mary" : TD "25"

Output

<table> <tbody> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>42</td> </tr> <tr> <td>Mary</td> <td>25</td> </tr> </tbody> </table>
