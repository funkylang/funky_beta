std_types::string/std::to_title_case

converts the string to Title Case

Parameter

self
the string to be converted

Result

title
the title case version of self

Description

All characters but the first character of each word are converted to lower case. The first characters of all words are converted to upper case.

Topic

Strings

See also

std_types::string/std::to_lower_case returns the string converted to lower case
std_types::string/std::to_upper_case returns the string converted to UPPER CASE

Example

$str "this is a test"
$title_case_str str.to_title_case
println! title_case_str

Output

This Is A Test
