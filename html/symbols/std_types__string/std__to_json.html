std_types::string/std::to_json

converts the string to a JSON-string

Parameters

self
the string to convert

Result

string
the JSON-string

Description

The string is converted to a JSON-string by enclosing it in double quotes and escaping the following characters:

  • double quotes
  • backslash
  • newline
  • carriage return
  • horizontal tab
  • backspace
  • form feed
