std_types::string/std::save

saves the supplied data to a file

Parameters

filename
the name of the file to save to
data
the data to save

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function opens the file with the specified name, writes the supplied data to the file and closes the file.
