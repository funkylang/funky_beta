std_types::string/std::load

loads the contents of the specified file into a string

Parameters

filename
the name of the file to load
max_length
the maximum number of bytes to load (optional)

Results

the contents of the file as a string

Description

Attention: This function must be called with I/O-access rights!

If max_length is specified then at most max_length bytes are loaded.

Topic

Input-Output

See also

std::save saves a file in some way

Example

load! $buf "test.dat"

Example

load! $buf "/dev/sda" 512 # load first sector of /dev/sda
