std_types::string/std::edit

edits a string

Parameters

text
the text to edit
cursor_position
the position of the cursor (1 .. length_of(text)+1)
key
the pressed key

Results

text
the updated text
cursor_position
the updated cursor position

Description

This function is used to edit a string.

The cursor position is updated according to the pressed key.

The following keys are supported:

  • CURSOR_LEFT: moves the cursor one position to the left
  • CURSOR_RIGHT: moves the cursor one position to the right
  • HOME: moves the cursor to the beginning of the line
  • END: moves the cursor to the end of the line
  • BACKSPACE: deletes the character left of the cursor
  • DELETE: deletes the character right of the cursor
  • CTRL_BACKSPACE: deletes all the text left of the cursor
  • CTRL_DELETE: deletes all the text right of the cursor
  • CTRL_K: deletes all the text
  • any other key: inserts the key at the cursor position
