std_types::string/std::without_prefix

removes a specific expression at the start of a string

Parameters

self
the string to process
prefix
the prefix to remove

Result

string
the processed string

Description

If the string starts with the specified prefix, the matching prefix of the string is removed and the resulting string is returned.

Otherwise the string is returned without alteration.
