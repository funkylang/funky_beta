std::PINK

the colour "pink" (0xffc0cb) as used by the terminal library

Topics

Colours
Terminal

Description

Colour constants are used to set the draw or clear colour.

The colour code for "pink" is 0xffc0cb.

See also

std::set_draw_colour sets the draw colour
std::set_clear_colour sets the clear colour of a terminal

Example

set_clear_colour &terminal PINK
