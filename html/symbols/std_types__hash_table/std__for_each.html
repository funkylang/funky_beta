std_types::hash_table/std::for_each

iterates over all items of a hash table

Parameters

self
the hash table to iterate over
body
the statement sequence to execute for each item
finally
called after all items have been iterated over

Results

none

Topic

Collections

See also

std::next starts the next "iteration" of a "loop"
std::break cancels execution of the current "loop"

Example

$ages
  hash_table
    "John" = 42
    "Mary" = 25
    "Irene" = 49

for_each ages:
  println! "Name: " key " Age: " value

Output

Name: Mary Age: 25
Name: Irene Age: 49
Name: John Age: 42
