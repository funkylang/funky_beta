std_types::lb2_table

the prototype object for all left-balanced 2-3-4-trees

Parameters

key
the key to use for insertion or retrieval
value
if supplied the value to associate with the key

Result

value_or_object
the retrieved value for retrievals or the updated object
for insertions

Description

This object is never used directly but cloned to create a new table.

When called it can act as a function to insert a new key-value-pair into the table (2 arguments) or to retrieve the value associated with a key (1 argument).
