std::chown

changes the owner and group of the specified file

Parameters

filename
the name of the file for which the owner and group are to be changed
owner
the new owner
group
the new group

Description

This function needs I/O-access rights.

Topic

POSIX

See also

std::chmod changes the mode of the specified file

Example

# change the owner and group of a file
chown! "myfile.txt" 1000 1000
