std::get_from_until

reads data from a file descriptor until a sentinel is found and decodes the data to a string

Parameters

fd
the file descriptor to read from
sentinel
the sentinel to search for
nth
the nth occurrence of the sentinel to search for

Results

the read data as a string

Description

Attention: This function must be called with I/O-access rights!

The function reads data from the file descriptor until the specified sentinel is found and returns the read data as a string. The sentinel is included in the returned string and the string is decoded from UTF-8.
