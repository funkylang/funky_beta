html::TABLE

creates a HTML table

Parameters

items
the table contents

Result

a string containing the HTML code

Description

The table contents are specified as a list of table rows.

A table row is specified as a list of table cells.

A table cell is specified as a list of table cell contents.

Topic

HTML

See also

html::TR creates a table row
html::TH creates a table header cell
html::TD creates a table data cell

Example

printnln! TABLE : TR : TH "Name" : TH "Age" : TR : TD "John" : TD "42" : TR : TD "Mary" : TD "25"

Output

<table> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>42</td> </tr> <tr> <td>Mary</td> <td>25</td> </tr> </table>
