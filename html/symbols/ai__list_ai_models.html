ai::list_ai_models

lists the available AI models on the specified AI server

Parameters

options (zero or more)
a list of options

Results

a list of available AI models

Description

Attention: This function must be called with I/O-access rights!

The function sends a request to the specified AI server and returns the list of available AI models.
