std::from_base64

converts a base64 string to a string

Parameter

str
the base64 string

Result

string
the decoded string

Description

If the base64 string is not valid or not a multiple of four characters long, then the special value std::undefined is returned.

Topic

Collections

See also

std::to_base64 converts a string to base64

Example

println! from_base64("SGVsbG8sIHdvcmxkIQ==")

Output

Hello, world!
