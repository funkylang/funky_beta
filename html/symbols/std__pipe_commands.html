std::pipe_commands

pipes a series of commands together

Parameters

feed_or_fd
either a string to feed into the first command or a file descriptor
commands
a list of commands to pipe together

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function pipes a series of commands together.

Topic
