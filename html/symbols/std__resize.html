std::resize

resizes a terminal or screen object

Parameters

self
the terminal or screen object to resize
new_width
the new width of the terminal or screen
new_height
the new height of the terminal or screen

Result

self
the resized terminal or screen object

Description

The function resizes the terminal or screen object to the specified width and height. If the new width or height is smaller than the current width or height, the content of the terminal or screen object is truncated. If the new width or height is larger than the current width or height, the content of the terminal or screen object is padded with spaces.

Example
