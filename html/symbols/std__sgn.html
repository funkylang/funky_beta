std::sgn

returns the sign of its (numerical) argument

Parameter

value
a number

Result

-1, 0, or 1

Description

If the argument is less than zero then -1 is returned.

If the argument equals zero then 0 is returned.

If the argument is greater than zero then 1 is returned.

Math
