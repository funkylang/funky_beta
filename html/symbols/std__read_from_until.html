std::read_from_until

reads data from a file descriptor until a sentinel is found

Parameters

fd
the file descriptor to read from
sentinel
the sentinel to search for
nth
the nth occurrence of the sentinel to search for

Results

the read data or an error

Description

Attention: This function must be called with I/O-access rights!

The function reads data from the specified file descriptor until the specified sentinel is found.

The function returns the read data or an error.
