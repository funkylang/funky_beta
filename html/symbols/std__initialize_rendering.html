std::initialize_rendering

initializes the rendering of widgets

Parameters

screen
the screen to use for rendering

Result

continuation
a continuation that can be used to render widgets

Description

The function initializes the rendering of widgets by setting the template screen to use for rendering.

Topic

Widgets

See also

std::text_box creates a text box widget
std::push_button creates a push button
std::horizontal_group creates a horizontal group of widgets
std::vertical_group creates a vertical group of widgets
std::table_group creates a table group
std::yes_no_dialog creates a dialog with a question and two buttons "Yes" and "No"
std::okay_dialog displays a dialog with a message and an "Ok" button
std::okay_cancel_dialog creates a dialog with an "Ok" and a "Cancel" button

Example

$screen create_screen(80 25) $continuation initialize_rendering(screen) $text_box text_box(continuation "This is a text box") $push_button push_button(continuation "Ok" = OKAY) $horizontal_group horizontal_group(text_box push_button) $vertical_group vertical_group(text_box push_button) $table_group table_group(list(text_box push_button) list(push_button text_box)) $yes_no_dialog yes_no_dialog(continuation "Question" "Is this a question?") $okay_dialog okay_dialog(continuation "Message" "This is a message.") $okay_cancel_dialog okay_cancel_dialog(continuation "Message" "This is a message.")
