ai::tokenize

tokenizes a text string

Parameters

self
the object from which to get the tokenizer
text
the text string to tokenize

Result

tokens
a list of tokens

Description

The tokenizer breaks down the text string into individual tokens.

Topic

AI

See also

Example

$modelname "sauerkrauteml-una-solar-instruc.Q5_K_M.gguf" $prompt "Once upon a time" ai::load_ai_model! $model modelname ai::tokenize! $tokens model prompt print! tokens
