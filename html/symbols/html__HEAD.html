html::HEAD

generates a HEAD element

Parameters

items (zero or more)
the content of the element

Result

a string containing the HEAD element

Description

The function is used as a block function, with the content placed between the HEAD tags.

Topic

HTML

Example

<require basic/stdlib>
<require basic/export/html>

<using std>
<using html>

println!
  HEAD
    : TITLE "My great Homepage"
    : LINK "rel" = "stylesheet" "href" = "style.css"

Output

<head>
<title>My great Homepage</title>
<link rel="stylesheet" href="style.css"/>
</head>
