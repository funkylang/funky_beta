std::is_a_process_id

checks if a value is a process id

Parameter

self
the value to check

Result

true if the value is a process id, false otherwise

Topic

Input-Output

See also

std::process_id constructs a process id from its integer representation

Example

$pid std::create_process "ls" if std::is_a_process_id(pid) println! "pid is a process id" : println! "pid is not a process id"

Implementations

std_types::object indicates that this object is not a process ID
std_types::process_id indicates that this object is a process ID
