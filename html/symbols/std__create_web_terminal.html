std::create_web_terminal

creates a web terminal

Parameters

io
the I/O-object
fd
the file descriptor of the client
message
the message that was received from the client

Result

io
the I/O-object
vt
the virtual terminal

Description

This function is called when a client connects to a server.

The message must start with "DOM@nl;" or "VT@nl;".

If the message starts with "DOM@nl;" then a web terminal is created.

If the message starts with "VT@nl;" then a generic terminal is created.

Topic

Network Clients and Servers

See also

std::create_terminal creates a new terminal object

Example

$io create_io
$server create_server(io "localhost" 8080)
$server.on_connect:
  : (io fd message)
    create_web_terminal io fd message
    -> io
