std::join

concatenates the list items using the specified separator

Parameters

items
the list of items to concatenate
separator
the separator to use (default: " ")

Result

a string which is the concatenation of all the list items separated by
the specified separator

Description

If the list is empty then the empty string is returned.

Topic

Strings

See also

string/split [missing]

Example

$items list("a" "b" "c")
$str join(items ";")
println! str
