std::set_menu

sets the menu

Parameters

window_manager
the window manager
menus
a list of menus

Result

window_manager
the updated window manager

Description

The menu is a list of tuples. Each tuple contains the menu title and a list of menu items. Each menu item is a tuple containing the menu item title and the menu item action.

Example

set_menu &my_window_manager "File" "New" new "Open" open "Save" save "Save as" save_as "Quit" quit "Edit" "Cut" cut "Copy" copy "Paste" paste "Help" "About" about
