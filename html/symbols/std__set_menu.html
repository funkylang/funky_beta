std::set_menu

sets the menu

Parameters

window_manager
the window manager
menu
the menu

Result

window_manager
the updated window manager

Description

The menu is a list of key-value-pairs. The key is the menu item's description and the value is the function to be called when the menu item is selected.

Window Manager

Example

$my_window_manager create_window_manager(terminal) set_menu &my_window_manager "File" -> "New" -> new_file "Open" -> open_file "Save" -> save_file "Save as" -> save_file_as "Quit" -> quit "Edit" -> "Cut" -> cut "Copy" -> copy "Paste" -> paste
