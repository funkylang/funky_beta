std::shrink

shrinks a window

Parameters

window_manager
a window manager
id
the id of the window to shrink

Result

window_manager
the updated window manager

Description

If no id is given then the focused window is shrinked.

If a window is set to MINIMIZED then at least one other window must be set to DEFAULT or MAXIMIZED.
