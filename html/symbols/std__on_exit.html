std::on_exit

registers a function to be called when the task exits

Parameters

self
the task object
handler
the function to be called
arguments
the arguments to be passed to the handler function

Result

self
the task object

Description

The handler function is called when the task exits, regardless of whether the exit is normal or due to an error.
