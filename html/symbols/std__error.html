std::error

creates an error object

Parameters

category (optional)
the category of the error (default: std::APPLICATION_ERROR)
message_text
the error message
details (optional)
additional details (default: std::undefined)

Result

the error object

Description

This function constructs an (application specific) error object. There are no restrictions about the kind of object that details may hold. It's up to the application program to handle this information when displaying the error message.
