std_types::set/debug::to_string

converts the set to a string

Parameters

self
the set to convert

Result

string
the string representation of the set

Description

The string representation of the set is a list of the set's elements enclosed in curly brackets.

Example

$s set "John" 42 "Mary" 25 println! debug::to_string(s)

prints

{"John", 42, "Mary", 25}
