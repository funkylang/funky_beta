std_types::generic_terminal/std::set_text_attributes

sets the text attributes for the terminal

Parameters

terminal
the terminal
attributes
the text attributes

Result

terminal
the updated terminal

Description

The text attributes are a bit mask of the following values:

  • std::BOLD
  • std::ITALIC
  • std::UNDERLINE
  • std::STRIKETHROUGH
