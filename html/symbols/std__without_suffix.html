std::without_suffix

remove a specific expression at the end of a string

Parameters

string
the string to process
suffix
the suffix to remove

Result

string
the string without the suffix

Description

Returns the string without the specified suffix if the string ends with the suffix. Otherwise the string is returned without alteration.

Topic

Strings

See also

std::without_prefix remove a prefix from a string
std::has_suffix checks whether the string has the specified suffix
std::has_prefix checks whether the first argument has the second argument as a prefix
std::before returns the substring before the specified expression
std::truncate_until returns the substring behind the specified expression

Example

$path "/home/user" $base_dir path .without_suffix. "/user" println! base_dir

Output

/home
