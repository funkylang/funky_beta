std::enlarge

enlarges a window

Parameters

window_manager
a window manager
id
the id of the window to enlarge

Result

window_manager
the updated window manager

Description

If no id is given then the focused window is enlarged.

If the window is minimized then it is enlarged.

If the window is not maximized then it is maximized.

If the window is maximized then it is minimized.

Topic
