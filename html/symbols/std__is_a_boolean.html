std::is_a_boolean

determines whether a value is a boolean

Parameter

value # any value

Result

true # if the value is or
false # otherwise

Topic

Boolean

See also

std::true the boolean value "true"
std::false the boolean value "false"

Example

$x true
println! std::is_a_boolean(x)

Output:

  true

$x 5
println! std::is_a_boolean(x)

Output:

  false

Implementations

std_types::false always returns true
std_types::object always returns false
std_types::true always returns true
