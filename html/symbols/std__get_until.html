std::get_until

reads data from the standard input until a sentinel is found and decodes the data to a string

Parameter

sentinel
the sentinel to search for

Results

the read string

Description

Attention: This function must be called with I/O-access rights!

The function reads data from the standard input until the specified sentinel is found and returns the read data as a string. The sentinel is included in the returned string and the string is decoded from UTF-8.
