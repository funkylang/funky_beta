std::failed_attribute_of

retrieves the attribute associated with the error object

Description

This field is used in conjunctions with the "NO SUCH ATTRIBUTE" error.

Parameters

error
the error object

Result

attribute
the attribute associated with the error object

Topic

Errors

See also

std_types::error the prototype object for all error objects
std::error_category_of retrieves the error category of an error object
std::error_message_text_of retrieves the error message text of an error object
std::error_details_of retrieves the details associated with the error object
std::errno_of retrieves the errno associated with the error object

Example

$my_attr_of () $attr my_attr_of("abc") on attr.is_an_error: dump! `error_message_text_of(attr) dump! `failed_attribute_of(attr)

Output

error_message_text_of(attr): "NO SUCH ATTRIBUTE" failed_attribute_of(attr): "my_attr_of"
