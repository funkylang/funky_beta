std::append_or_error

append two collections

Parameters

left
the first collection
right
the second collection

Result

collection
the concatenated collection

Description

If the left collection is an error object then this error object is returned else if the right collection is an error object then this error object is returned otherwise the default std::append method is applied.

Topic

Collections

See also

std::append appends two or more collections to form a single collection

Example

$my_list create_list(1 2 3)
$my_error create_error("error")
$result append_or_error my_list my_error

# result is now the error object my_error

$result append_or_error my_error my_list

# result is now the error object my_error

$result append_or_error my_list create_list(4 5 6)

# result is now the list (1 2 3 4 5 6)
