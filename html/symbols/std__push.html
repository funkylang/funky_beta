std::push

adds a single item at the end of a collection

Parameters

collection
the collection to be modified
item
the item to be added

Result

collection
the modified collection

Topics

Lists
Strings

See also

std::pop removes and returns the last item of a collection
std::peek returns the last item of a collection without removing it
std::drop remove the last item of a collection without returning it

std::append appends two or more collections to form a single collection

Example

$items list(1 2 3)
push &items 4
dump! `items

Output

items: list <4>
  1
  2
  3
  4
