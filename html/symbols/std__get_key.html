std::get_key

wait for a keyboard input event or until the timeout expires

Parameters

timeout
the timeout in seconds (a decimal number)

Results

the key that was pressed or undefined if no key was pressed within the
specified timeout

Description

Attention: This function must be called with I/O-access rights!

The function waits for a keyboard input event or until the timeout expires.

Topic

Terminals

See also

Example

$key get_key(1) if key.is_defined println! "You pressed the key: " key : println! "Timeout!"
