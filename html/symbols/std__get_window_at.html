std::get_window_at

returns the window at the given position

Parameters

window_manager
the window manager
x
x-coordinate
y
y-coordinate

Result

id
the id of the window at the given position
content
the content of the window at the given position
x
the x-coordinate of the character at the given position
y
the y-coordinate of the character at the given position

Description

Returns the window at the given position.

Topic

Window Manager

See also

std_types::window_manager/get_window_at [missing]

Example

$wm create_window_manager(terminal) $window create_window(wm "My Window" CONTENT "Hello, World!") $id $content $x $y get_window_at(wm 10 5) if id.is_defined: println! "Window at position (10,5):" println! " id: " id println! " content: " content println! " character at position (10,5): " println! " x: " x println! " y: " y println! " character: " content(y x) else: println! "No window at position (10,5)" next
