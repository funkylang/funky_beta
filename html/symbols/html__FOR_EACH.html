html::FOR_EACH

iterates over a collection and calls a function for each item

Parameters

items
the collection to iterate over
body
the function to call for each item

Result

text
the concatenated results of the function calls

Description

The function is called with the current item as parameter.

Topic

HTML

See also

html::GROUP groups multiple items into a single item
html::FROM_TO generates a string by calling a function for each index in a range

Example

$names list "John" "Mary" "Irene" println! HTML : BODY : UL : FOR_EACH names : (name) : LI name

Output

<html> <body> <ul> <li>John</li> <li>Mary</li> <li>Irene</li> </ul> </body> </html>
