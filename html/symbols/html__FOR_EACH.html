html::FOR_EACH

generates elements for each item in a collection

Parameters

items
the collection of items
body
a function that generates the element for a given item

Result

a string containing the generated elements

Description

The function is used to generate elements for each item in a collection.

If the body function takes two parameters, it is assumed that the items are key-value pairs.

Topic

HTML

Example

<require basic/stdlib>
<require basic/export/html>

<using std>
<using html>

println!
  FOR_EACH list("apple" "banana" "cherry")
    : (item)
      P "Fruit: @(item)"

Output

<p>Fruit: apple</p>
<p>Fruit: banana</p>
<p>Fruit: cherry</p>
