std::move_window_down

moves a window down

Parameters

wm
a window manager
id
the id of the window to move down

Result

wm
the updated window manager

Description

If the window is the last window then it is moved to the first position.

Topic

Window Manager

See also

std::move_window_up moves a window up

Example

$wm create_window_manager(terminal) $id create_window(wm CONTENT "Hello, World!") move_window_down &wm &id
