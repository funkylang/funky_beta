std::red_component_of

returns the red component of a character

Parameter

chr
the character

Result

red
the red component of the character

Description

The red component is an integer between 0 and 255.

Topic

Colours

See also

std::green_component_of extracts the green component of a character
std::blue_component_of extracts the blue component of a character
std::rgb_character creates a character representing a colour with the given red, green and blue

Example

$my_character create_character(0xff0000) println! std::red_component_of(my_character) # prints 255
