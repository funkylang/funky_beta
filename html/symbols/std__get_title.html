std::get_title

returns the title of a window

Parameters

window_manager
a window manager
id
the id of the window

Result

title
the title of the window

Description

If the window does not exist then "undefined" is returned.

Topic

Window Manager

See also

Example

$my_window_manager create_window_manager(terminal)
$my_window create_window(&my_window_manager "My Window")
println! get_title(&my_window_manager my_window)
