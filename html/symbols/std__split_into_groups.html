std::split_into_groups

splits already indented lines into groups

Parameters

lines
the indented lines
keep_empty_lines
if true then empty lines are kept

Result

groups
a list of indented lines where each line has a subgroups-attribute
which is itself a list of "indented lines"

Description

The division into subgroups is only done on the top-level! The subgroups do not contain any further subgroups.

Topic

Strings

See also

split_into_indented_lines [missing]

Example

$text " a 1 2 3 b 4 5 $lines split_into_indented_lines(text) $groups split_into_groups(lines) dump! 3 `groups dump! 3 `subgroups_of(groups(1))
