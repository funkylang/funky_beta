std_types::object/std::load_configuration

loads the configuration from the given file

Parameters

configuration
the configuration object
filename
the filename of the configuration file
fields*
the fields and their names

Results

configuration
the configuration object
errors
a string containing a list of errors

Description

Attention: This function must be called with I/O-access rights!

In addition to the configuration object, a string containing a list of errors is returned. If this list is empty, then the configuration was successfully loaded and contained no errors.

If only some lines could not be parsed, then the configuration items of the remaining lines are still obeyed.

The fields parameters can either contains multiple key-value-pairs or a a single list of key-value-pairs.
