std_types::object/std::load_configuration

loads a configuration from a file

Parameters

configuration
the configuration object
base_filename
the base filename of the configuration file
fields
the fields and their names

Results

configuration
the configuration object
errors
a list of errors

Description

Attention: This function must be called with I/O-access rights!

The function loads the configuration from the specified file and returns the configuration object and a list of errors (invalid configuration entries).

If there are no configuration errors the list of errors is empty.
