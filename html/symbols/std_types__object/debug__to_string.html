std_types::object/debug::to_string

creates a string representation of the object

Parameters

self
the object to convert
indent
the number of spaces to use for indentation
depth
the maximum recursion depth

Result

string
the string representation of the object

Description

The function creates a string representation of the object.

The string representation is indented by the specified number of spaces.

The function uses introspection to access the object's attributes.

The function can be called without I/O-access rights.
