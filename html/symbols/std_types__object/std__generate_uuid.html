std_types::object/std::generate_uuid

generates and returns a Universally Unique Identifier (UUID)

Parameters

self
the object to generate the UUID for
_kind
how to create the UUID

Result

uuid
the generated UUID

Description

The UUID is a 128-bit number that can be used to identify information in computer systems.

The UUID is generated using one of the following methods:

  • random-based UUID (std::RANDOM_BASED_UUID)
  • time-based UUID (std::TIME_BASED_UUID)
  • name-based UUID (std::NAME_BASED_UUID)

The default is a time-based UUID.
