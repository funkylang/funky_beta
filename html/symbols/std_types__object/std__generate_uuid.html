std_types::object/std::generate_uuid

generates a UUID

Parameter

kind
the kind of UUID to generate (one of std::RANDOM_BASED_UUID, std::TIME_BASED_UUID, or std::NAME_BASED_UUID)

Result

a UUID string

Description

This function must be called with I/O-access rights!

This function generates a UUID and returns it as a string.

Topic

Authentication

See also

std::generate_uuid generates a Universally Unique Identifier (UUID)

Example

<require basic/stdlib>
<require basic/uuid>

std::generate_uuid! $uuid RANDOM_BASED_UUID
println! "UUID: @(uuid)"

Output

UUID: 1659c75a-198c-baee-cd3c-c4da824979cc
