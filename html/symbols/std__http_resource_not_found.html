std::http_resource_not_found

sends a "404 Not Found" reply to a client

Parameters

io
the I/O-object
fd
the file descriptor of the client connection
uri
the URI of the requested resource

Result

none

Description

The function sends a "404 Not Found" reply to the client and closes the connection.

Topic

Network Clients and Servers

See also

std::return_http_resource returns an HTTP resource to a client

Example

$std::start_web_server! &io 8080 $std::handle_web_server_events! &io $events for_each events : (event) event $type $fd $argument case type GET: $uri argument if uri == "/": return_http_resource! &io fd uri "text/html" " <html> <head> <title>Welcome</title> </head> <body> <h1>Welcome</h1> <p>This is the home page.</p> </body> </html> next : http_resource_not_found! &io fd uri next : log &io -> "UNHANDLED EVENT - @(serialize(event))" next next
