debug::to_string

converts an object to a string

Parameters

obj
the object to convert
indent
the indentation level
depth
the maximum recursion depth

Result

str
the string representation of the object

Description

The function converts an object to a string.

The function can be called without I/O-access rights.

The string representation is converted to utf-8 before beeing written.

The optional depth argument specifies the maximum recursion depth used to display the object's attributes.

Topic

Debugging

Example

$name_of ()
$age_of ()
$obj
  std_types::object
    .name_of "Albert"
    .age_of 42
debug::to_string(obj 2 2)

Output

obj: object
  name_of: "Albert"
  age_of: 42
