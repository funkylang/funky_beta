html::DD

creates a std::dd-tag

Parameters

items
the contents of the std::dd-tag

Result

the
-tag

Description

The std::dd-tag is used to describe a term in a std::dl-tag.

Topic

HTML

See also

html::DT creates a definition term
html::DL creates a HTML description list

Example

printnln! HTML : BODY : DL : DT "Funky" : DD "A programming language" : DT "Python" : DD "A programming language"

Output

<html> <body> <dl> <dt>Funky</dt> <dd>A programming language</dd> <dt>Python</dt> <dd>A programming language</dd> </dl> </body> </html>
