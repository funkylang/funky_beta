std::chain

executes a statement sequence multiple times

Parameters

initializers
a list of initializers
body
the statement sequence

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function executes the statement sequence multiple times.

The initializers are executed in the order they are specified.

The statement sequence is executed in the order of the initializers.

Topic

Primitives

See also

std::do executes a statement sequence

Example

chain $i 1 $i 2 $i 3 writeln! i
