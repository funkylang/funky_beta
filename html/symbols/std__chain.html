std::chain

chains multiple initializers

Parameters

initializers (zero or more)
the initializers to chain
body
the body to execute after the initialization

Result

the results returend by the body

Description

This function supports I/O-access. If it is called with I/O-access rights than these access rights are passed on to body.

This function is used to chain multiple initializers.
