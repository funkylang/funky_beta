std::update_terminal

updates the terminal's contents

Parameters

terminal
the terminal to be updated
do_log
if true then log updates to std::stderr

Results

the updated character stream

Description

Attention: This function must be called with I/O-access rights!

The function creates a character stream that updates the terminal's contents to match the contents of the specified std_types::terminal object.

The function uses the following escape sequences:

  • CSI std::row ; std::column H: move cursor to row, column
  • CSI std::n A: move cursor up by n rows
  • CSI std::n B: move cursor down by n rows
  • CSI std::n C: move cursor right by n columns
  • CSI std::n D: move cursor left by n columns
  • CSI std::n K: clear to the end of the line
  • CSI 38 ; 2 ; std::r ; std::g ; std::b m: set text colour to r, g, b
  • CSI 48 ; 2 ; std::r ; std::g ; std::b m: set background colour to r, g, b
