std::show_message

shows a message

Parameters

wm
a window manager
id
the id of the window that opened the message (optional)
timeout
the time in seconds after which the message is removed
screen
the screen to show

Result

wm
the updated window manager

Description

The message is shown in the message area.

If the id is given then the message is removed when the window with the given id is closed.

Topic

Window Manager

See also

std::remove_message removes a message from the window manager
std::set_default_message sets the default message

Example

$my_wm create_window_manager(my_terminal) $my_screen create_screen(my_terminal 40 25) draw_text &my_screen 1 1 "Hello, World!" show_message &my_wm 1 10 &my_screen
