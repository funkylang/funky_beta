std::plot

plots a character at the given coordinates

Parameters

screen # the screen or terminal onto which to plot
x # the x-cordinate of the *half block (zero or more)
y # the y-coordinate of the *half block (zero or more)

Result

screen
the updated screen

Description

The character is plotted at the given coordinates.

The coordinates are one-based.

The character is plotted as a half block character.

If the character is not a half block character, it is plotted as a full block character.

If the character is not a valid character, it is plotted as a space.
