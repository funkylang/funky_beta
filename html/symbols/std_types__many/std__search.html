std_types::many/std::search

searches for the specified expression

Parameters

self
the expression to search for
str
the string to search in
nth
the nth occurrence to search for

Results

pos
the position of the nth occurrence (or std::undefined on fail)
len
the length of the nth occurrence (or std::undefined on fail)

Description

Searches for the nth occurrence of the specified expression in the specified string.

If the search is successful the position and the length of the nth occurrence are returned.

If the search fails then one or two std::undefined-values are returned.
