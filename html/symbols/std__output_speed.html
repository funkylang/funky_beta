std::output_speed

set or get the output speed of a terminal

Parameters

attributes
the terminal attributes
speed
the output speed in bits per second

Result

new_attributes
the new terminal attributes

Topic

POSIX

See also

std::input_speed set or get the input speed
std::get_terminal_attributes retrieve the current terminal attributes
std::set_terminal_attributes update the current terminal attributes

Example

# get the output speed of the terminal speed = std::output_speed(std::get_terminal_attributes(std::stdin)) # set the output speed of the terminal std::set_terminal_attributes(std::stdout, std::output_speed(std::get_terminal_attributes(std::stdout), 9600))
