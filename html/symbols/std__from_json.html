std::from_json

converts a JSON string into a corresponding object

Parameter

text
the JSON string to parse

Result

the parsed object

Topic

Data Exchange

Description

To use this function it is necessary to import the defining module:

std::require basic/std::import/std::json

See also

std::to_json converts a value to a JSON string

Example

$json_text " { "name": "John Doe", "age": 35, "is_student": false, "courses": ["Math", "Physics"], "address": { "street": "123 Main St", "city": "Anytown" } } $data from_json(json_text) debug::dump 99 `data
