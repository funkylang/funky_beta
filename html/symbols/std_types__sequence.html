std_types::sequence

the prototype object for all sequences

Derived from

std_types::list an ordered, enumerated sequence of items

Topic

Collections

Description

A sequence is a list with the subtype "sequence" that is usually used for "syntactically" used lists.

See also

std_types::list an ordered, enumerated sequence of items

Example

$a sequence(1 2 3)
println! a

Output

(1 2 3)
