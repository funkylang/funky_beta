std::get_next_rectangle

returns the next rectangle

Parameters

screen
some kind of screen
rectangle
a rectangle

Result

rectangle
the next rectangle

Description

Returns the next rectangle in the rectangle list.

If the rectangle is undefined then the first rectangle is returned.

If there is no next rectangle then undefined is returned.

Topic

Terminal

See also

std::get_rectangle [missing]
std::get_previous_rectangle returns the previous rectangle

Example

$screen create_screen(40 25)
draw_rectangle &screen 10 5 20 10
draw_rectangle &screen 15 8 10 5
$rect get_rectangle(&screen)
while
  rect.is_defined
  invert &screen rect
  $rect get_next_rectangle(&screen rect)
