std::find_first

returns the first item of a collection that fullfills the condition

Parameters

self
the collection to search in
test
a function that takes an item of the collection as argument and returns true if the item fullfills the condition

Result

item
the first item that fullfills the condition

Description

If no item fullfills the condition then undefined is returned.

Collections

$my_list create_list(1 2 3 4 5) $result std::find_first my_list : (x) x > 3 println! result prints 4
