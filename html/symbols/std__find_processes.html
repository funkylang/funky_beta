std::find_processes

searches for all processes with the specified name

Parameters

name
the name of the process to search for

Results

a list of process ids

Description

The function searches for all processes with the specified name.

Topic

POSIX

See also

std::find_first_process searches for the first process with the specified name
std::get_process_name returns the name of the specified process

Example

find_processes! $pids "bash" for_each pids : (pid) println! pid next!
