std::find_first_process

searches for the first process with the specified name

Parameters

name
the name of the process to search for

Results

the process id of the first process with the specified name or
undefined if no such process exists

Description

Attention: This function must be called with I/O-access rights!

The function searches for the first process with the specified name in the "/proc"-directory.

Topic

POSIX

See also

std::find_processes searches for all processes with the specified name
std::get_process_name returns the name of the specified process

Example

find_first_process! $pid "bash" if pid.is_defined: println! "bash's process id is @(pid)" : println! "bash is not running"
