std::hexdump

writes a hexdump of the given string to stdout

Parameters

str
the string to dump

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function displays the string's hexadecimal representation.

Topic

Debugging

See also

std::ehexdump writes a hexdump of the given string to stderr
std::hexdump_to writes a hexdump of the given string to the given file descriptor
std::dump writes a string representation of some objects to stdout
debug::hexdump writes a hexdump of the given string to stderr

Example

$data "Hello, world!"
hexdump! data
