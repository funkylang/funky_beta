std::serialize

returns a machine readable string representation of the object

Description

If possible the object ist transformed into its source code form.

Topic

Serialization

See also

std::serialization_tag_of returns the serialization tag of the object
std::to_string returns a human readable presentation of the object

Example

$my_string "Hello World" $serialized_string serialize(my_string) dump! `serialized_string

Output

serialized_string: "@quot;Hello World@quot;@0xa;"
