std::deregister_handlers

deregisters handlers for specific event types.

Parameters

Description

self: the input-output object

id: the id of the handler set

types: a list of event types to deregister

Result

self: the updated input-output object

The function deregisters handlers for the specified event types from the handler set with the given id.

Related topics

Input-Output Events

Example

$io std_types::io # create an input-output object register_handlers &io "my_handlers" tuple(READ "read_handler") tuple(WRITE_COMPLETED "write_completed_handler") deregister_handlers &io "my_handlers" "READ" # now only the "write_completed_handler" is registered
