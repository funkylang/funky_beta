std::get_size

returns the size of a window

Parameters

wm
a window manager
id
the id of the window (optional)

Result

width
the width of the window
height
the height of the window

Description

If no id is given then the size of the focused window is returned.

Topic

Window Manager

See also

std::get_xy returns the cursor position
std::get_focused_window returns the content of the focused window
std::get_window_at returns the window at the given position

Example

$wm create_window_manager(terminal) $id create_window(wm CONTENT "Hello, World!") $width $height get_size(wm id)
