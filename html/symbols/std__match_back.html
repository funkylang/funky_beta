std::match_back

matches the end of a string with the expression

Parameters

self
the object from which to get the expression
str
the string which's end is matched

Result

len
the length of the match if the match is successful

Description

If the match is unsuccessful, then len is std::undefined.

Topic

Grammar Objects

See also

Example

$str "12345abcdef" match_back many(DIGIT) str $len dump! `len Output: len: 6
