std::is_an_error

returns true if the argument denotes an error object

Parameter

self
the object to check

Result

true if the object is an error, false otherwise

Description

When an error occurs a function returns a so-called error object (based on std_types::error).

To check whether a function actually did return an error object, std::is_an_error can be used.

Topic

Errors

See also

Example

load! $buf "some-non-existing-file"
on buf.is_an_error:
  println! error_message_text_of(buf)

Output:

  OPEN FAILED

Implementations
