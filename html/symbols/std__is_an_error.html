std::is_an_error

returns true if the argument denotes an error-object

Parameter

self
the object to check

Result

boolean
true if the object is an error, false otherwise

Topic

Errors

Example

load! $buf "some-non-existing-file" on buf.is_an_error: println! error_message_text_of(buf) Output: OPEN FAILED
