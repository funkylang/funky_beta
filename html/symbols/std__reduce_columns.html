std::reduce_columns

reduces the columns of a table

Parameters

self
the table to reduce
reduction
a function that takes two arguments and returns a single value

Result

a table with the same number of rows as the original table, but with only one column

Description

The reduction function is applied to each column of the table, starting with the first column. The result of the reduction function for each column is then placed in a new column.

Topic

Collections

See also

std::reduce_rows reduces a list of lists (a table) row-wise
std::map_reduce_columns applies a map and a reduce function to each column of a table
std::map_reduce_rows applies a map and a reduce function to each row of a table

Example

$my_table create_table( "Name" "Age" "City" "John" 25 "New York" "Jane" 30 "London" "Peter" 20 "Paris" ) $sum_ages reduce_columns my_table (a b) -> a + b println! sum_ages Output: Name Age City John 75 New York
