std::set_timeout

sets a timeout for the event loop.

Parameters

self
the input-output object

Description

timeout: the timeout in seconds

Result

self
the updated input-output object

The timeout is used to limit the time the event loop waits for events. If no events occur within the timeout period, the event loop will terminate.

Related topics

Input-Output Events

Example

$io std_types::io # create an input-output object
set_timeout &io 5 # set a timeout of 5 seconds
loop: # the event loop
  get_events! &io $events # get one or more events
  # ... process the events ...
  if
    timeout_of(io) <= current_time_of(io):
      # timeout occurred
      terminate! # terminate the program
    next # continue processing events
