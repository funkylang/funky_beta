std_types::key_order_set/std::map_reduce

maps and reduces a set into a single value

Parameters

self
the set to work on
map_function
the function to apply to each element of the set
reduce_function
the function to apply to pairs of values
unit
the value to return for empty sets

Result

value
the result of the reduction

Description

The function std::map_function is applied to each element of the set.

The function std::reduce_function is applied to pairs of values.

If the set is empty then std::unit is returned.
