std::draw_character

draws a single character at the specified position on the screen.

Parameters

screen # the screen to draw on
x
the x-coordinate of the character
y
the y-coordinate of the character
chr
the character to draw

Result

screen
the updated screen

Description

The character is drawn using the current draw colour.
