std_types::hash_bag/std::add

adds an item to the hash bag

Parameters

self
the hash bag to which the item will be added
item
the item to add

Result

hash_bag
the updated hash bag

Description

The item is added to the hash bag.

If the item is already in the hash bag then the item's count is increased by one.
