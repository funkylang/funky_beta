std_types::hash_bag/std::add

adds some items to the bag

Parameters

self
the bag to add the items to
pairs
the items to add

Result

bag
the updated bag

Description

The items are added to the bag.

If an item is already in the bag then the count is increased.

If the count of an item becomes zero then the item is removed from the bag.
