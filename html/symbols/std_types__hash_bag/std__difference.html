std_types::hash_bag/std::difference

computes the difference of two hash bags

Parameters

self
the hash bag to subtract from
other
the hash bag to subtract

Result

hash_bag
the difference of the two hash bags

Description

The difference of two hash bags is the hash bag of all items that are contained in the first hash bag but not in the second hash bag.
