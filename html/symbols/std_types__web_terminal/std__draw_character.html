std_types::web_terminal/std::draw_character

draws a character at the specified position

Parameters

self
the screen to draw on
x
x-coordinate of the character
y
y-coordinate of the character
chr
the character to draw

Result

screen
the updated screen

Description

The character is drawn using the current draw colour.

The character is clipped at the screen's borders.

This function also works for "graphical screens" (line_height != 1).
