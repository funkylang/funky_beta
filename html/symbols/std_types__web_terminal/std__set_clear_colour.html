std_types::web_terminal/std::set_clear_colour

sets the clear colour

Parameters

self
the terminal
colour
the new clear colour

Result

terminal
the updated terminal

Description

The clear colour is used to clear the screen.

If the colour is undefined then the clear colour is reset to the default clear colour (black).
