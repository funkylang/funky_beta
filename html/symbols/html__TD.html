html::TD

generates a TD element

Parameters

items (zero or more)
the content of the element

Result

a string containing the TD element

Description

The function is used as a block function, with the content placed between the TD tags.

Topic

HTML

Example

<require basic/stdlib> <require basic/export/html> <using std> <using html> println! TD "Alice"
