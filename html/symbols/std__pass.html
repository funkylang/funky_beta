std::pass

does nothing

Description

pass must be called without any arguments and expecting no results.

It is mostly used as a placeholder for not yet implemented functionality or in places where a function is expected by the caller.

Topic

Basics

See also

std::assign returns all its arguments unaltered
std::spread returns the items of a collection as separate results

Example

$items list("apples" "bananas" "cherries")
for_each items
  : (item)
    println! item
    next!
  pass # <for_each> expects a "finally" function
