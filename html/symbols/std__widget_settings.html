std::widget_settings

the prototype object for widget settings

Description

This object is never used directly but cloned to create a new settings object.

Topics

Widgets

Example

$settings empty_widget_settings !settings.dialog_clear_colour_of BLUE !settings.text_box_clear_colour_of YELLOW !settings.text_box_draw_colour_of RED !settings.push_button_clear_colour_of GREEN !settings.push_button_draw_colour_of BLACK !settings.push_button_shortcut_colour_of MAGENTA
