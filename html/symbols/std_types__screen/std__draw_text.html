std_types::screen/std::draw_text

draws text on the screen

Parameters

self
the screen to draw on
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
args
the text to draw

Result

screen
the updated screen

Description

The text is drawn using the current draw colour.

The text is clipped at the screen's borders.

This function also works for "graphical screens" (line_height != 1).
