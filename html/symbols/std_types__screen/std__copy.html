std_types::screen/std::copy

copies a rectangular area from one screen to another

Parameters

destination
the destination screen
source
the source screen
x
x-coordinate of the upper left corner of the destination area
y
y-coordinate of the upper left corner of the destination area
sx
x-coordinate of the upper left corner of the source area
sy
y-coordinate of the upper left corner of the source area
width
width of the area
height
height of the area

Result

screen
the updated destination screen

Description

The specified area is copied from the source screen to the destination screen.

If the source area is outside the source screen then the destination area is filled with spaces using the the current draw- and clear colour.

If the destination area is outside the destination screen then the source area is ignored.

This function also works for "graphical screens" (line_height != 1).
