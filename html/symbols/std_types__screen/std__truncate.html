std_types::screen/std::truncate

truncates a screen to the specified number of rows and columns

Parameters

self
the screen to be truncated
width
the new width of the screen
height
the new height of the screen

Result

screen
the truncated screen

Description

If the new width or height is greater than the current width or height then the screen is not changed.

If the new width or height is less than the current width or height then the screen is truncated.
