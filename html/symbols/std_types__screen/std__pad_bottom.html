std_types::screen/std::pad_bottom

pads the screen at the bottom

Parameters

self
the screen to pad
height
the new height of the screen
clear_colour
the clear colour to use for the new rows

Result

screen
the padded screen

Description

If the screen's height is less than the specified height then the screen is padded at the bottom with new rows.

The new rows are filled with spaces using the specified clear colour.

If no clear colour is specified then the clear colour of the last row is used.
