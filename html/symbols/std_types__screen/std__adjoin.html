std_types::screen/std::adjoin

adjoins a list of screens to the right of the screen

Parameters

self
the screen to which the other screens will be adjoined
screens
the screens to adjoin

Result

screen
the updated screen

Description

The screens are adjoined to the right of the screen.

The screens must have the same height as the screen.
