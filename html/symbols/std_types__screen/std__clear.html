std_types::screen/std::clear

clears the specified rectangular area

Parameters

self
the screen to be (partially) cleared
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area

Result

screen
the updated screen

Description

The specified area is filled with spaces using the the current draw- and clear colour.

The area is clipped at the screen's borders.

This function also works for "graphical screens" (line_height != 1).
