std_types::screen/std::set_cursor

sets the cursor position

Parameters

self
the screen for which to set the cursor
x
the x-coordinate of the cursor
y
the y-coordinate of the cursor

Result

screen
the updated screen

Description

The cursor position is set to the specified position.

The cursor position is not updated immediately but only when the screen is updated.
