std_types::screen/std::draw_shadow

draws a shadow around a rectangular area

Parameters

self
the screen to draw on
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area
draw_colour
the colour to use for drawing
clear_colour
the colour to use for clearing

Result

screen
the updated screen

Description

The shadow is drawn using the specified draw- and clear colour.

The shadow is clipped at the screen's borders.

This function also works for "graphical screens" (line_height != 1).

Topic

Terminal

See also

std_types::screen/std::set_draw_colour sets the draw colour of the screen
std_types::screen/std::set_clear_colour sets the clear colour of the screen
