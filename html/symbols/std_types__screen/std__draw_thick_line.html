std_types::screen/std::draw_thick_line

draws a thick line between two points

Parameters

self
the screen to draw on
x1
x-coordinate of the first point
y1
y-coordinate of the first point
x2
x-coordinate of the second point
y2
y-coordinate of the second point

Result

screen
the updated screen

Description

The line is drawn using the current draw colour.

The line must be horizontal or vertical.

The line is clipped at the screen's borders.
