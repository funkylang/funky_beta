std_types::screen/std::append

appends two or more screens to form a single screen

Parameters

self
the first screen
screens (zero or more)
the screens to be appended

Result

screen
the resulting screen

Description

The screens to be appended must have the same width as the first screen.

The resulting screen has the same width as the first screen and its height is the sum of the heights of the screens to be appended.
