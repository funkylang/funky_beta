std_types::screen/std::draw_border

draws a border around the specified rectangular area

Parameters

self
the screen to draw on
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area
character
the character to draw the border with
draw_colour
the colour to draw the border with
clear_colour
the colour to clear the background with

Result

screen
the updated screen

Description

The specified area is filled with the specified character using the specified draw- and clear colour.

The area is clipped at the screen's borders.

This function also works for "graphical screens" (line_height != 1).

Topic

Terminal

See also

std_types::screen/std::set_draw_colour sets the draw colour of the screen
std_types::screen/std::set_clear_colour sets the clear colour of the screen
std_types::screen/std::copy copies a rectangular area from one screen to another

Example

$screen std::create_screen() $screen draw_border(screen 1 1 80 24 '*' std::WHITE std::BLUE) $screen draw_text(screen 1 1 "Hello, World!" std::YELLOW std::BLUE) $screen update(screen)
