returns the previous rectangle

Parameters

self
the screen

Result

rectangle
the previous rectangle (or std::undefined if there is no previous
rectangle)

Description

Returns the previous rectangle in the screen's rectangle list.

If the rectangle is not specified then the last rectangle is returned.
