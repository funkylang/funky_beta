std_types::screen/std::scroll_up

scrolls the screen up

Parameters

self
the screen to be scrolled
delta
the number of lines to scroll

Result

screen
the updated screen

Description

The specified number of lines are scrolled up.

The lines that are scrolled out of the screen are lost.

The lines that are scrolled in are filled with spaces using the current clear colour.
