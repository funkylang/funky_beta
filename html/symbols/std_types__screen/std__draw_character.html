std_types::screen/std::draw_character

draws a single character at the specified position

Parameters

screen
the screen to draw on
x
x-coordinate of the character
y
y-coordinate of the character
chr
the character to draw

Result

screen
the updated screen

Description

The character is drawn using the current draw colour.

If the character is a wide character then the character is drawn using the wide character's first code point and the second code point is drawn using the current clear colour.

If the character is outside the screen's borders then it is clipped.
