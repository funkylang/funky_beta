std_types::screen/std::plot

plots a single character at the specified coordinates

Parameters

Description

self: the screen or terminal onto which to plot x: the x-coordinate of the half block y: the y-coordinate of the half block

Result

screen
the updated screen

The character is plotted at the specified coordinates.

The coordinates are in logical screen coordinates.

The character is plotted as a half block character.

If the specified coordinates are outside the bounds of the screen, the screen is returned unchanged.
