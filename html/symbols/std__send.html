std::send

sends a message to one or more tasks

Parameters

self
the task object
destination
the recipient(s) of the message
message
the message to be sent

Result

self
the task object

Description

The message is added to the recipient's inbound message queue.

Topic

Input-Output Task Manager

See also

std::receive receives a message
std::fork creates a new task

Example

$my_task create_task $my_message create_string "Hello, World!" send &my_task my_task my_message
