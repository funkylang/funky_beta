std::print

writes the specified arguments to the standard output

Parameters

args
the arguments to write

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function converts each argument to a string and writes it to the standard output.

Topic

Input-Output

See also

std::println writes the specified arguments to the standard output
std::print_to prints a string to a file descriptor
std::print_string writes the specified string to the standard output
std::print_string_to writes a string to the specified file descriptor

Example

print! "Hello, world!"
