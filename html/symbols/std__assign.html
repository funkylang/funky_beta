std::assign

returns all its arguments unaltered

Parameters

args (zero or more)
the arguments to return

Results

the arguments passed to the function

Description

This function is used to assign multiple values to multiple variables. Because all arguments are evaluated first it implements a parallel assignment and can be used to swap values between variables.
