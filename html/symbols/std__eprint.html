std::eprint

writes the specified arguments to the standard error stream

Parameters

args
the arguments to write

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function converts the arguments to strings and writes them to the standard error stream.

Topic

Input-Output

See also

std::eprintln writes the specified arguments to the standard error output
std::print writes the specified arguments to the standard output
std::println writes the specified arguments to the standard output

Example

eprint! "Hello, world!"
