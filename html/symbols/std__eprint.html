std::eprint

prints its arguments to the standard error output

Parameter

args
the arguments to write

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function converts its arguments to a single string, encodes this string to UTF-8 and writes this encoded string to the standard error output.

Topic

Input-Output

See also
