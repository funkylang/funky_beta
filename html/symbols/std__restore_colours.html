std::restore_colours

resets the text and background colour to the default

Parameters

none

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function sends the ANSI escape sequence "@esc;[39m@esc;[49m" to the standard output.

Topic

Colours

See also

std::ansi_text_colour returns the ANSI escape sequence to set the text colour to the given colour
std::ansi_background_colour returns the ANSI escape sequence to set the background colour to the given
std::ansi_reset_colour returns the ANSI escape sequence to reset the text and background colours

Example

print! ansi_text_colour(RED) "Hello, "
print! ansi_text_colour(BLUE) "World!"
restore_colours!
