std::alt

a grammar object that matches any of the specified expressions

Parameters

expr1, expr2, ... # the expressions to match

Result

len # the number of matched characters or if the match is
unsuccessful

Topic

Grammar Objects

See also

std_types::many the prototype object for "many"-expressions
std_types::some the prototype object for "some"-expressions
std_types::optional the prototype object for optional expressions
std_types::anything the prototype object for the "anything"-grammar object
std_types::any_character_but the prototype object for the "any character but" grammar object
std_types::capture the prototype object for capturing expressions

Example

$str "abcdef" match alt(DIGIT 'a'..'c') str $len dump! `len
