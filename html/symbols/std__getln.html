std::getln

reads a line from the standard input and decodes the data to a string

Parameters

none

Results

the line read from the standard input

Description

Attention: This function must be called with I/O-access rights!

The function reads data from the standard input until a newline character is found and returns the read data as a string. The newline character is included in the returned string and the string is decoded from UTF-8.
