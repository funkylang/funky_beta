std::set_clear_colour_at

sets the clear colour of a character at a given position in a screen

Parameters

screen # the screen object
x
x-coordinate of the character
y
y-coordinate of the character
colour
the new clear colour

Result

screen
the updated screen

Description

The clear colour is used when clearing the screen or parts of it.

Topic

Terminal

See also

std::set_draw_colour_at sets the draw colour of a character at a given position
std::get_clear_colour_at returns the clear colour of a character at a given position in a screen

Example

$my_screen create_screen(terminal 40 25) set_clear_colour &my_screen 10 5 RED # now the character at position (10,5) has a red clear colour

Implementations

std_types::screen sets the clear colour at the specified position
std_types::view sets the clear colour of the specified character
