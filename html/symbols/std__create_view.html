std::create_view

creates a new view based on the given screen

Parameters

screen
the screen to base the view on
left_x
the x coordinate of the left edge of the view
top_y
the y coordinate of the top edge of the view
right_x
the x coordinate of the right edge of the view
bottom_y
the y coordinate of the bottom edge of the view

Result

a new view object

Description

The column described by left_x and the row described by "bottom_y" are included.

Topic

IO

See also

std::create_horizontal_panes creates a set of horizontal panes based on the given screen
std::create_vertical_panes creates a set of vertical panes based on the given screen

Example

$terminal create_terminal 80 24
$view create_view(terminal 1 1 40 24) # left half
