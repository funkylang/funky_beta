std::contains

checks whether a collection contains the specified item or not

Parameters

collection
the collection to search in
item
the item to search for

Result

true
if the item is found in the collection
false
if the item is not found in the collection

See also

std::matches checks whether a string matches the specified expression
std::has_prefix checks whether the first argument has the second argument as a prefix
std::has_suffix checks whether the string has the specified suffix

Example

$my_list = [1 2 3 4 5] if my_list .contains. 3: println! "3 is in the list" : println! "3 is not in the list"
