std::prepare_event_channel

prepares an event channel

Parameters

self
the io task
my_fd
the file descriptor of the terminal
my_options
a set of options
continuation
a continuation function

Result

io
the updated io task

Description

This function prepares an event channel for reading events from the terminal.

The options parameter can contain the following values:

  • TRACK_MOUSE_MOVEMENT: track mouse movements
  • TRACK_WINDOW_SIZE: track window size changes
  • ENABLE_ALL_CTRL_KEYS: enable all control keys
