std::from_html

parses a string containing HTML code

Parameters

text
the text to parse

Result

element
the root element of the HTML document

Description

The function parses the specified text and returns the root element of the HTML document.

Topic

HTML

See also

std::to_html [missing]

Example

$html " <html> <head> <title>My first HTML document</title> </head> <body> <h1>My first HTML document</h1> <p>This is a paragraph.</p> <p>This is another paragraph.</p> </body> </html> " $element from_html(html) dump! 2 `element
