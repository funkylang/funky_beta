std::scroll_left

scrolls the contents of a screen to the left by a given number of characters.

Parameters

screen # the screen to scroll
delta
the number of characters to scroll

Result

screen
the updated screen

Description

The function scrolls the contents of the screen to the left by the specified number of characters. Characters that are scrolled off the left edge of the screen are lost.

Topic

Terminal

See also

std::scroll_right Scrolls the contents of a screen to the right.
std::scroll_up Scrolls the contents of a screen upwards.
std::scroll_down Scrolls the contents of a screen downwards.

Example

$my_screen create_screen(terminal 40 25)
draw_text &my_screen 1 1 "Hello, world!"
scroll_left &my_screen 5
# The text "Hello, world!" will be scrolled 5 characters to the left.
