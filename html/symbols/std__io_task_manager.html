std::io_task_manager

manages the execution of I/O tasks

Parameters

first_task
the first task to be executed
start
a continuation function to be called after the first task has been started
start_arguments
arguments for the start continuation function

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function starts the main loop of the I/O task manager.

Input-Output Task Manager

Example

$task1 ( $io std_types::io write io STDOUT_FILENO "Hello, world!" ) std::io_task_manager task1 (lambda () -> println! "Task finished!")
