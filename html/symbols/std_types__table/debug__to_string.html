std_types::table/debug::to_string

converts the table to a string

Parameters

self
the table to convert

Result

string
the string representation of the table

Description

The string representation of the table is a list of key-value-pairs enclosed in curly brackets.

Example

$table table "John" = 42 "Mary" = 25 "Irene" = 49 println! debug::to_string(table)

Output

{"John" = 42, "Mary" = 25, "Irene" = 49}
