std::create_vertical_panes

creates a list of views that are arranged vertically

Parameters

screen
the screen to create the views on
pane_description
a list of pane widths or an integer that specifies the
number of equal width panes

Result

views
a list of views

Description

The function creates a list of views that are arranged vertically.

If the pane description is an integer then the function creates this number of equal width panes.

If the pane description is a list then the function creates a pane for each element of the list. If an element is an integer then the function creates a pane with this width. If an element is a floating point number then the function creates a pane with a width that is proportional to this number.

Topic

Terminal

Example

$screen create_terminal() $views create_vertical_panes(screen [0.3 0.7]) $left_view views(1) $right_view views(2) draw_text &left_view 1 1 "Left pane" draw_text &right_view 1 1 "Right pane"
