std::trim

removes leading and trailing whitespace

Parameters

self
the string to trim
test
a function that checks whether a character is a whitespace character

Result

the trimmed string

Description

The function returns a string which is the specified string without any leading or trailing whitespace.

Topic

Strings

See also

std::trim_left removes leading whitespace
std::trim_right removes trailing whitespace
std::normalize trims the string and replaces whitespace by a single space

Example

$str "   text   "
$trimmed_str trim(str)
println! '*' trimmed_str '*'

Output

*text*
