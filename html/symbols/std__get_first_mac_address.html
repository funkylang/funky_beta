std::get_first_mac_address

returns the first available MAC address of the system

Parameters

none

Results

mac_address
the MAC address as a 6-byte binary string

Topic

POSIX

See also

std::get_terminal_size retrieves the current width and height of the specified terminal

Example

get_first_mac_address! $mac_address println! "MAC address: " mac_address.to_hex_string
