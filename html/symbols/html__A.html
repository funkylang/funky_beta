html::A

creates an HTML "a"-tag

Parameters

items
the items to be enclosed in the "a"-tag

Result

the HTML "a"-tag

Description

The "a"-tag is an inline tag.

Topic

HTML

See also

html::HTML generates an HTML document
html::B creates a bold HTML tag
html::I creates an HTML "i"-tag
html::SPAN creates a span-element

Example

printnln! HTML : BODY : P : "This is a link to " : A "href" = "https://www.funky-lang.org" "Funky" : "."

Output

<html> <body> <p>This is a link to <a href="https://www.funky-lang.org">Funky</a>.</p> </body> </html>
