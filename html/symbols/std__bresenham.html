std::bresenham

returns an iterator that produces count equally distributed values starting with the first value and ending with the last value.

Parameters

count
the number of values to return
first
the first value to return
last
the last value to return

Result

iterator
an iterator that produces the values

Example

count 5
$start 4
$end 11

$iter bresenham(count start end)
loop:
  next_value &iter $value
  println! value
  if
    value < end
    next
    :
      println! "done"

Output

4
6
7
9
11
done
