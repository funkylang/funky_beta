std::any_character_but

creates grammar object to match anything but the specified expression

Parameters

expr
the expression to match

Result

an "any character but" grammar object

Description

This function creates a grammar object that matches any single character as long as the specified expression does not match.

Topic

Grammar Objects

Example

$str "abc123 def456" match any_character_but(FUNKY_IDENTIFIER) str $length dump! `length
