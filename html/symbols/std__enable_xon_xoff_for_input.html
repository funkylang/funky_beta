std::enable_xon_xoff_for_input

set or clear IXOFF in c_iflag

Parameters

attributes
the terminal attributes
do_set
if true then IXOFF is set, otherwise it is cleared

Result

new_attributes
a copy of the terminal attributes with the changed IXOFF flag

Topic

POSIX

See also

std::enable_xon_xoff_for_output set or clear IXON in c_iflag

Example

# enable XON/XOFF for input # attributes = std::get_terminal_attributes(std::stdin) if attributes != undefined then attributes = std::enable_xon_xoff_for_input(attributes, true) std::set_terminal_attributes(std::stdin, attributes) end if
