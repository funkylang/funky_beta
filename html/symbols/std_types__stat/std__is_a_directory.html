std_types::stat/std::is_a_directory

checks if the file is a directory

Parameters

self
the file information

Results

true if the file is a directory, false otherwise

Description

The function checks if the file is a directory by checking the mode of the file information.

The file information can for a file can be retrieved by calling std::stat.
