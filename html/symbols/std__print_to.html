std::print_to

prints a string to a file descriptor

Parameters

self
the object from which to get the print function
fd
the file descriptor to print to
args*
the arguments to print

Result

fd
the updated file descriptor

Description

The arguments are concatenated into a string and then printed to the specified file descriptor.

Topic

Input-Output

See also

std::write_to writes the specified arguments to the specified file descriptor
std::println_to writes the specified arguments to the specified file descriptor

Example

$fd create_file_descriptor("/dev/stdout") print_to! fd "Hello, world!"
