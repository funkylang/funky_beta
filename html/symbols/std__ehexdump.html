std::ehexdump

writes a hexdump of the given string to stderr

Parameters

str
the string to dump

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function displays the string's hexadecimal representation.

Topic

Input-Output

See also

std::hexdump writes a hexdump of the given string to stdout
std::hexdump_to writes a hexdump of the given string to the given file descriptor
debug::hexdump writes a hexdump of the given string to stderr

Example

$data "Hello, world!" ehexdump! data
