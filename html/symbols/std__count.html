std::count

counts items in a collection

Parameters

collection
the collection to count items in
test
a function to test each item (optional)
argument
an optional argument to pass to the test function

Results

the number of items in the collection that fullfill the specified criterion

Description

If no test function is given the collection must contain boolean values.

If a test function is given, it is called for each item in the collection. If the test function returns true, the item is counted.

If an argument is given, it is passed as the second argument to the test function.

Topic

Collections

See also

std::any_of checks whether at least one item of a collection passes the specified test
std::all_of checks whether all items of a collection pass the specified test

Example

$persons
  hash_table
    "Gregory" = 39
    "Julia" = 37
    "Maria" = 33
    "James" = 41
    "Sarah" = 38

println! count(persons less 40)
