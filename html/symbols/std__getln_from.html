std::getln_from

reads a line from the specified file descriptor

Parameters

fd
the file descriptor to read from

Results

the line read from the file descriptor

Description

Attention: This function must be called with I/O-access rights!

The function reads from the file descriptor until it encounters a newline character.

The newline character is not included in the result.
