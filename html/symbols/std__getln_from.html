std::getln_from

reads a line from the specified file descriptor and decodes the data to a string

Parameter

fd
the file descriptor to read from

Results

the line read from the file descriptor

Description

Attention: This function must be called with I/O-access rights!

The function reads data from the file descriptor until a newline character is found and returns the read data as a string. The newline character is included in the returned string and the string is decoded from UTF-8.
