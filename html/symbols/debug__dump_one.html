debug::dump_one

dumps the contents of a single object

Parameters

obj
the object to dump

Result

none

Description

The object is dumped to the standard error stream.

Topic

Debugging

See also

debug::dump writes a string representation of an object to stderr
debug::hexdump writes a hexdump of the given string to stderr

Example

$x 42 debug::dump_one x

Output

x: int <1> 42
(defined in basic/debug.fky)
(generated by Codestral-22B-v0.1-Q5_K_M.gguf)
(2024-10-01 18:46:25)
(prompt)
