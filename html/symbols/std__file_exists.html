std::file_exists

checks if a file exists

Parameters

filename
the name of the file to check

Results

true if the file exists, false otherwise

Description

Attention: This function must be called with I/O-access rights!

Topic

Input-Output

See also

std::directory_exists [missing]
std::is_a_directory checks if a file is a directory
std::stat retrieves information about a file

Example

file_exists! $does_exist "test.dat" if does_exist: println! "test.dat exists" : println! "test.dat does not exist"
