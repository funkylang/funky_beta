std_types::character/std::match_back

matches the end of a string with the character

Parameters

self
the character to match with
str
the string which's end is matched

Result

len
1 if the character is the last character of the string

Description

If the match is unsuccessful, then len is std::undefined.

Topic

Strings

See also

std_types::character/std::match matches the start of a string with the character

Example

$str "abcdef"
$len match_back('f' str)
dump! `len

Output

len: 1

$len match_back('z' str)
