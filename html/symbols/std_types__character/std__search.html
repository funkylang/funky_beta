std_types::character/std::search

searches for the character from the start or the end of a string

Parameters

self
the character to search for
stream
the string to search in
nth
search for the nth occurrence; negative value search backwards

Results

pos
the position of the first occurrence of the character
len
1 if successful

Description

If the character is not found, then pos and len are std::undefined.

Topic

Strings

See also

std_types::character/std::match matches the start of a string with the character
std_types::character/std::match_back matches the end of a string with the character

Example

$str "abcdef" search $pos $len 'd' str dump! `pos `len

Output

pos: 4 len: 1

Example

$str "abcdef" search $pos $len 'z' str dump! `pos `len

Output

pos: <undefined> len: <undefined>

Example

$str "In the wild, wild west we were alone." search $pos $len 'w' str 2 dump! `pos `len

Output

pos: 14 len: 1

Example

$str "In the wild, wild west we were alone." search $pos $len 'w' str -2 dump! `pos `len

Output

pos: 24 len: 1
