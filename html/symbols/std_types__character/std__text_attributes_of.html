std_types::character/std::text_attributes_of

returns the text attributes of the given rgb-character

Parameters

chr
a rgb-character

Result

the text attributes

Description

A colour is represented as a 24-bit value (256 shades of red, green and blue) and stored as a character. The bits 24..31 of the character are used to store text attributes.

Topic

Colours, Terminal

See also

std::rgb_character creates a character representing a colour with the given red, green and blue
std::red_component_of returns the red component of a character

Example

$BOLD_ITALIC_RED rgb_character(BOLD|ITALIC 255 0 0) $attributes text_attributes_of(BOLD_ITALIC_RED)
