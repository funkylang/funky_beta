std_types::character/std::text_attributes_of

returns the text attributes of the character

Parameters

self
the character

Result

text_attributes
the text attributes of the character

Description

The text attributes of a character are a bitmask.

The following text attributes are defined:

  • 0x01: bold
  • 0x02: faint
  • 0x04: italic
  • 0x08: underline
  • 0x10: blink
  • 0x20: reverse
  • 0x40: invisible
  • 0x80: strikethrough
