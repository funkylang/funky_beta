std_types::character/std::red_component_of

returns the red component of the given rgb-character

Parameters

chr
a rgb-character

Result

the red component (0-255)

Description

A colour is represented as a 24-bit value (256 shades of red, green and blue) and stored as a character. The bits 16..23 of the character are used to store the red component.

Topic

Colours, Terminal
