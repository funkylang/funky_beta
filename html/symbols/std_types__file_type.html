Description

File types are used to identify the type of a file.

The following file types are defined:

  • BLOCK_DEVICE
  • CHARACTER_DEVICE
  • DIRECTORY
  • NAMED_PIPE
  • SYMBOLIC_LINK
  • REGULAR_FILE
  • SOCKET
  • UNKNOWN_FILE_TYPE
