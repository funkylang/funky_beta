std_types::binary_table/std::map_reduce

combines a map and a reduce operation into a single operation

Description

Does not create an intermediate table to hold the "mapped" values.

Parameters

self
the table to work on
map_function
the function to apply to each element of the table
reduce_function
the function to apply to pairs of values
unit
the value to return for empty tables

Result

value
the result of the reduction

Topic

Tables

See also

std_types::binary_table/std::map applies a function to all values of the key order table
std_types::binary_table/std::reduce computes a kind of sum from the values stored in the table

Example

<require basic/stdlib> <require basic/types/collections/tables/binary_table> $ages empty_binary_table !ages("Victor") 23 !ages("Robert") 25 !ages("Eva") 27 println! map_reduce(ages: (name age) -> age)

Output

75
