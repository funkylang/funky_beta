std_types::WHITESPACE/std::match

matches the start of the stream with one or more whitespace characters

Parameters

self
the whitespace object
str
the stream to match against

Result

length
the number of matched characters (or std::undefined on fail)

Description

Attempts to match the start of the stream with one or more whitespace characters.

If the match is successful the length of the match is returned.

If the match fails then std::undefined is returned.
