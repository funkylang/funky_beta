std::command_line_arguments

a list containing the application's command line arguments

Derived from

std_types::list an ordered, enumerated sequence of items

Description

Attention: The first list item is really the first argument.

The name of the current process is stored in std::program_name.

Topicd: Parameters, Environment

See also

std::program_name the name of the current process

Example

for_each command_line_arguments
  : (argument)
    println! argument
    next!
  pass
