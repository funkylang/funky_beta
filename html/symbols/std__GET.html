std::GET

the GET event

Topic

Network Clients and Servers

Description

The GET event is generated when a GET request is received by a web server.

See also

std::start_web_server starts a web server
std::handle_web_server_events handles events from a web server

Example

$io start_web_server! 8080
loop:
  handle_events! $events io
  for_each events
    : (event)
      event $type $fd $uri
      case type
        GET:
          $resource get_resource(uri)
          if
            resource.is_defined:
              return_http_resource! &io fd uri "text/html" resource
              next
            :
              http_resource_not_found! &io fd uri
              next
        :
          next
      next
    next
  next
