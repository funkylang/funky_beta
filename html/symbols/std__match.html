std::match

matches the start of a string with the specified expression

Parameters

expr
the expression to match
str
the string to match against

Result

len
the number of matched characters or std::undefined if the match is
unsuccessful

Topic

Grammar Objects

See also

std::match_back matches the end of a string with the expression
std::search searches for the first occurrence of a grammar object within a string

Example

$str "123abc" match FUNKY_IDENTIFIER str $len dump! `len Output: len: 3 Example: $str "123abc" match DIGIT str $len dump! `len Output: len: 3
