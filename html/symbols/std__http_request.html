std::http_request

sends an HTTP request

Parameters

io
the I/O object
id
a unique identifier for the request
address
the IP address of the server
port_no
the port number of the server
request
the HTTP request

Result

io
the updated I/O object

Description

The function opens a TCP connection to the server, sends the HTTP request, and waits for the server's response.

The response is parsed and returned as a tuple containing the status code, headers, and data.

Topic

Network Clients and Servers

See also

std::parse_http_reply parses a HTTP reply

Example

$my_io create_io
$request "GET / HTTP/1.1@cr;Host: www.example.com@cr;@cr;"
$response http_request &my_io 1 "127.0.0.1" 80 request
if
  response.is_defined:
    $status response(1)
    $headers response(2)
    $data response(3)
    println! "Status: @(status)"
    println! "Headers: @(headers)"
    println! "Data: @(data)"
  :
    println! "Error: @(response)"
