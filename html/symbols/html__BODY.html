html::BODY

generates a BODY element

Parameters

items (zero or more)
the content of the element

Result

a string containing the BODY element

Description

The function is used as a block function, with the content placed between the BODY tags.

Topic

HTML

Example

<require basic/stdlib> <require basic/export/html> <using std> <using html> print! BODY : H1 "My great Homepage" : P "This is my great homepage."

Output

<body> <h1>My great Homepage</h1> <p>This is my great homepage.</p> </body>
