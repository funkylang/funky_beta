ai_types::model/ai::detokenize

converts a token or a list of tokens into a string

Parameters

model
the model to use
token_or_list
a token or a list of tokens
prefix
a list of octets to be prepended to the result

Results

string
the string associated with the token or list of tokens
prefix
the updated prefix

Description

The prefix is a list of octets that is prepended to the result.

The prefix is used to handle incomplete UTF-8 sequences.
