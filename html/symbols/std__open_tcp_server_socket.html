std::open_tcp_server_socket

opens a TCP socket to listen for incoming connections

Parameters

port_no
the port number to listen on
backlog_count
the maximum number of pending connections
reuse = false
reuse the socket, even it is still in TIME_WAIT state

Result

socket
the opened and listening socket

Topic

POSIX

See also

Example
