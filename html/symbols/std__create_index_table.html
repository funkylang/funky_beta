std::create_index_table

creates an index table for the specified collection

Parameters

items
the collection to create an index table for
attribute_of
a function that returns the attribute to use as index

Result

index_table
a hash table that maps the attribute to the index of the item

Description

The function creates a hash table that maps the attribute of each item in the collection to the index of the item.

Topic

Tables

See also

std_types::table the prototype object for all tables
std_types::hash_table a table for fast random access of items
std_types::key_order_table a table which items are stored in key-order
std_types::association_list a very primitive kind of a table

Example

$employees
  list
    object
      name = "John"
      age = 42
    object
      name = "Mary"
      age = 25
    object
      name = "Irene"
      age = 49

$index_table create_index_table(employees : (employee) employee.name)
dump! 2 `index_table

Output

index_table: hash_table <3>
  "Mary"
  1
  "Irene"
  2
  "John"
  0
