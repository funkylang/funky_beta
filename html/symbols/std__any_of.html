std::any_of

checks whether at least one item of a collection passes the specified test

Parameters

collection
the collection which items to check
test
the test function
argument
optional additional argument that is passed to the test function

Result

true if at least one item in the collection fullfills the test

Description

The function returns true if at least one item in the specified collection fullfills the specified test.

The test function is called for each item in the collection until it returns true or all items have been processed.

If the test function returns true for any item, the function returns true. Otherwise it returns false.

Topic

Collections

See also

std::all_of checks whether all items of a collection pass the specified test
std::count counts items in a collection

Example

$persons
  hash_table
    "Gregory" = 39
    "Julia" = 37
    "Maria" = 33
    "James" = 41
    "Sarah" = 38

if
  any_of(persons less 40):
    println! "true"
  :
    println! "false"

Output

true
