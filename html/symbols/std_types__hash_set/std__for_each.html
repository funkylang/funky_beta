std_types::hash_set/std::for_each

iterates over all items of a hash set

Parameters

self
the hash set to iterate over
body
the statement sequence to execute for each item
finally
called after all items have been iterated over

Results

none

Description

The statement sequence "body" is executed for each item of the hash set. The item is available as "item".

The statement sequence "finally" is executed after all items have been iterated over.
