std_types::file_descriptor/std::load

loads all available data from a file descriptor into a string

Parameter

fd
the file descriptor to load

Results

the contents of the specified file descriptor

Description

Attention: This function must be called with I/O-access rights!

The function reads all available data from the specified file descriptor into a string, closes the file descriptor and returns the string.

Topic

Input-Output

Example
