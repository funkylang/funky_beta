std_types::file_descriptor/std::serialize

returns a machine readable string representation of the file descriptor

Parameters

self
the file descriptor to serialize
indent
the indentation level

Result

string
the serialized file descriptor

Description

The serialized file descriptor is a string of the form

file_descriptor std::n

where std::n is the file descriptor number.

If the indentation level is specified then the serialized file descriptor is indented by that amount.
