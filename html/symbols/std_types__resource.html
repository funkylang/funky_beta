std_types::resource

the prototype object for all resources

Description

This object is never used directly but cloned to create a new resource.

Topics

Input-Output Events

See also

std::register_resource registers a resoource with the input-output object.

Example

$io std_types::io $resource std_types::resource register_resource &io "my resource" resource

Methods

std::do_log_of if true then the resource logs all input-output events
