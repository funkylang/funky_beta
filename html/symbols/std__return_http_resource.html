std::return_http_resource

returns an HTTP resource to a client

Parameters

io
the I/O-object
fd
the file descriptor of the client
uri
the URI of the resource
type
the MIME type of the resource
content
the content of the resource

Result

none

Description

The function sends an HTTP reply to the client.

Topic

Network Clients and Servers

See also

std::http_resource_not_found sends a "404 Not Found" reply to a client

Example

$std::start_web_server! &io 8080
$std::handle_events &io
  : (event)
    event $type $fd $argument
    case type
      GET:
        $uri argument
        if
          uri == "/":
            return_http_resource! &io fd uri "text/html" "
              <html>
                <head>
                  <title>Hello, World!</title>
                </head>
                <body>
                  <h1>Hello, World!</h1>
                </body>
              </html>
            next
          :
            http_resource_not_found! &io fd uri
            next
      :
        log &io -> "UNHANDLED EVENT - @(serialize(event))"
        next
    next
