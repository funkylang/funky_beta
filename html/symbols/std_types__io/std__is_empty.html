std_types::io/std::is_empty

checks whether the input-output object is empty

Parameters

self
the input-output object to be checked

Result

empty
true if the input-output object is empty, false otherwise

Description

The input-output object is empty if there are no read jobs, no write jobs, no close jobs and no watched processes.
