std_types::io/std::load

loads a file in some way; returns a string

Parameters

io
the I/O-task manager
id
the job's id
filename
the name of the file to load
max_length
the maximum number of bytes to load

Result

job
a job object

Description

This function starts a job that loads the specified file.

The job's id is the specified id.

If the job is successful then the result is a string containing the file's contents.

If the job fails then the result is an error object.
