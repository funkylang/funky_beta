std_types::io/std::close

closes a file descriptor

Parameters

self
the I/O-object
fd
the file descriptor

Result

self
the updated I/O-object

Description

This closes the specified file descriptor. The I/O-object will generate a std::CLOSED event when the file descriptor is closed.
