std_types::io/std::start_reading_from

starts reading from a file descriptor

Parameters

Description

self: the input-output object fd: the file descriptor to read from do_close: if true the file descriptor will be closed after reading max_length: the maximum number of bytes to read

Result

self
the input-output object

This method starts reading from the specified file descriptor.

If `do_close` is true, the file descriptor will be closed after reading.

If `max_length` is specified, the maximum number of bytes to read is limited.
