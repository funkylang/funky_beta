std_types::io/std::initialize_terminal

initializes a terminal

Parameters

io
the I/O-object to use
options*
the options to use

Results

io
the I/O-object
terminal
the created terminal

Description

The options are:

DRAW_COLOUR = BLACK CLEAR_COLOUR = WHITE FILE_DESCRIPTOR = STDIN_FILENO RECEIVE_CTRL_C_AND_CTRL_Z = false RECEIVE_CTRL_Q_AND_CTRL_S = false MAP_CR = false RECEIVE_SIGWINCH = false RECEIVE_MOUSE_CLICKS = false CURSOR_OFF = false TITLE = undefined

The terminal is initialized using the specified options.

The terminal is finalized when the I/O-object is finalized.
