std_types::io/std::enable_logging

enables logging for the I/O-object

Parameters

self
the I/O-object
level
the log level (default: 1)

Result

self
the updated I/O-object

Description

This enables the logging feature of the I/O-object. All log messages up to the specfied log level will be output.

The lower the log level number the more important is the message. Severe errors have a log level of 1.
