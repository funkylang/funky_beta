std_types::io/std::run

starts a new process

Parameters

Description

self: the input-output object

command: the command to execute

arguments: the arguments to pass to the command

Result

self
the input-output object

This method starts a new process with the specified command and arguments.

The process will be added to the list of running processes.

The process can be waited for using the `std::wait` method.
