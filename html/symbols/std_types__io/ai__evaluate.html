std_types::io/ai::evaluate

evaluates the prompt using the specified model

Parameters

io
the I/O-object to use
id
the job's id
model
the model to use
prompt
the prompt to evaluate
mode
the evaluation mode (default: ai::SMART)

Results

token
the token with the highest probability
confidence_values
a list of tuples (token confidence)

Description

This function is asynchronous.

The result is a tuple (token confidence_values).

The confidence values are the probabilities of the 10 most likely tokens.
