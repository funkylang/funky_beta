std_types::io/std::write

writes data to a file descriptor

Parameters

self
the I/O-object
fd
the file descriptor
data
the data to write

Result

self
the updated I/O-object

Description

This writes the data to the specified file descriptor. The I/O-object will generate a std::WRITE_COMPLETED event when the data has been written.

For each partial write a std::WRITE_PENDING event is sent.

If an error occurs a std::WRITE_ERROR event ist sent.
