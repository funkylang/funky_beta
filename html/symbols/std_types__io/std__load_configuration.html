std_types::io/std::load_configuration

loads a configuration from a file asynchronously

Parameters

io
the I/O-object
id
a caller supplied id to identify the I/O-job
configuration
the configuration object
base_filename
the base filename of the configuration file
fields
the fields and their names

Result

the updated I/O-object

Description

This function adds a job for loading a configuration file to the I/O-object. The job is executed asynchronously within std::get_events.

When the configuration file has been loaded successfully, a std::JOB_COMPLETED-event is generated.

If the configuration file cannot be loaded, a std::JOB_FAILED-event is generated.
