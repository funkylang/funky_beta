std_types::io/std::handle_jobs

handles all jobs that are due

Parameters

io
the input-output object

Result

io
the updated input-output object

Description

This function is called by the system when the input-output object's timeout expires.

It handles all jobs that are due (i.e. all jobs with a scheduled time = the current time).
