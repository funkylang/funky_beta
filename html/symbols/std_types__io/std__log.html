std_types::io/std::log

logs a message

Parameters

self
the input-output object
level
the log level (1-5)
message
the message to log

Result

io
the input-output object

Description

The message is logged if the log level of the input-output object is greater than or equal to the specified log level.

The message is written to STDERR.
