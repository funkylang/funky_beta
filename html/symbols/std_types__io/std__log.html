std_types::io/std::log

logs a message to the I/O-object

Parameters

self
the I/O-object
level
the log level (default: 1)
message
the message to log

Result

self
the updated I/O-object

Description

This logs the given message to the I/O-object. The message is only logged if logging is enabled and the log level is less than or equal to the current log level.
