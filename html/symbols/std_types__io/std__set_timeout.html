std_types::io/std::set_timeout

sets the timeout for the I/O-object

Parameters

self
the I/O-object
timeout
the timeout

Result

self
the updated I/O-object

Description

This sets the timeout for the I/O-object. The timeout can be a number of seconds to wait or an absolute time.

This timeout will be obeyed by the next call to std::get_events.
