std_types::io/std::join_process

waits for the specified process to terminate

Parameters

self
the input-output object
pid
the process id of the process to wait for

Result

io
the updated input-output object

Description

The function returns immediately.

When the process terminates, a std::PROCESS_JOINED-event is injected into the event loop.
