std_types::io/std::save

saves a file

Parameters

self
the I/O-object
filename
the name of the file to save
data
the data to save
context
the context to be passed to the callback function

Result

id
the id of the job

Description

The data is saved to the specified file.

If the file already exists then it is overwritten.

The function returns immediately.

The result of the operation is reported to the callback function.

The callback function is called with the following parameters:

self: the I/O-object id: the id of the job filename: the name of the file context: the context passed to the save function success: a boolean value indicating whether the operation was successful error: the error message (if any)
