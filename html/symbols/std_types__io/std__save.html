std_types::io/std::save

saves a file asynchronously

Parameters

io
the I/O-object
id
a caller supplied id to identify the I/O-job
filename
the name of the file to save
data
the data to save
context
some caller supplied context to attach to the I/O-job (optional)

Result

the updated I/O-object

Description

This function adds a job for saving a file to the I/O-object. The job is executed asynchronously within std::get_events.

When the file has been saved successfully, a std::JOB_COMPLETED-event is generated.

If the file cannot be saved, a std::JOB_FAILED-event is generated.
