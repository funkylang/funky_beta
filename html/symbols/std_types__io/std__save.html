std_types::io/std::save

saves a file in some way

Parameters

io
the I/O-task manager
id
the job's id
filename
the name of the file to save
data
the data to save
context
an optional context object

Result

job
the job's id

Description

The data is saved to the specified file.

If the file already exists then it is overwritten.

If the file does not exist then it is created.

If the file cannot be created then an error is returned.

If the file is saved successfully then the job's id is returned.

If the file cannot be saved then an error is returned.

If the caller requested a context object then it is returned with the job's id.
