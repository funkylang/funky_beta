std_types::io/std::start_catching_signals

starts catching the specified signals

Parameters

self
the input-output object
signals
the signals to catch

Result

io
the updated input-output object

Description

The specified signals are added to the set of catched signals.

If a signal is catched then an event of type std::SIGNAL is injected into the event loop.
