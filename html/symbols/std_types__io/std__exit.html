std_types::io/std::exit

exits the I/O-object

Parameters

self
the I/O-object
exit_code
the exit code

Result

self
the updated I/O-object

Description

This tells the I/O-object to terminate the program with the specified exit code.

All read requests will be cancelled.

Pending write operations will be completed before the program terminates.
