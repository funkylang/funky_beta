std_types::io/std::exit

terminates the current program or task

Parameters

self
the input-output object
exit_code
the exit code to return to the parent process

Result

io
the updated input-output object

Description

The function std::exit terminates the current program or task.

The function std::exit is called by the function std::finalize.

The function std::exit is also called by the function std::get_events if there is no more input to process and the exit code is defined.
