std_types::io/std::run_at

schedules a command to be run at a specific time in the future

Parameters

Description

self: the input-output object

time: the time at which the command should be run (in seconds since the epoch or relative to the current time)

command: the command to be run

arguments: the arguments to be passed to the command

Result

self
the updated input-output object

The command will be added to the list of timed jobs and will be executed when the specified time is reached.
