std_types::io/std::deregister_handlers

deregisters event handlers

Parameters

self
the input-output object
id
the id of the event handlers to be deregistered
types
the event types of the event handlers to be deregistered

Result

io
the updated input-output object

Description

Deregisters the event handlers for the specified event types.

If no event types are specified then all event handlers for the specified id are deregistered.
