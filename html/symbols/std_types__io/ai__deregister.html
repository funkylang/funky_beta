std_types::io/ai::deregister

deregisters a model from the AI-server

Parameters

io
the I/O-object
id
the job id
uuid_or_model
the UUID of the model or the model object
address
the address of the AI-server
port_no
the port number of the AI-server

Result

io
the updated I/O-object

Description

Deregisters the specified model from the AI-server.

If the model is not registered with the AI-server then this function does nothing.
