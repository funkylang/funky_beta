std_types::io/std::register_handlers

Registers event handlers for a specific id.

Parameters

Description

self: the io object

id: the id for which to register handlers

type_handlers: a list of tuples, each containing the event type and the corresponding handler function

Result

self
the io object

Example

$io std_types::io register_handlers io "my_id" tuple("READ" (fd) -> println! "Read from fd: @(fd)") tuple("WRITE_COMPLETED" (fd) -> println! "Write to fd: @(fd) completed")
