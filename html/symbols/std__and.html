std::and

performs a "short cut" and-operation

Parameters

left
must be std::true or std::false
right
a function that evaluates to std::true or std::false

Result

the result of the right argument if the left argument is ,
otherwise

Description

The right argument is only evaluated if the left argument is std::true.
