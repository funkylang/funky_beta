std::set_text_attributes

sets the text attributes of a terminal

Parameters

terminal # some kind of terminal
attributes
the new text attributes

Result

terminal
the terminal with the updated text attributes

Description

The text attributes are a bitmask.

Topic

Terminal

See also

std::BOLD bold text
std::ITALIC italic text
std::UNDERLINE underlined text
std::STRIKETHROUGH strikethrough text

Example

$my_screen create_screen(terminal 40 25) set_text_attributes &my_screen std::BOLD+std::ITALIC draw_text &my_screen 1 1 "Hello World!"
