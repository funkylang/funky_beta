std::which

searches for a file in the directories listed in the PATH environment variable

Parameters

name
the name of the file to search for

Results

the full pathname of the file or undefined if the file was not found

Description

Attention: This function must be called with I/O-access rights!

Topic

Input-Output

See also

std::find_first_process searches for the first process with the specified name
std::find_processes searches for all processes with the specified name
std::get_process_name returns the name of the specified process

Example

which! $pathname "ls" if pathname.is_defined: println! "ls is @(pathname)" : println! "ls not found"
