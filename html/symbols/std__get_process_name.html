std::get_process_name

returns the name of the specified process

Parameters

pid
the process id

Results

the process name or undefined if the process does not exist

Description

Attention: This function must be called with I/O-access rights!

Topic

Input-Output

See also

std::find_first_process searches for the first process with the specified name
std::find_processes searches for all processes with the specified name

Example

get_process_name! $name 1 println! name
