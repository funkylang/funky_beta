std::open_unix_socket

opens a UNIX domain socket

Description

The optional mode parameter is a string of the form "rwxrwxrwx". The default value is "rw-r--r--".

Parameters

filename
the path and name of the socket

Result

socket
the opened and connected socket

Topic

Input-Output

See also

Example
