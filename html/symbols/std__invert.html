std::invert

inverts a rectangular area

Parameters

screen
some kind of screen
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area

Result

screen
the updated screen

Description

The area is inverted using the current draw colour. Use std::set_draw_colour to change the current draw colour.

The area is clipped at the screen's borders.
