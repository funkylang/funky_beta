std::flatten

flattens a collection (probably a list)

Parameters

self
the collection to flatten

Result

a flattened collection

Description

The function flattens a collection by recursively applying itself to all sub-collections.

Topic

Collections

See also

std::zip merges two collections
std::merge merges two collections

Example

$my_list create_list(1 2 create_list(3 4) 5)
flatten &my_list

Output:

1 2 3 4 5
