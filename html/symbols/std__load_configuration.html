std::load_configuration

loads the configuration from the given file

Parameters

self
the object from which to get the configuration
filename
the filename of the configuration file
fields*
the fields and their names

Result

configuration
the configuration object
errors
a list of errors

Description

The fields parameters can either contains multiple key-value-pairs or a a single list of key-value-pairs.

Topic

Configuration

Example

$my_config create_object() load_configuration &my_config "config.txt" "database_host" "localhost" "database_port" 12345
