std::draw_shadow

draws a shadow

Parameters

screen
some kind of screen
factor
the factor by which to darken the colours
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area

Result

screen
the updated screen

Description

The area is clipped at the screen's borders.
