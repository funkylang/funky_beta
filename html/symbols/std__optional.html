std::optional

creates a grammar object to match zero or one occurrence of an expression

Parameters

expr
the expression to match

Result

a grammar object

Description

This function creates a grammar object that matches zero or one occurrence of the given expression.

Topics

Grammar Objects

Example

$str "123"
match optional(DIGIT) str $len
println! len
