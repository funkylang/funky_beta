std_types::int8_array

an array of signed 8-bit numbers

Derived from

std_types::generic_array the prototype object for all kind of arrays

Parameters

indices
the indices of the array elements
value
the new value (only for updates)

Result

value_or_array
the value at the specified position or the updated array

Description

Updates to arrays are delayed until the array is read.

Array indices are one-based.
