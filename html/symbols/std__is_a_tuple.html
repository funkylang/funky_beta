std::is_a_tuple

checks if an object is a tuple

Parameter

self
the object to check

Result

boolean
true if the object is a tuple, false otherwise

Topic

Tuple

See also

std_types::tuple the prototype object for all tuples
std::first returns the first element of a tuple
std::second returns the second element of a tuple

Example

$my_tuple create_tuple(1 2 3)
println! is_a_tuple(&my_tuple) # prints true

Implementations

std_types::object always returns false
std_types::tuple always returns true
