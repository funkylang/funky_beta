std_types::tuple/std::serialize

returns a machine readable string representation of the tuple

Parameters

self
the tuple to be serialized
indent
the indentation level

Result

string
the serialized tuple

Description

The serialized tuple is a string of the form

tuple std::n item1 item2 ... itemn

where "n" is the number of items in the tuple and "item1", "item2", ..., "itemn" are the serialized items.

If the indentation level is specified then the serialized tuple is indented by that number of spaces.
