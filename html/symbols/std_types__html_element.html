std_types::html_element

the prototype object for HTML elements

Parameters

tag
the tag name of the element
attributes
the attributes of the element
contents
the contents of the element

Result

html_element
the created HTML element

Description

This object is never used directly but cloned to create a new HTML element.

Topics

HTML

See also

std::from_html parses a string containing HTML code

Example

$html_element from_html "<html> <head> <title>My first HTML page</title> </head> <body> <h1>Hello, World!</h1> </body> </html>" println! html_element

Output

<html> <head> <title>My first HTML page</title> </head> <body> <h1>Hello, World!</h1> </body> </html>
