std::realpath

returns the canonicalized absolute pathname

Parameters

pathname
the path to canonicalize

Result

path
the canonicalized absolute pathname

Description

On success a std_types::string is returned, on failure an appropriate error-object is returned.

This function needs I/O-access rights.

Topic

POSIX

See also

std::chdir changes the current working directory
std::getcwd returns the current working directory

Example

realpath! $path "/etc/../etc/passwd" println! path
