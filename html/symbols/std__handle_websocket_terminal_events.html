std::handle_websocket_terminal_events

handles events for websocket terminals

Parameters

io
the I/O-object
events
the events to handle

Results

the remaining events

Description

Attention: This function must be called with I/O-access rights!

This function handles the following events:

  • UPGRADE_TO_WEBSOCKET
  • WEBSOCKET_MESSAGE
  • WEBSOCKET_CLOSED

Topic

Network Clients and Servers

Example

$io create_io
$events
  list
    UPGRADE_TO_WEBSOCKET = "..."
    WEBSOCKET_MESSAGE = "..."
    WEBSOCKET_CLOSED = "..."
handle_websocket_terminal_events io events
