std::difference

computes the difference of two hash bags

Parameters

hash_bag1
the first hash bag
hash_bag2
the second hash bag

Result

hash_bag
the difference of the two hash bags

Description

Returns a hash bag containing all items of the first hash bag that are not contained in the second hash bag.
