std::print_string_to

writes a string to the specified file descriptor

Parameters

fd
the file descriptor to write to
str
the string to write

Results

none

Description

Attention: This function must be called with I/O-access rights!

The function writes the string to the specified file descriptor.

Topic

Input-Output

See also

Example

print_string_to! STDOUT_FILENO "Hello, world!"
