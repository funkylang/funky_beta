std::pipe

runs a pipe of commands

Parameters

args
a list of commands and arguments

Results

a list of exit codes

Description

Attention: This function must be called with I/O-access rights!

The function runs a pipe of commands.

The first argument may be a string or a file descriptor.

If the first argument is a string then it is fed to the first command.

If the first argument is a file descriptor then it is used as the input file descriptor for the first command.

The remaining arguments are commands and arguments.

A command is a function that is called with the arguments that follow the command.

The output of a command is fed to the next command.

The last command's output is returned as a string.

The last command's error output is returned as a list of strings.

Topic

POSIX

See also

std::pipe_commands pipes a series of commands together
std::call calls the specified command with the specified arguments

Example

$cat: ( args* ) for_each args : (arg) print! arg next -> undefined $grep: ( args* ) $pattern args(1) for_each args : (arg) if arg .matches. pattern: println! arg next next -> undefined pipe! "Hello, world!" cat grep "world"
