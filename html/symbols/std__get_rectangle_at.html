std::get_rectangle_at

returns the rectangle at the given position

Parameters

screen
some kind of screen
x
x-coordinate
y
y-coordinate

Result

rectangle
the rectangle at the given position or undefined

Topic

Terminal

See also

std::get_rectangle_by_key returns a rectangle by its key
std::get_next_rectangle returns the next rectangle
std::get_previous_rectangle returns the previous rectangle

Example

$my_screen create_screen(terminal 40 25) $rectangle get_rectangle_at(&my_screen 10 10) if rectangle.is_defined println! "Rectangle at (10,10): " rectangle
