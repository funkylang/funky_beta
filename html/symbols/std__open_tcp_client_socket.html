std::open_tcp_client_socket

opens a TCP socket and connects to the given host and port

Parameters

uri
the universal resource identifier of the socket
port_no
the port number to connect to

Result

socket
the opened and connected socket

Topic

POSIX

See also

std::open_tcp_server_socket opens a TCP socket to listen for incoming connections
std::open opens a file
std::close closes a resource
std::read a low level read function
std::write writes data to a stream

Example

open_tcp_client_socket! $fd "funkylang.org" 80
write! $_bytes_written fd "
  GET / HTTP/1.1@cr;
  Host: funkylang.org@cr;
  Connection: close@cr;
  @cr;
loop
  :
    read! fd $buf 4096
    if
      buf.is_empty:
        close! fd
        println!
      :
        print! buf
        next!
