std_types::anything/std::search

always succeeds

Parameters

self
the grammar object
str
the string to search
nth
the nth occurrence to find (irrelevant)

Result

start
always 1
length
the length of the whole string

Description

This method always succeeds and returns the start (1) and length of the string within to search.

To search for "anything" is more or less useless and only supplied for completeness.

The std::ANYTHING grammar object ist normally used within a sequence of grammar objects.

Example

$str "This is a text."
search $pos $len ANYTHING str
dump! `pos `len

len: 15
