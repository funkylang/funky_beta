std_types::anything/std::match_back

matches the end of the string with the specified expression

Parameters

self
the expression to match
str
the string to match with

Result

len
the number of matched characters (or std::undefined on fail)

Description

Attempts to match the end of the string with the specified expression.

If the match is successful the length of the match is returned.

If the match fails then std::undefined is returned.
