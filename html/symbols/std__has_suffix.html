std::has_suffix

checks whether the string has the specified suffix

Parameters

self
the string to check
suffix
the suffix to check for

Result

true if the string has the specified suffix, false otherwise

See also

std::has_prefix checks whether the first argument has the second argument as a prefix

Example

$path "/home/user/" if path .has_suffix. '/': println! "path ends with a slash" : println! "path does not end with a slash"
