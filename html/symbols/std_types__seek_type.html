std_types::seek_type

the prototype object for all seek types

Derived from

std_types::object the prototype object for all regular objects

Parameters

value
the new value (only for updates)

Result

value_or_seek_type

Description

Seek types are used to specify the position for the "lseek" function.

Topics

POSIX

See also

std::lseek repositions the read/write file offset of the open file description
