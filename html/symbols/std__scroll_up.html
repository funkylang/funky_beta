std::scroll_up

Scrolls the contents of a screen upwards.

Parameters

screen
the screen to scroll
delta
the number of lines to scroll (default: 1)

Result

screen
the updated screen

Description

The function scrolls the contents of the screen upwards by the specified number of lines. New lines are added at the bottom of the screen, filled with the screen's default clear colour.

Topic

Terminal

See also

Example

$my_screen create_screen(terminal 40 25)
scroll_up &my_screen 2
