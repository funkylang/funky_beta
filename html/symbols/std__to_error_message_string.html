std::to_error_message_string

converts an error object to a string

Parameter

err
the error object

Result

the error message as a string

Description

The function converts an error object to a string. The error message is composed of the error message text, the error number and the error details.

Topic

Errors

See also

std::errno_of the errno associated with an error object

Example

$err error("An error occurred")
println! to_error_message_string(err)
