std::get_mac_address

returns the MAC-address of the first network interface

Parameters

none

Results

the MAC-address as a string of hexadecimal digits or undefined

Description

Attention: This function must be called with I/O-access rights!

Topic

POSIX

See also

std::get_process_name returns the name of the specified process
std::find_processes searches for all processes with the specified name
std::find_first_process searches for the first process with the specified name

Example

get_mac_address! $mac if mac.is_defined: println! "MAC-address: @(mac)" : println! "No MAC-address found"
