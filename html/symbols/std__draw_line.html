std::draw_line

draws a line on a screen

Parameters

self
the screen to draw on
x1
x-coordinate of the starting point
y1
y-coordinate of the starting point
x2
x-coordinate of the ending point
y2
y-coordinate of the ending point

Result

screen
the updated screen

Description

The line is drawn using the current draw colour.

The line is clipped at the screen's borders.

Topic

Terminal

See also

std::draw_text draws text on a screen
std::draw_rectangle [missing]
std::clear clears a rectangular area

Example

$my_screen create_screen(terminal 40 25) set_draw_colour &my_screen GREEN draw_line &my_screen 1 1 20 10
