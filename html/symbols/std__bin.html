std::bin

converts a positive integer value into a string consisting of binary digits

Parameters

value
the value to convert
size
the minimum number of digits to generate (optional)

Result

a string consisting of binary digits

Description

The optional second parameter specifies the minimum length of the resulting string. If there are fewer binary digits than the specified length, the string is left-padded with zeros.
