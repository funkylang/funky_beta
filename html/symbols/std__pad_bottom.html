std::pad_bottom

pads a screen to the specified height

Parameters

screen
some kind of screen
height
the desired height
clear_colour
the clear colour to use

Result

screen
the updated screen

Description

The screen is padded at the bottom using the current clear colour. Use std::set_clear_colour to change the current clear colour.

Topic

Terminal
