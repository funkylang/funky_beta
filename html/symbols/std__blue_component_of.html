std::blue_component_of

extracts the blue component of a character

Parameter

chr
a character

Result

blue
the blue component of the character

Description

The blue component is the least significant byte of the character.

Topic

Colours

See also

std::red_component_of returns the red component of a character
std::green_component_of extracts the green component of a character
std::rgb_character creates a character that represents a colour

Example

$my_character create_character(0xff00ff) $blue std::blue_component_of(my_character) println! blue # prints 255
