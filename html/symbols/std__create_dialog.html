std::create_dialog

creates a dialog

Parameters

window_manager
the window manager
title
the title of the dialog
content
the content of the dialog
buttons
the buttons of the dialog
options
the options of the dialog

Result

window_manager
the updated window manager

Description

The dialog is created with the following default options:

  • std::USE_CURSOR
  • std::DO_NOT_FOCUS
  • std::CONTENT
  • std::CLOSE_BUTTON
