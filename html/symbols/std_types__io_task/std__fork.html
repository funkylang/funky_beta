std_types::io_task/std::fork

forks a new task

Parameters

self
the task that forks a new task
task
the new task
continuation
the function to call when the new task has terminated
argument
the argument to pass to the continuation function

Result

self
the updated task

Description

The new task is started immediately.

The continuation function is called with the following arguments:

  • the new task
  • the argument passed to std::fork
  • the exit status of the new task
