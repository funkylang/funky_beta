std_types::io_task/std::prepare_event_channel

prepares an event channel for reading keyboard and mouse events

Parameters

self
the I/O-task
my_fd
the file descriptor to read from
my_options
a list of options
continuation
the function to call when the event channel is ready

Result

io
the I/O-object

Description

This function prepares an event channel for reading keyboard and mouse events.

The event channel is ready when the function std::continuation is called.

The following options are supported:

  • std::TRACK_MOUSE_MOVEMENT: enables mouse movement tracking
  • std::TRACK_WINDOW_SIZE: enables window size tracking
  • std::ENABLE_ALL_CTRL_KEYS: enables all control keys
