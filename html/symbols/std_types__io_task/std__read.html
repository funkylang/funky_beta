std_types::io_task/std::read

reads at least the specified number of octets

Parameters

self
the task that performs the I/O operation
my_fd
the file descriptor to read from
my_length
the number of octets to read
continuation
the function to call when the I/O operation is completed

Result

the result of the I/O operation is returned to the continuation function

Description

This function starts an asynchronous I/O operation.

The result of the I/O operation is returned to the continuation function.

The continuation function is called with the following parameters:

task: the task that performed the I/O operation io: the I/O object that performed the I/O operation result: the result of the I/O operation

The result of the I/O operation is a string containing the read octets.

If the I/O operation fails then the result is an error object.

If the I/O operation is cancelled then the result is the string containing the octets that have been read so far.
