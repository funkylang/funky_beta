std_types::io_task/std::wait_for_message

waits until at least one message is ready to be received

Parameters

self
the task to wait for
continuation
the function to call when the task is ready to receive

Result

request
the request to be processed

Description

This function is called by a task to wait until at least one message is ready to be received.

The task is suspended until at least one message is ready to be received.

The task is resumed by calling the continuation function.
