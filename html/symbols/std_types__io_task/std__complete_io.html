std_types::io_task/std::complete_io

completes the check of an I/O operation

Parameters

self
the task that performs the I/O operation
body
the function to call when the I/O operation is completed
argument
the argument to pass to the function

Result

task
the updated task

Description

This function is called by an I/O-function to complete the check of an I/O operation.

The function std::body is called with the argument std::argument when the I/O operation is completed.
