std_types::io_task/std::join_process

waits for the specified process to terminate

Parameters

self
the task that wants to wait for the process
my_pid
the process id of the process to wait for
continuation
the function to call when the process has terminated

Result

task
the updated task

Description

The specified task is updated to wait for the specified process to terminate.

When the process has terminated the specified continuation is called with the task and the exit status of the process as arguments.
