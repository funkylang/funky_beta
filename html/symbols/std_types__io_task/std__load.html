std_types::io_task/std::load

loads a file in some way; returns a string

Parameters

self
the task that performs the I/O operation
my_filename
the name of the file to load
continuation
the function to call when the I/O operation is completed

Result

the task that performs the I/O operation

Description

The file is loaded asynchronously.

When the I/O operation is completed the continuation is called with the task as first argument and the result as second argument.

If the I/O operation fails then the result is an error object.
