std_types::io_task/std::exit

terminates the current task

Parameters

self
the task to terminate
status
the exit status (default: EXIT_SUCCESS)

Result

none

Description

This function terminates the current task.

If the task is the main task then the program is terminated.

If the task is not the main task then the task's parent is notified.
