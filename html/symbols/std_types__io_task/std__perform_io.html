std_types::io_task/std::perform_io

starts an I/O-operation

Parameters

self
the task that performs the I/O-operation
body
the function to call when the I/O-operation is started
argument
the argument to pass to the function std::body
continuation
the function to call when the I/O-operation is completed

Result

request
the request object

Description

The function std::body is called when the I/O-operation is started.

The function std::body must call std::performed_io when the I/O-operation is completed.

The function std::continuation is called when the I/O-operation is completed.

The function std::continuation is called with the following arguments:

  • the task object
  • the result of the I/O-operation
