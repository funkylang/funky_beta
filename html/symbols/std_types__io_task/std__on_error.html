std_types::io_task/std::on_error

sets the error handler for the task

Parameters

self
the task
handler
the error handler

Result

task
the updated task

Description

The error handler is called if an error occurs in the task.

The error handler is called with the task object as the first argument and the error object as the second argument.

The error handler must return a value that is interpreted as the result of the task.

If the error handler is not set then the error is propagated to the parent task.
