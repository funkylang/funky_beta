std_types::io_task/std::receive

receives a message

Parameters

self
the task object

Results

task
the task object
message
the received message (or std::undefined if no message is available)

Description

If a message is available then the message is removed from the task's inbound message queue and returned.

If no message is available then std::undefined is returned.
