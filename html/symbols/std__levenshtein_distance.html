std::levenshtein_distance

computes the Levenshtein distance between two strings

Parameters

s
the first string
t
the second string

Result

the Levenshtein distance between the two strings

Description

The Levenshtein distance is a measure of the difference between two strings.

It is the number of characters that need to be substituted, inserted, or deleted, to transform s into t.
