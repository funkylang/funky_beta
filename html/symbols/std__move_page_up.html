std::move_page_up

moves the first visible row of a window up by one page

Parameters

window_manager
a window manager
id
the id of the window

Result

window_manager
the updated window manager

Description

If the id is omitted then the focused window is used.

Topic

Window Manager

Example

$my_window_manager create_window_manager(terminal) $my_window create_window(&my_window_manager "My Window") move_page_up &my_window_manager
