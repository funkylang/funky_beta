html::SPAN

creates a span-element

Parameters

items
the items to be contained in the span-element

Result

the span-element

Description

The span-element is an inline element that does not cause a line break.

Topic

HTML

See also

html::I creates an HTML "i"-tag
html::B creates a bold HTML tag

Example

printnln! HTML : BODY : P : "This is a " : SPAN "red" "style" = "color: red;" "text" : "."

Output

<html> <body> <p>This is a <span style="color: red;">red</span> text.</p> </body> </html>
