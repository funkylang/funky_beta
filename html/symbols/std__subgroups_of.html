std::subgroups_of

returns a list of subgroups of the specified list

Description

The subgroups are defined as contiguous sublists of the input list.

The function returns a list of tuples. Each tuple contains two elements:

  • The first element is the starting index of the subgroup.
  • The second element is the ending index of the subgroup.

The function assumes that the input list is already sorted.

Example

$list list(1 2 3 4 5 6) $subgroups subgroups_of(list) dump! 2 `subgroups
