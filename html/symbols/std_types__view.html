std_types::view

a view of a screen

Derived from

std_types::generic_terminal a generic terminal object

Description

A view is a rectangular region that restricts drawing operations on their parent screen.

All drawing operations occur on a screen (not the view).

To draw using a view both the view's parent screen and the view must be specified.
