std_types::view

the prototype object for views

Parameters

screen
the screen to use for drawing
x
the x-coordinate of the upper left corner of the view
y
the y-coordinate of the upper left corner of the view
width
the width of the view
height
the height of the view

Result

view
the created view

Description

This object is never used directly but cloned to create a new view.

Topics

Terminal

See also

std::create_view creates a view on a terminal

Example

$screen create_terminal $view create_view(&screen 2 2 10 5) draw_text &view 2 2 "Hello World!"

Output

+----------+ | | | Hello | | World! | | | +----------+
