std::has_prefix

checks whether the first argument has the second argument as a prefix

Parameters

string
the string to check
prefix
the prefix to check for

Result

boolean
true if the string has the prefix, false otherwise

Topic

Collections

See also

std::has_suffix checks whether the string has the specified suffix
std::without_prefix remove a prefix from a string
std::contains checks whether a collection contains the specified item or not

Example

$path "/home/user/" if path .has_prefix. "/home/": println! "path starts with /home/" : println! "path does not start with /home/"
