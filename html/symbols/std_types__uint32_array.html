std_types::uint32_array

an array of unsigned 32-bit numbers

Derived from

std_types::generic_array the prototype object for all kind of arrays

Parameters

indices
the indices of the array elements
value
the new value (only for updates)

Result

value_or_array
the value at the specified position or the updated array

Description

Updates to arrays are delayed until the array is read.

Array indices are one-based.

Topics

Arrays

See also

std_types::array an array of objects
std_types::int32_array an array of signed 32-bit numbers

Example

$a uint32_array(3 2) # a new array with 3 rows and 2 columns !a(2 1) 47 # use syntactic sugar to assign a value to an array element !a a(3 2 11) # without syntactic sugar println! a(2 1) println! a(3 2)

Output

47 11
