ai::detokenize

detokenizes a token or a list of tokens

Parameters

model
the model to use
token_or_list
the token or list of tokens to detokenize
prefix
a prefix to add to the detokenized string

Result

string
the detokenized string

Description

The function returns the detokenized string.

Topic

AI

See also

$modelname "sauerkrautem-una-solar-instruc.Q5_K_M.gguf" $prompt "Once upon a time" ai::load_ai_model! $model modelname ai::tokenize! $tokens model prompt ai::dettokenize! $piece model tokens print! piece
