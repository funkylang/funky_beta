std::error_check

raises an error if any of its arguments is an error object

Description

This function does not need I/O-access rights.

When error objects are stored into variables or collections it might take some time until the error is propagated. error_check helps in speeding up the error propagation.

Topic

Errors

See also

std::try executes the specified body and catches any error

Example

do: load! $data_1 filename_1 load! $data_2 filename_2 error_check data_1 data_2 println! "should only be printed if both files were loading successfully"
