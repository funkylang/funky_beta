std::initialize_logging

initializes logging for the I/O-object

Parameters

io
the I/O-object
log_level
the log level (default: 3)
log_filename
the file to log to (optional)

Result

io
the updated I/O-object

Description

This initializes logging for the I/O-object. The log level is set to the specified value. If a log filename is specified, the log is written to that file. Otherwise, the log is written to the standard error output.
