std::initialize_logging

initializes the logging system

Parameters

io
the input-output object
log_level
the log level (default: 3)
log_filename
the log filename (default: undefined)

Results

the input-output object

Description

Attention: This function must be called with I/O-access rights!

If the log filename is undefined then the log output is written to STDERR.

Topic

Input-Output Events

See also

std::enable_logging Enables logging for the given input-output object.
std::disable_logging disables logging for the given input-output object.
std::log logs a message to the standard error stream.

Example

$io std_types::io # create an input-output object initialize_logging &io 3 "my_program.log"
