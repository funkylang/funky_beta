html::DIV

generates a DIV element

Parameters

items (zero or more)
the content of the element

Result

a string containing the DIV element

Description

The function is used as a block function, with the content placed between the DIV tags.

Topic

HTML

Example

<require basic/stdlib>
<require basic/export/html>

<using std>
<using html>

println!
  DIV
    : P "This is my great homepage."

Output

<div>
<p>This is my great homepage.</p>
</div>
