std::get_draw_colour_at

returns the draw colour of the character at the given coordinates.

Parameters

screen # some kind of screen
x
x-coordinate of the character
y
y-coordinate of the character

Result

colour
the draw colour of the character

Related topics

Terminal

See also

std::set_draw_colour_at sets the draw colour of a character at a given position
std::get_clear_colour_at returns the clear colour of a character at a given position in a screen

Example

$my_screen create_screen(terminal 40 25)
set_draw_colour &my_screen RED
draw_text &my_screen 10 5 "Hello, world!"
get_draw_colour_at &my_screen 10 5 # returns RED
