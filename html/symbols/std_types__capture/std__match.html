std_types::capture/std::match

matches the start of the string with the specified expression and captures the matched string

Parameters

self
the capture object
str
the string to match

Results

len
the number of matched characters (or std::undefined on fail)
capture
the matched string (if requested)

Description

Attempts to match the start of the string with the specified expression.

If the match is successful the length of the match is returned.

If the caller requested two result values then in addition to the string length the string itself ("the capture") is also returned.

If the match fails then one or two std::undefined-values are returned.
