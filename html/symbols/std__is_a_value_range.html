std::is_a_value_range

checks if an object is a value range

Parameter

self
the object to check

Result

true if the object is a value range, false otherwise

Topic

Collections

See also

std_types::value_range the prototype object for all value-range-objects
std_types::object the prototype object for all regular objects

Example

$my_range 'a'..'z'
println! std::is_a_value_range(my_range)

Output:

true

Implementations

std_types::object always returns false
std_types::value_range always returns true
