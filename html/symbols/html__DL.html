html::DL

creates a HTML description list

Parameters

items
a list of items

Result

a string containing the HTML code

Description

The items are expected to be key-value-pairs.

Topic

HTML

See also

html::DT creates a definition term
html::DD creates a std::dd-tag

Example

printnln! DL "Name" = "John Doe" "Age" = 42 "Address" = "123 Main St."

Output

<dl> <dt>Name</dt> <dd>John Doe</dd> <dt>Age</dt> <dd>42</dd> <dt>Address</dt> <dd>123 Main St.</dd> </dl>
