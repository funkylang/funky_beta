std::on_websocket_message

sets the websocket message handler for a websocket connection

Parameters

io
the I/O-object
fd
the file descriptor of the websocket connection
handler
the websocket message handler

Result

io
the I/O-object

Description

If the file descriptor is undefined then the default websocket message handler is set.

The websocket message handler is called with the following parameters:

io: the I/O-object fd: the file descriptor of the websocket connection message: the message received from the websocket connection

Topic

Network Clients and Servers

See also

std::on_key_press [missing]
std::on_mouse_click [missing]

Example

$io create_io
$server create_server(io 8080)
$server.on_websocket_message:
  : (io fd message)
    log io 1 "got websocket message from @(fd.serialize): @(message)"
    -> io
