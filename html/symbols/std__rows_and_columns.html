std::rows_and_columns

computes the number of rows and the length of the longest row of a multi-line text

Parameters

text
the text to analyze

Result

rows
the number of rows
longest
the length of the longest row

Description

The text is interpreted as a multi-line text. The number of rows is the number of newline characters plus one. The length of the longest row is the number of characters between two newline characters.

Topic

Strings

See also

std::wrap_words wraps a string consisting of words onto multiple lines

Example

$str "This is a multi-line\ntext." $rows $longest rows_and_columns(str) println! "rows = " rows ", longest = " longest

Output

rows = 2, longest = 17
