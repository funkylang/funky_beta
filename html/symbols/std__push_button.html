std::push_button

creates a push button

Parameters

text_and_key
a tuple consisting of the text to display and the key
to return when the button is activated

Result

screen
a screen object representing the push button

Description

The push button is a widget that can be activated by pressing the specified key.

The text of the push button is displayed in the specified colour.

If the first character of the text is a letter then this letter is displayed in the specified shortcut colour.
