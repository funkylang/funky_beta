std::normalize

trims the string and replaces whitespace by a single space

Parameters

self
the string to normalize

Result

the normalized string

Description

The function returns a string which is the specified string without any leading or trailing whitespace and where all sequences of whitespace are replaced by a single space.

Topic

Strings

See also

std::trim removes leading and trailing whitespace

Example

$str " This is a strange text! " $normalized_str normalize(str) println! '*' normalized_str '*'

Output

*This is a strange text!*
