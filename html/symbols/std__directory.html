std::directory

lists the contents of a directory

Parameters

options
a list of options
pathname
the path to the directory

Results

a list of directory entries

Description

Attention: This function must be called with I/O-access rights!

The function displays the names of the files and directories in the specified directory.

Topic

Input-Output

See also

std::file_exists checks if a file exists
std::open opens a file

Example

directory! $entries "/usr/bin" for_each entries : (entry) println! name_of(entry) next! pass
