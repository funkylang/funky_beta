std::directory

returns a list of entries in a directory

Parameters

options (zero or more)
options to control the behavior of the function
pathname
the path to the directory

Result

a list of directory entries

Description

Attention: This function must be called with I/O-access rights!

The function opens the specified directory, reads all entries and returns them as a list.

The function supports the following options:

- SORT: if set to true, the entries are sorted by name

Topic

Input-Output

See also

std::file_exists checks if a file exists
std::is_a_directory checks if a file is a directory
std::stat retrieves information about a file
std::readdir reads the next directory entry

Example

directory! $entries "/usr/bin" for_each entries : (entry) println! name_of(entry) next! pass
