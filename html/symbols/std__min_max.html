std::min_max

returns the minimum and maximum of the two given values

Parameters

first_value
the first value
second_value
the second value

Results

the minimum and maximum of the two values

Topic

Math

See also

std::min returns the smallest of the given values
std::max returns the largest of the given values

Example

$a 47 $b 11 min_max &a &b println! "min: " a println! "max: " b

Output

min: 11 max: 47
