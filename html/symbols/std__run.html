std::run

starts a new process

Parameters

self
the input-output object

Description

command: the command to execute

arguments: the arguments to pass to the command

Result

self
the input-output object

The command is executed asynchronously. The process ID of the new process is added to the list of watched processes.

Related topics

Input-Output Events

Example

create_process! $pid $in_fd $out_fd $err_fd command arguments ... join_process &io pid ... loop: get_events! &io $events for_each events : (event) event $type $epi
