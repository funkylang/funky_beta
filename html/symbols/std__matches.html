std::matches

checks whether a string matches the specified expression

Parameters

self
the string to check
expr
the expression to match

Result

*true* if the string matches the expression, *false* otherwise

Topic

Strings

See also

std::contains checks whether a collection contains the specified item or not
std::has_prefix checks whether the first argument has the second argument as a prefix
std::has_suffix checks whether the string has the specified suffix

Example

$my_string = "Hello, World!" println! my_string.matches("Hello, World!") # prints *true* println! my_string.matches("Hello") # prints *false*
