std_types::FUNKY_IDENTIFIER/std::match

matches the start of the stream with a Funky identifier

Parameters

self
the Funky identifier to match
stream
the stream to match against

Results

length
the number of matched characters (or std::undefined on fail)
capture
the matched string (if requested)

Description

Attempts to match the start of the stream with the specified Funky identifier.

If the match is successful the length of the match (= the length of the Funky identifier) is returned.

If the caller requested two result values then in addition to the string length the string itself ("the capture") is also returned.

If the match fails then one or two std::undefined-values are returned.
