std_types::FUNKY_IDENTIFIER/std::search

searches for a Funky identifier

Parameters

self
the Funky identifier to search for
str
the string to search in
nth
the n-th occurrence to search for

Results

pos
the position of the n-th occurrence (or std::undefined on fail)
len
the length of the n-th occurrence (or std::undefined on fail)

Description

Attempts to search for the n-th occurrence of the specified Funky identifier in the specified string.

If the search is successful the position and the length of the occurrence are returned.

If the search fails then two std::undefined-values are returned.
