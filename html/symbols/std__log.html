std::log

logs a message to the standard error stream.

Parameters

self
the object from which to get the log level

Description

level: the log level (default: 1)

message: the message to log

Result

self
the updated object

The log level determines the severity of the message. Higher levels indicate more severe messages.

The default log level is 1.

Topic

Input-Output Events

See also

std::initialize_logging initializes the logging system
std::enable_logging Enables logging for the given input-output object.
std::disable_logging disables logging for the given input-output object.

Example

$io std_types::io # create an input-output object enable_logging &io 3 # enable logging with level 3 log &io 1 "This is a log message." log &io 2 "This is a warning message." log &io 3 "This is an error message."
