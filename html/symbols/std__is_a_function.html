std::is_a_function

checks if an object is a function

Parameter

object
the object to check

Result

boolean
true if the object is a function, false otherwise

Topic

Functions

See also

std_types::function the prototype object for all functions
std_types::polymorphic_function the prototype object for all polymorphic functions

Example

$my_function create_function(x -> x*2) println! std::is_a_function(my_function) # prints true println! std::is_a_function(10) # prints false

Implementations

std_types::function always returns true
std_types::object succeeds for function objects
std_types::polymorphic_function succeeds for polymorphic function objects
