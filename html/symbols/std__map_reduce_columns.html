std::map_reduce_columns

applies a map and a reduce function to each column of a table

Parameters

self
the table
map_function
a function that maps a single item to a new item
reduce_function
a function that combines two items into a single item

Result

a new table with the same number of rows as the original table

Description

The map function is applied to each item in each column of the table. The reduce function is then applied to the results of the map function for each column.

Topic

Collections

See also
