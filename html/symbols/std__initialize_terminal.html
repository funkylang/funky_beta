std::initialize_terminal

initializes a terminal

Parameters

options* # a list of options to use
body # the function to call

Result

terminal # the initialized terminal

Description

The function initializes a terminal using the provided options.

The options can be used to specify the draw colour, clear colour, file descriptor, whether to receive CTRL-C and CTRL-Z, whether to receive CTRL-Q and CTRL-S, whether to map CR to LF on input, whether to receive SIGWINCH, whether to receive mouse clicks, and whether to switch the cursor off.

The body function is called with the initialized terminal as an argument.

The function returns the initialized terminal.

Example
