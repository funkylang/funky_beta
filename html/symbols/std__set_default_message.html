std::set_default_message

sets the default message

Parameters

wm
a window manager
screen
the default message

Result

wm
the updated window manager

Description

The default message is shown when no other message is active.

Topic

Window Manager

See also

Example

$my_wm create_window_manager(terminal)
set_default_message &my_wm "Hello, World!"
