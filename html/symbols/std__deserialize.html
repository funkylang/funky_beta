std::deserialize

deserializes a machine readable string representation of an object

Parameters

text
the string to deserialize

Result

the deserialized object

Description

The string must have been created by std::serialize.

Topic

Serialization

See also

std::serialize returns a machine readable string representation of the object
std::deserializers a hash table that maps serialization tags to deserializers
std::register_type registers a type for deserialization
std::register_types registers a list of types for deserialization

Example

$str serialize(42) println! str $obj deserialize(str) println! obj
