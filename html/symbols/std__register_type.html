std::register_type

registers a type for deserialization

Parameters

deserializers
a hash table of deserializers
type
the type to register

Result

deserializers
the updated hash table of deserializers

Description

The function registers a type for deserialization.

Topic

Serialization

See also

std::deserializers a hash table that maps serialization tags to deserializers
std::deserializer_of returns a function that can deserialize an object of the given type
std::register_types registers a list of types for deserialization
std::serialize returns a machine readable string representation of the object
std::serialization_tag_of returns the serialization tag of the object

Example

$deserializers empty_hash_table
register_type(deserializers Point)
register_type(deserializers Line)
register_type(deserializers Rectangle)
register_type(deserializers Circle)
register_type(deserializers Ellipse)
register_type(deserializers Polygon)
register_type(deserializers Polyline)
register_type(deserializers Path)
register_type(deserializers Text)
register_type(deserializers Image)
register_type(deserializers Group)

dump! 2 `deserializers
