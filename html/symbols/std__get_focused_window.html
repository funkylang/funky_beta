std::get_focused_window

returns the content of the focused window

Parameter

window_manager
the window manager

Result

id
the id of the focused window
content
the content of the focused window

Topic

Window Manager

See also

std::get_window_at returns the window at the given position

Example

$wm create_window_manager(terminal)
$id create_window(&wm CONTENT "Hello, World!")
focus_window &wm id
$id content get_focused_window(&wm)
println! content
