ai::effective_context_size_of

Returns the effective context size to use with the model.

Description

The effective context size must be less or equal to the context size defined for the model (see ai::context_size_of).

If the effective context size is not defined, the context size defined for the model is used.

Parameter

self
the model

Result

context_size
the effective context size
