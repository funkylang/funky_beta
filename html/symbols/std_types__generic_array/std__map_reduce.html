std_types::generic_array/std::map_reduce

combines a map and a reduce operation into a single operation

Description

Does not create an intermediate array to hold the "mapped" values.

Parameters

self
the array to work on
map_function
the function to apply to each element of the array
reduce_function
the function to apply to pairs of values
_unit
the value to return for empty arrays

Result

result
the result of the reduce operation
