std::capture

creates a capture grammar object

Parameter

expr
the expression to capture

Result

a capture grammar object

Description

This function creates a capture grammar object that captures the matched text.

Topics

Grammar Objects

Example

$GLOBAL_DEFINITION
  '$', capture(FUNKY_IDENTIFIER), "::", capture(FUNKY_IDENTIFIER)
$str "$std::print 123"
match GLOBAL_DEFINITION str $len $captured
println! len
dump `captured
