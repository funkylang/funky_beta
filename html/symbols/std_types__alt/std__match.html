std_types::alt/std::match

Attempts to match the start of the string against any of the expressions contained in the alt object.

Description

If any of the expressions matches, the length of the match is returned.

If none of the expressions match, std::undefined is returned.

Parameters

self: the alt object

str: the string to match against

Result

length: the length of the match (or std::undefined on fail)
