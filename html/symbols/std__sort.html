std::sort

sorts a collection

Parameters

self
the collection to sort
compare
a comparison function (optional)

Result

a new sorted collection

Description

The result is a new collection containing all the items of the original collection in sorted order.

If no comparison function is supplied then the items are sorted in ascending order using the std::less-method.
