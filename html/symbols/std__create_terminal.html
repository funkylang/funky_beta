std::create_terminal

creates a new terminal object

Parameters

width
the width of the terminal
height
the height of the terminal
options
a list of key-value-pairs
DRAW_COLOUR
the default draw colour (default: BLACK)
CLEAR_COLOUR
the default clear colour (default: WHITE)
VISIBLE_CURSOR
a boolean value indicating whether the cursor is visible
(default
true)

Result

terminal
a new terminal object

Topic

Terminal

<require basic/stdlib> <require terminal/terminal> get_terminal_size!
