std::is_a_key_order_table

checks if an object is a key-order table

Parameter

self
the object to check

Result

true if the object is a key-order table, false otherwise

Topic

$my_table std::key_order_table println! std::is_a_key_order_table my_table # prints true $my_list std::list println! std::is_a_key_order_table my_list # prints false
