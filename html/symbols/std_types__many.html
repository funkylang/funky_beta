std_types::many

the prototype object for "many"-expressions

Parameters

expr
the expression to match multiple times

Result

object
the created "many"-expression

Description

This object is never used directly but cloned to create a new "many"- expression.

Topics

Grammar Objects

See also

std_types::many/std::match checks whether the specified string starts with one or more occurrences
std_types::many/std::match_back matches the end of the string with the expression
std_types::many/std::search searches for the specified expression

Example

$str "12345abcdef" match many(DIGIT) str $len dump! `len

Output

len: 5
