std::perform_io

starts an I/O operation

Parameters

self
the task object
body
the I/O function to be called
arguments
the arguments for the I/O function
continuation
the function to be called after the I/O operation has
completed

Result

none

Description

The I/O operation is started asynchronously. The continuation function will be called when the I/O operation has completed.

Topic

Input-Output Task Manager

See also

std::check_io checks the result of an I/O operation
std::complete_io completes an I/O operation

Example

$my_task create_task perform_io &my_task read "/dev/tty" 1000 my_continuation
