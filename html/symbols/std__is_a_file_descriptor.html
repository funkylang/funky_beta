std::is_a_file_descriptor

checks if an object is a file descriptor

Parameter

self
the object to check

Result

true if the object is a file descriptor, false otherwise

Topic

Input-Output

See also

std::file_descriptor constructs a file descriptor from its integer representation

Example

$fd open! "test.dat" "r"
println! std::is_a_file_descriptor(fd) # prints true
println! std::is_a_file_descriptor(1) # prints false
