std::get_rectangle_by_key

returns a rectangle by its key

Parameters

screen
some kind of screen
key
the key of the rectangle

Result

rectangle
the rectangle or undefined

Description

Returns the rectangle with the given key.

Topic

Terminal

See also

std::get_rectangle_at returns the rectangle at the given position
std::get_next_rectangle returns the next rectangle
std::get_previous_rectangle returns the previous rectangle

Example

$my_screen create_screen(terminal 40 25) draw_border &my_screen 1 1 38 24 "My Screen" get_rectangle_by_key &my_screen "My Screen" $rectangle draw_text &my_screen 2 3 "The rectangle's width is " draw_text &my_screen 26 3 width_of(rectangle)
