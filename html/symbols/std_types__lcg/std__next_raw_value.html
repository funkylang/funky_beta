std_types::lcg/std::next_raw_value

generates the next raw value

Parameters

self
the random number generator

Result

value
the next raw value

Description

The next raw value is calculated using the formula

new_seed = (multiplierseed + increment) mod modulus

The new seed value is stored in the generator.
