std::split

splits a string into several parts and returns a list of these parts

Description

The separators are removed.

Parameters

self
the string to split
separator
the separator to use (default: ' ')

Result

items
a list of the parts

Topic

Strings

See also

string/join [missing]

Example

$items split("a;b;c" ";") dump! `items

Output

items: list <3> "a" "b" "c"

Example

$items split("a;b;" ";") dump! `items

Output

items: list <3> "a" "b" ""
