std::scroll_down

Scrolls the contents of a screen downwards.

Parameters

screen
the screen to scroll
delta
the number of lines to scroll (default: 1)

Result

screen
the updated screen

Description

The function scrolls the contents of the screen downwards by the specified number of lines. New lines are added at the top of the screen, filled with the screen's default clear colour.

Topic

Terminal

See also

std::scroll_up Scrolls the contents of a screen upwards.
std::scroll_left scrolls the contents of a screen to the left by a given number of
std::scroll_right Scrolls the contents of a screen to the right.

Example

$my_screen create_screen(terminal 40 25) set_clear_colour &my_screen GREEN draw_text &my_screen 1 1 "Hello, world!" scroll_down &my_screen 2
