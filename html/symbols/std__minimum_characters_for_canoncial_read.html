std::minimum_characters_for_canoncial_read

set or get the value of VMIN in c_cc

value
the new value for VMIN (optional)

Result

Example

# get the current value of VMIN
attributes = get_terminal_attributes(stdin)
value = minimum_characters_for_canoncial_read(attributes)
print("VMIN = ", value)

# set the value of VMIN to 10
attributes = minimum_characters_for_canoncial_read(attributes, 10)
set_terminal_attributes(stdin, attributes)
