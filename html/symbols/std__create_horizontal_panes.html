std::create_horizontal_panes

creates a number of horizontal panes

Parameters

screen
the screen to create the panes on
pane_description
a list of pane heights or an integer

Result

views
a list of views

Description

If pane_description is an integer then this number of panes of equal height are created.

If pane_description is a list then the list elements are interpreted as pane heights. If a pane height is less than 1 then the pane height is calculated as a fraction of the remaining height.

Topic

Terminal

See also
