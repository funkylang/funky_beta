std_types::date_and_time/std::to_string

returns a string representation of the date-and-time object

Parameters

self
the date-and-time object

Results

a string representation of the date-and-time object

Description

The function returns a string representation of the date-and-time object in the format "YYYY-MM-DD HH:MM:SS.SSS GMT+HH".

The time is adjusted by the time shift of the date-and-time object.

The time shift is the difference between the local time and the UTC time.

The function uses the following functions to extract the components of the date-and-time object:

  • year_of
  • month_of
  • day_of
  • hour_of
  • minute_of
  • second_of
  • day_of_week_of
  • time_shift_of
