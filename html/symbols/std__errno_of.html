std::errno_of

retrieves the errno associated with the error object

Description

This field is used in conjunctions with the "IO ERROR" error.

Topic

Errors

See also

std::error_message_text_of retrieves the error message text of an error object
std::error_details_of retrieves the details associated with the error object
std::failed_attribute_of retrieves the attribute associated with the error object

Example

load! $buf "some-non-existing-file" on buf.is_an_error: dump! `errno_of(buf)

Output

errno_of(buf): object <0>
