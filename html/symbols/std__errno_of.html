std::errno_of

the errno associated with an error object

Description

This field is used in conjunctions with the "IO ERROR" error and represents the contents of the errno variable when the error object was created.

Topic

Example

load! $buf "some-non-existing-file"
on buf.is_an_error:
  debug::dump `errno_of(buf)
