std::read_from

reads a specified number of bytes from a file descriptor

Parameters

fd
the file descriptor to read from
length
the number of bytes to read

Results

a string containing the read bytes or an error

Description

Attention: This function must be called with I/O-access rights!

The function reads the specified number of bytes from the file descriptor and returns these bytes as a string.

If the end of the file is reached, the function returns the already read bytes as a string.

If an error occurs, the function returns the error.
