ai::evaluate

evaluates a prompt using a loaded AI model

Parameters

self
the object from which to get the AI model
prompt
the prompt to evaluate
mode
the evaluation mode (SMART or EXACT)
start
the starting index for evaluation

Result

result
the evaluation result

Description

The result is a list of tokens or a list of tuples containing the token and its confidence value.

Topic

Deprecated AI

See also

ai::load_ai_model loads an AI model
ai::tokenize tokenizes a text string
ai::detokenize detokenizes a token or a list of tokens

Example

$modelname "sauerkrautem-una-solar-instruc.Q5_K_M.gguf" $prompt "Once upon a time" ai::load_ai_model! $model modelname ai::evaluate! $result model prompt print! result

Output

[("Once", 0.999), ("upon", 0.998), ("a", 0.997), ("time", 0.996)]
