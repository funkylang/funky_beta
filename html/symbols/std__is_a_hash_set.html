std::is_a_hash_set

checks if an object is a hash set

Parameter

self
the object to check

Result

true if the object is a hash set, false otherwise

Topic

Collections

See also

Example

$my_set create_hash_set() println! std::is_a_hash_set(my_set) # prints true $my_list create_list() println! std::is_a_hash_set(my_list) # prints false
