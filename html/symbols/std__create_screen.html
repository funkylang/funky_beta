std::create_screen

creates a new screen object.

Parameters

screen # a screen object

Description

width: width of the screen

height: height of the screen

options: a list of options

Result

screen
the newly created screen object

The function creates a new screen object with the specified width and height.

The options parameter can be used to specify the initial draw and clear colours of the screen.

The following options are supported:

DRAW_COLOUR: the initial draw colour of the screen

CLEAR_COLOUR: the initial clear colour of the screen

Example

$my_screen create_screen(40 25)
