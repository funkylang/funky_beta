std::enable_logging

Enables logging for the given input-output object.

Parameters

self
the input-output object

Description

level: the logging level (default: 1)

Result

self
the input-output object

The logging level determines the verbosity of the logging output.

Topic

Input-Output Events

See also

std::disable_logging disables logging for the given input-output object.
std::log logs a message to the standard error stream.

Example

$io std_types::io # create an input-output object
enable_logging &io 3 # enable logging with level 3
# ... your code ...
disable_logging &io # disable logging
