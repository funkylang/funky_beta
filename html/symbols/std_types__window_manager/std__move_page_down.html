std_types::window_manager/std::move_page_down

moves the content of a window down one page

Parameters

window_manager
a window manager
id (optional)
the id of the window to move (default: the focused window)

Result

window_manager
the updated window manager

Description

Moves the contents of the window with the specified id down one page.

If no id is specified then the contents of the focused window are moved down.

If the window has a cursor, then the cursor is moved down one page instead. In this case the contents do not necessarily scroll!
