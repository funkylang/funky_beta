std_types::window_manager/std::move_window_up

moves the specified window one position up

Parameters

self
the window manager
id
the id of the window to move

Result

window_manager
the updated window manager

Description

The specified window is moved one position up in the window stack.

If the window is already at the top of the stack then it is moved to the bottom of the stack.
