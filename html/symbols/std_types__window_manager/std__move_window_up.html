std_types::window_manager/std::move_window_up

moves the specified window one position up

Parameters

wm
the window manager
id
the id of the window to move

Result

wm
the updated window manager

Description

If no id is specified then the focused window is moved.

If the window is already at the top then it is moved to the bottom.
