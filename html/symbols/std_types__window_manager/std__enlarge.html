std_types::window_manager/std::enlarge

maximizes a window

Parameters

window_manager
a window manager
id (optional)
the id of the window to maximize (default: the focused window)

Result

window_manager
the updated window manager

Description

Maximizes the window with the specified id.

If no id is specified then the focused window is maximized.

Topic

Window Manager

See also

std_types::window_manager/std::shrink minimizes a window

Example

$wm create_window_manager(terminal)
create_window &wm $id "My Window"
...
enlarge &wm id
