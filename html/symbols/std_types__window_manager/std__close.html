std_types::window_manager/std::close

closes a window

Parameters

self
the window manager
id
the id of the window to be closed

Result

window_manager
the updated window manager

Description

If the window is the currently focused window then the focus is moved to the next window (or to the previous window if the currently focused window is the last window).

If the window manager is empty after the window has been closed then the focus is unset.
