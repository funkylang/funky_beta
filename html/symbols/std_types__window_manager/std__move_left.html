std_types::window_manager/std::move_left

moves the leftmost visible column of the specified window one column to the left

Parameters

self
the window manager
id
the id of the window to move (if omitted then the focused window is
moved)

Result

window_manager
the updated window manager

Description

If the leftmost visible column is already at the left border then the window manager is not changed.

Window Manager

Example

$wm create_window_manager(std::screen) $wm create_window("Window 1" CONTENT "Hello, World!") $wm create_window("Window 2" CONTENT "Hello, World!") $wm move_left(2)

Output

Window 1 Window 2 Hello, World! Hello, World!
