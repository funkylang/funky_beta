std_types::window_manager/std::handle_event

handles an event

Parameters

wm
the window manager
event
the event to handle
wm_fd
the file descriptor of the window manager

Results

wm
the updated window manager
event
the updated event

Description

This function handles an event.

If the event is a mouse click then the window manager's menu bar and dialogs are checked first.

If the event is a key press then the window manager's menu bar and dialogs are checked first.

If the event is a new window size then the window manager is resized.

If the event is not handled by the window manager then it is passed to the focused window.

If the event is handled by the window manager then the updated window manager and the updated event are returned.

If the event is not handled by the window manager then the updated window manager and the original event are returned.
