std_types::window_manager/std::create_window

creates a new window

Parameters

window_manager
the window manager
options (zero or more)
how to create the new window

Result

window_manager
the updated window manager
window_id
the id of the new window (if requested)

Description

Creates a new window with the specified options.

The options can be:

  • ID: the id of the window (default: auto-generated)
  • USE_CURSOR: whether to use a cursor (default: false)
  • AS_FIRST: whether to place the window at the beginning of the list (default: false)
  • DO_NOT_FOCUS: whether to not focus the window (default: false)
  • TITLE: the title of the window (default: "std::untitled")
  • CONTENT: the content of the window
  • CLEAR_COLOUR: the background color (default: WHITE)
  • HORIZONTAL_SCROLLBAR: whether to show a horizontal scrollbar (default: true)
  • VERTICAL_SCROLLBAR: whether to show a vertical scrollbar (default: true)
  • BUTTONS: additional buttons (default: empty list)
  • SIZE: the size of the window (default: DEFAULT)
  • CURSOR_X: the initial x position of the cursor (default: 1)
  • CURSOR_Y: the initial y position of the cursor (default: 1)
  • FIRST_VISIBLE_COLUMN: the first visible column (default: 1)
  • FIRST_VISIBLE_ROW: the first visible row (default: 1)
  • MODAL: whether the window is modal (default: false)
  • PARENT_ID: the id of the parent window (default: undefined)

If the AS_FIRST option is set, the new window is placed at the beginning of the window list.

If the DO_NOT_FOCUS option is set, the new window is not focused.

If the SIZE option is set to MAXIMIZED, the window is maximized.
