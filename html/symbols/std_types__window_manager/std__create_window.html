std_types::window_manager/std::create_window

creates a new window

Parameters

self
the window manager
title
the window's title
options
a list of key-value-pairs

Results

window_manager
the updated window manager
window_id
the id of the created window

Description

Creates a new window and adds it to the window manager.

The window is initially focused.

If the caller requests two result values then the window id is also returned.

The following options are supported:

CONTENT: the window's content CLEAR_COLOUR: the window's clear colour HORIZONTAL_SCROLLBAR: true if the window should have a horizontal scrollbar VERTICAL_SCROLLBAR: true if the window should have a vertical scrollbar FIRST_VISIBLE_COLUMN: the index of the first visible column FIRST_VISIBLE_ROW: the index of the first visible row
