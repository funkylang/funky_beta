std_types::window_manager/std::handle_window_event

handles events for a window

Parameters

Result

window_manager

Description

Handles events for the window manager.

The event is processed if it is a key press event for the specified file descriptor.

The following key presses are handled:

  • CURSOR_LEFT: move the window left
  • CURSOR_RIGHT: move the window right
  • CURSOR_UP: move the window up
  • CURSOR_DOWN: move the window down
  • PAGE_UP: move the window up a page
  • PAGE_DOWN: move the window down a page
  • HOME: move the window to the beginning of the line
  • END: move the window to the end of the line
  • CTRL_HOME: move the window to the top
  • CTRL_END: move the window to the bottom
