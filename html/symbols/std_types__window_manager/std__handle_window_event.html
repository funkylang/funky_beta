std_types::window_manager/std::handle_window_event

handles a window event

Parameters

wm
the window manager
event
the event to handle
wm_fd
the file descriptor of the window manager

Result

event
the event to be handled by the caller

Description

This method handles the following events:

  • CURSOR_LEFT
  • CURSOR_RIGHT
  • CURSOR_UP
  • CURSOR_DOWN
  • PAGE_UP
  • PAGE_DOWN
  • HOME
  • END
  • CTRL_HOME
  • CTRL_END

If the event is handled by this method then std::undefined is returned.

If the event is not handled by this method then the event is returned unchanged.
