std_types::window_manager/std::goto_xy

moves the cursor to the specified position

Parameters

wm
the window manager
id
the id of the window
x
the x-coordinate
y
the y-coordinate

Result

wm
the updated window manager

Description

If the window is not focused then the window is focused.

If the window is not a dialog then the cursor is moved to the specified position.

If the window is a dialog then the cursor is moved to the specified position if the position is within the dialog's content.
