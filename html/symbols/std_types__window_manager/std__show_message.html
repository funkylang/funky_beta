std_types::window_manager/std::show_message

shows a message in the window manager's status line

Parameters

wm
the window manager
id
the message id (if omitted then a new id is generated)
timeout
the number of seconds the message is shown
screen
the message to show

Result

wm
the updated window manager

Description

The message is shown in the window manager's status line.

If the message is shown for a limited time then it is removed after the specified time.

If the message is shown permanently then it can be removed by calling std_types::window_manager/std::remove_message.

If the message is shown permanently and a message with the same id is shown again then the old message is replaced by the new one.
