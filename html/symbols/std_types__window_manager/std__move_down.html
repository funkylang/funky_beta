std_types::window_manager/std::move_down

moves the window down

Parameters

self
the window manager
id
the id of the window to move (if undefined then the focused window
is moved)

Result

window_manager
the updated window manager

Description

Moves the window down by one line.

If the window is already at the bottom then it is not moved.
