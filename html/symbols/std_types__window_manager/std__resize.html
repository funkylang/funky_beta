std_types::window_manager/std::resize

resizes the window manager

Parameters

window_manager
a window manager
width
the new width
height
the new height

Result

window_manager
the updated window manager

Description

Resizes the window manager to the specified width and height.

This function should be called after the terminal managed by the window manager has changed its size.
