std_types::window_manager/std::move_end

shows the rightmost part of a window's contents

Parameters

window_manager
a window manager
id (optional)
the id of the window to move (default: the focused window)

Result

window_manager
the updated window manager

Description

Shows the rightmost part of the contents of the window with the specified id.

If no id is specified then the contents of the focused window are moved.

If the window has a cursor, then the cursor is moved to the end of the line.
