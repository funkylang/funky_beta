std_types::window_manager/std::needs_updating

checks if the window manager needs an update

Parameters

self
the window manager

Result

needs_updating
std::true if the window manager needs an update

Description

The window manager needs an update if

  • the window manager has no valid layout
  • the window manager has a new size
  • the window manager is not up to date
