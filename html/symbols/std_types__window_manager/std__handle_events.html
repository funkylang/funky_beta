std_types::window_manager/std::handle_events

handles events for the window manager

Parameters

self
the window manager
events
the events to handle
wm_fd
the file descriptor of the window manager

Result

window_manager
the updated window manager
events
the events that were not handled by the window manager

Description

This function handles events for the window manager.

The window manager handles the following events:

  • std_types::event/std::KEY_PRESS events with the following arguments: std_types::event/std::CURSOR_LEFT: moves the cursor left std_types::event/std::CURSOR_RIGHT: moves the cursor right std_types::event/std::CURSOR_UP: moves the cursor up std_types::event/std::CURSOR_DOWN: moves the cursor down std_types::event/std::PAGE_UP: moves the cursor up one page std_types::event/std::PAGE_DOWN: moves the cursor down one page std_types::event/std::HOME: moves the cursor to the beginning of the line std_types::event/std::END: moves the cursor to the end of the line std_types::event/std::CTRL_HOME: moves the cursor to the beginning of the content std_types::event/std::CTRL_END: moves the cursor to the end of the content std_types::event/std::CTRL_CURSOR_RIGHT: focuses the next window std_types::event/std::CTRL_CURSOR_LEFT: focuses the previous window std_types::event/std::CTRL_W: closes the focused window

All other events are passed on to the caller.
