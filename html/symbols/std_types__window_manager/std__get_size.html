std_types::window_manager/std::get_size

returns the size of the specified window

Parameters

wm
the window manager
id
the id of the window

Results

width
the width of the window
height
the height of the window

Description

If the window is minimized then the size of the window's title bar is returned.

If the window is maximized then the size of the screen is returned.

If the window is in default size then the size of the window's content is returned.

If the window is not found then std::undefined is returned for both results.
