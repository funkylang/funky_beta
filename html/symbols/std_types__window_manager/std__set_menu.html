std_types::window_manager/std::set_menu

sets the menu bar

Parameters

self
the window manager
menu
the menu bar

Result

window_manager
the updated window manager

Description

The menu bar is a list of menu items.

Each menu item is a key-value-pair.

The key is the menu item's label.

The value is a list of menu entries.

Each menu entry is a key-value-pair.

The key is the menu entry's label.

The value is a function that is called when the menu entry is selected.

The function is called with the window manager as its only parameter.

Example

$wm std::create_window_manager $wm set_menu list "File" list "New" -> new "Open" -> open "Save" -> save "Save as" -> save_as "Quit" -> quit "Edit" list "Cut" -> cut "Copy" -> copy "Paste" -> paste
