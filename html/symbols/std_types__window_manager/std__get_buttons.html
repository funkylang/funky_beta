std_types::window_manager/std::get_buttons

returns the additional buttons of the specified window

Parameters

self
the window manager
id
the id of the window

Result

buttons
the additional buttons of the specified window

Description

If no window id is specified then the id of the focused window is used.

If the window does not exist then std::undefined is returned.
