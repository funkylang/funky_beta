std_types::window_manager/std::shrink

shrinks the specified window

Parameters

self
the window manager
id
the id of the window to shrink

Result

window_manager
the updated window manager

Description

If the window is in DEFAULT state then it is set to MINIMIZED.

If the window is in MAXIMIZED state then it is set to DEFAULT.

If the window is in MINIMIZED state then nothing happens.

If the window is the last window in DEFAULT or MAXIMIZED state then nothing happens.

If the window is the last window in DEFAULT or MAXIMIZED state then the next window is set to DEFAULT.
