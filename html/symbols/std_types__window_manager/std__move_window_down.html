std_types::window_manager/std::move_window_down

moves a window down in the window list

Parameters

window_manager
a window manager
id (optional)
the id of the window to move (default: the focused window)

Result

window_manager
the updated window manager

Description

Moves the window with the specified id down one position in the window list.

If no id is specified then the focused window is moved down.

If the window is already at the bottom of the list, it is not moved.
