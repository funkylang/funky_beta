std_types::window_manager/std::move_window_down

moves the specified window down

Parameters

self
the window manager
id
the id of the window to move down

Result

window_manager
the updated window manager

Description

The specified window is moved down in the window stack.

If the window is already at the bottom of the stack then it is moved to the top of the stack.
