std::program_parameters

parses the program parameters

Parameters

templates (zero or more)
the accepted options and arguments

Result

the parsed parameters

Description

This function must be called with IO-access rights.

This function parses the program parameters according to the given templates.

To make the code more readable it is recommended to interleave the option templates and the results.
