std::program_parameters

This function parses the command line arguments and returns the results.

Description

The function accepts a variable number of templates.

Each template has the form

OPTION = default value

The following options are supported:

  • std::VALUED_OPTION
  • std::MULTI_VALUED_OPTION
  • std::MANDATORY_PARAMETER
  • std::OPTIONAL_PARAMETER
  • std::SOME_PARAMETERS
  • std::MANY_PARAMETERS

To make the code more readable it is recommended to interleave the option templates and the results.

Example

program_parameters! $verbose list "verbose" " output detailed information about the build process $rebuild list "rebuild" " rebuild all source files $print_c list "print-c" " print C source code for a single module $input_files list SOME_PARAMETERS "input" " the input files to use; if none are specified <stdin> is used
