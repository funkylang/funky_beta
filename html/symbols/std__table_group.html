std::table_group

creates a table group

Parameters

rows
the number of rows
columns
the number of columns
spacing
the spacing between the cells
alignment
the alignment of the cells
widgets
the widgets to add to the table

Result

group
the table group

Description

The table group is a widget group that arranges its widgets in a table.

Topic

Widgets

See also

std::horizontal_group creates a horizontal group of widgets
std::vertical_group creates a vertical group of widgets

Example

$group table_group(2 3 2 CENTER) $group.add_widget text_box("First name") $group.add_widget text_box("Last name") $group.add_widget text_box("Age") $group.add_widget text_box("John") $group.add_widget text_box("Doe") $group.add_widget text_box("42") $group.add_widget text_box("Mary") $group.add_widget text_box("Smith") $group.add_widget text_box("25") $dialog okay_dialog(group) $dialog.run
