std::table_group

creates a table group

Parameters

rows
a list of lists of widgets

Result

group
a table group

Description

The table group is a widget group that arranges the widgets in a table.

Topic

Widgets

See also

std::horizontal_group creates a horizontal group of widgets
std::vertical_group creates a vertical group of widgets

Example

$dialog create_dialog(wm "Table Group")
  table_group
    list
      list
        text_box("Name")
        text_box("Age")
      list
        text_box("John")
        text_box("42")
      list
        text_box("Mary")
        text_box("25")
      list
        text_box("Irene")
        text_box("49")
    horizontal_group
      push_button
        "Ok" = OKAY
      push_button
        "Cancel" = CANCEL
$result show_dialog(dialog)
