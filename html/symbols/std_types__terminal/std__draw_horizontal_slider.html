std_types::terminal/std::draw_horizontal_slider

draws a horizontal slider

Parameters

self
the terminal to draw on
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the slider
height
height of the slider
total
the number of total items
current
the number of currently visible items
pos
the index of the first visible item (one-based)

Result

terminal
the updated terminal

Description

Draws a horizontal slider on the specified terminal.

The slider is drawn using the current draw- and clear colour.
