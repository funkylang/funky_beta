std_types::terminal/std::update

updates the terminal with the changes made to the screen

Parameters

self
the terminal to be updated

Result

terminal
the updated terminal

Description

This method updates the terminal with the changes made to the screen.

It iterates over all rows of the screen and compares the characters, draw colours and clear colours of the current row with the corresponding values of the previous row. If there are any differences, the method generates the appropriate ANSI escape sequences to update the terminal.

The method also handles cursor positioning and cursor colour.
