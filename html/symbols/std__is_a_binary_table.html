std::is_a_binary_table

checks if an object is a binary table

Parameter

self
the object to check

Result

true if the object is a binary table, false otherwise

Topic

Tables

See also

std_types::binary_table a table based on binary trees

Example

$my_table create_binary_table()
println! std::is_a_binary_table(my_table)

Output:

true

Implementations

std_types::binary_table always returns true
std_types::object always returns false
