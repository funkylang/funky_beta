std::reduce_rows

reduces a list of lists (a table) row-wise

Parameters

self
the table to reduce
reduction
a function that takes two arguments: the current row and the accumulated result

Result

a list containing the reduced rows

Description

The reduction function is applied to each row of the table. The first argument to the reduction function is the current row, and the second argument is the accumulated result from the previous rows.

The initial value of the accumulated result is undefined.

Topic

Collections

See also

std::reduce_columns reduces the columns of a table
std::map_reduce_rows applies a map and a reduce function to each row of a table
std::map_reduce_columns applies a map and a reduce function to each column of a table

Example

$my_table create_table( list(1 2 3) list(4 5 6) list(7 8 9) ) $sum_rows std::reduce_rows(my_table (row sum) -> sum + row) println! sum_rows prints list(12 15 18)
