std::bresenham_delta

returns an iterator

Description

The iterator produces count equally distributed values that sum up to amount.

Parameters

count
the number of values to return
amount
the sum of all returned values (a positive value)

Result

iterator
an iterator that produces the values

Example

count 5
$amount 7

$iter bresenham_delta(count amount)
repeat count:
  next_value &iter $value
  println! value
  next!

Output

1
2
1
2
1
