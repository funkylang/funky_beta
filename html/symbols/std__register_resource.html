std::register_resource

registers a resoource with the input-output object.

Parameters

self
the input-output object
key
the key to use for the resoource
resource
the resoource to register

Result

self
the updated input-output object

Related topics

Input-Output

Example

$io std_types::io # create an input-output object $file create_file "my_file.txt" register_resource &io "my_file" &file # register the file with the key "my_file"
