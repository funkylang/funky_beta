std::log_level

the used log-level

Topic

Input-Output Task Manager

Description

The log-level is used to control the amount of log-messages that are printed.

The following log-levels are defined:

0: no logging 1: just print the log messages 2: prefix the log messages with the task no. 3: add iteration separator lines to the log output 4: log the creation and deletion of tasks
