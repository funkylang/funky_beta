std::yes_no_dialog

displays a dialog with a question and two buttons "Yes" and "No"

Parameters

title
the title of the dialog
question
the question to ask

Result

YES_BUTTON or NO_BUTTON

Description

The dialog is displayed in the middle of the screen.

Topic

Widgets

See also

std::okay_dialog displays a dialog with a message and an "Okay"-button
std::okay_cancel_dialog displays a dialog with a message and an "Okay" and a "Cancel" button

Example

$answer yes_no_dialog("Question", "Do you like Funky?") if answer == YES_BUTTON: println! "That's great!" : println! "That's too bad!"
