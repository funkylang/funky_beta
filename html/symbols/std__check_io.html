std::check_io

checks the result of an I/O operation

Parameters

self
the task object
io
the I/O object
body
the I/O function
arguments
the arguments of the I/O function

Result

self
the task object

Description

The result of the I/O operation is checked. If the operation was successful, the task proceeds to the next phase. If the operation failed, the task's error handler is called.
