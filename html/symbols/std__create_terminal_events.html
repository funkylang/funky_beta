std::create_terminal_events

creates a list of input events from the input of a terminal

Parameters

io
the I/O-object
fd
the file descriptor of the terminal
input
the input of the terminal

Result

events
a list of input events

Description

The function creates a list of input events from the input of a terminal. The input is a string that contains the input of the terminal in UTF-8 encoding. The function extracts the input events from the string and returns a list of input events.

Input-Output Events

$io create_io_object $fd open_terminal(io) $input read_terminal(io fd) $events create_terminal_events(io fd input) dump! 2 `events
