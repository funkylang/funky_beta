std::is_an_uppercase_letter

checks wheter its argument is an uppercase letter

Result

success # true if the argument is an uppercase letter

Description

This function must be called on character-values only.

Topic

Strings

See also

std::is_a_letter checks wheter its argument is a letter (upper or lower case)
std::is_a_lowercase_letter checks wheter its argument is a lowercase letter
std_types::character a unicode code point encoded as a 32-bit value
std::is_a_digit checks wheter its argument is a digit

Example

if
  is_an_uppercase_letter('T'):
    println! "T is an uppercase letter"
  :
    println! "T is not an uppercase letter"

Output

T is an uppercase letter

Example

if
  is_an_uppercase_letter('t'):

  :
    println! "t is not an uppercase letter"

Output

t is not an uppercase letter
