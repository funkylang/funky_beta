std::result_count

returns the expected number of results

Description

When a function is called then the caller supplies the number of expected results as an invisible argument.

Topic

Basics

See also

std::parameter_count_of returns the number of arguments to use when calling its function

Example

$items list("apples" "bananas" "cherries") for_each items : (item) println! item next! pass
