std::create_window

creates a new window

Parameters

window_manager
the window manager to create the window in
title
the title of the window
options*
optional options for the window

Result

window_manager
the updated window manager
window_id
the id of the newly created window

Description

The following options are supported:

CONTENT: the content of the window CLEAR_COLOUR: the clear colour of the window HORIZONTAL_SCROLLBAR: whether the window has a horizontal scrollbar VERTICAL_SCROLLBAR: whether the window has a vertical scrollbar
