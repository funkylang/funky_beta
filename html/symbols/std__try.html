std::try

executes the specified body and catches any error

Parameter

body
a function that is called with I/O-access rights

Result

undefined on success; an error object on failure

Description

The function calls the specified body with I/O-access rights.

If an error occurs whithin the excution of the body, the error is caught and returned.

If no error occurs, the function returns std::undefined.

Topic

Input-Output

See also

std::check checks if its argument is an error
std::is_an_error returns true if the argument denotes an error object

Example

try $success: load! $buf "test.dat" print! buf on success.is_an_error: eprintln! "Failed to load test.dat" exit EXIT_FAILURE
