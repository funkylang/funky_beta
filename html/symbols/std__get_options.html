std::get_options

fetches the options of a function call

Parameters

options
the options passed to the function
templates (zero or more)
the option templates

Result

the values of the options

Description

This function returns a variable number of results - one for each option template. Options which do not match any template are ignored.

Each template has the form

OPTION = default value

To make the code more readable it is recommended to interleave the option templates and the results.

    CLEAR_COLOUR = WHITE $clear_colour
    TEXT_ATTRIBUTES = NORMAL $text_attributes
  ...
