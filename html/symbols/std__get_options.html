std::get_options

parses command line options

Parameters

arguments
a list of command line arguments
templates
a list of option templates

Result

options
a hash table containing the parsed options

Description

The function parses the command line arguments according to the specified option templates.

Topic

Command line

See also

std::extract_options extracts the options of a function call
std::program_parameters This function parses the command line arguments and returns the results.

Example

$arguments list("--help" "--verbose" "--output=out.txt" "in.txt") $templates list( hash_table( "help" hash_table( "type" VALUED_OPTION "default" false ) ) hash_table( "verbose" hash_table( "type" VALUED_OPTION "default" false ) ) hash_table( "output" hash_table( "type" VALUED_OPTION "default" "out.txt" ) ) hash_table( "input" hash_table( "type" MANDATORY_PARAMETER ) ) ) $options get_options(arguments templates) dump! 2 `options
