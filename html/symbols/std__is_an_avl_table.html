std::is_an_avl_table

checks if a given object is an AVL-table

Parameter

self
the object to check

Result

true if the object is an AVL-table, false otherwise

Topic

Tables

See also

std_types::avl_table the prototype object for all AVL-based tables

Example

$my_table create_avl_table() println! std::is_an_avl_table(my_table) # prints true
