generates a pseudo random number

Parameters

the pseudo random number generator

Result

a pseudo random number

Description

The pseudo random number generator uses the Mersenne Twister algorithm to generate pseudo random numbers.

The generated numbers are uniformly distributed in the range [0, 0xffffffff].
