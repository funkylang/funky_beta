std::get_mouse_reporting

returns the current mouse reporting mode

Parameter

self
the terminal

Result

mode
the current mouse reporting mode

Description

The mouse reporting mode can be changed using std::set_mouse_reporting.

Topic

Terminal

See also

std::set_mouse_reporting sets the mouse reporting mode
std::DISABLED disable mouse reporting
std::CLICKS used to enable mouse reporting for clicks
std::DRAG used to enable mouse drag reporting
std::MOVE the move mouse event

Example

$mode get_mouse_reporting(terminal)
