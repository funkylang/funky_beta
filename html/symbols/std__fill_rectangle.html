std::fill_rectangle

fill a rectangular area with the specified colour and opacity

Parameters

screen
some kind of screen
colour
the colour to use
opacity
the opacity to use
x
x-coordinate of the upper left corner
y
y-coordinate of the upper left corner
width
width of the area
height
height of the area

Result

screen
the updated screen

Description

The area is clipped at the screen's borders.

Topic

Terminal
