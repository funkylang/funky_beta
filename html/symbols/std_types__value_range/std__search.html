std_types::value_range/std::search

searches for a value-range in a string

Parameters

expr
the value-range to search for
str
the string to search in
nth
the nth occurrence to search for

Results

pos
the position of the first character of the match (or std::undefined on fail)
len
the length of the match (or std::undefined on fail)

Description

Attempts to search for the specified value-range in the specified string.

If the search is successful the position of the first character of the match and the length of the match are returned.

If the search fails then two std::undefined-values are returned.
