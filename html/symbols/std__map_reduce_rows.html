std::map_reduce_rows

applies a map and a reduce function to each row of a table

Parameters

self
the table

Description

map_function: a function that is applied to each row

reduce_function: a function that is applied to the results of the map function

Result

a new table with the same number of rows as the original table

The map function takes a row as input and returns a list of values.

The reduce function takes two values as input and returns a single value.

The reduce function is applied to the results of the map function for each row.

Topic

Collections
